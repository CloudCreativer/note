## E1 銷售素材庫

### DX-001｜保健品目錄匯入

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 後台／資料 |
| **描述** | 後台可建立保健品項目，含品名、功效摘要、建議劑量、禁忌提醒、導購 URL、主圖 |
| **驗收標準** | 至少匯入大象現有保健品清單；AI 回覆可引用品名與連結 |
| **備註** | 客戶稱銷售素材「還沒丟上去」 |

### DX-002｜線上課程目錄匯入

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 後台／資料 |
| **描述** | 課程含：名稱、適合對象、定價、Landing URL、免費試看 URL、第一章 URL |
| **驗收標準** | 「多囊實戰攻略版」等核心課程可被查詢並插入回覆 |
| **備註** | 含免費小講、消講連結 |

### DX-003｜文章庫＋導購標記

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 後台／資料 |
| **描述** | 每篇文章：標題、摘要、全文 URL、關鍵字 tag、文內導購產品關聯 |
| **驗收標準** | 單篇文章可對應 3～4 種常見問題（tag 匹配） |
| **備註** | 文章結尾已有導購，系統只需 deep link |

### DX-004｜看診／檢查服務項目

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 後台／資料 |
| **描述** | 荷爾蒙檢查、門診、網路問診等服務的說明文案＋預約／報名連結 |
| **驗收標準** | 回覆可區分「需個人化評估 → 建議檢查或看診」並附連結 |

### DX-005｜社群／群組入口

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 後台／資料 |
| **描述** | 社群邀請文案、加入連結、社群可提供的價值說明（曬單、討論） |
| **驗收標準** | 作為 funnel 最後一環 CTA 可選項目 |

### DX-006｜素材與知識庫關聯

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 後台／資料 |
| **描述** | 知識庫段落可綁定「推薦下一步」：保健品 ID、文章 ID、課程 ID |
| **驗收標準** | 編輯知識庫時可選預設 CTA，AI 依匹配結果帶出 |
| **依賴** | DX-001～005 |

## E2 回覆＋CTA

### DX-010｜回覆內嵌產品卡片（Flex）

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 訊息／前端 |
| **描述** | LINE 回覆支援卡片：圖片、標題、簡述、按鈕（前往購買／了解更多） |
| **驗收標準** | 保健品推薦時同時有文字說明＋可點擊卡片 |
| **備註** | 客戶明確問「需要產品連結跟圖片嗎？」→ 要 |

### DX-011｜回覆字數超限 → 文章 deep link

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI／訊息 |
| **描述** | 當回覆超過 LINE 字數或 AI 判斷需詳述時，摘要＋「完整說明請見文章」連結 |
| **驗收標準** | 多囊、避孕等長議題可導向對應文章而非截斷 |

### DX-012｜單則回覆多 CTA 排序規則

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI／策略 |
| **描述** | 定義一則回覆內 CTA 順序：知識摘要 → 保健品 → 個人化提醒（檢查／看診）→ 課程 → 社群 |
| **驗收標準** | 同一對話不出現「只有知識、無下一步」的回覆 |
| **備註** | 核心痛點：銷售閉環 |

### DX-013｜非診斷類問題的保健品建議模式

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI／Prompt |
| **描述** | 多囊、月經不規則等「可給生活／保健建議、非直接診斷」問題：功能、劑量、連結 |
| **驗收標準** | 不直接診斷；仍給有幫助的保健品資訊＋「更個人化請看診」 |

### DX-014｜直接醫學問題的護欄回覆

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI／Prompt |
| **描述** | 明確診斷／處方類問題：不直接回答，改導看診＋可選文章／基礎保健 |
| **驗收標準** | 與 DX-013 路由正確；符合初訪「保護病人」原則 |
| **依賴** | 初訪 Tone 設定 |

### DX-015｜CTA 點擊追蹤

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 分析 |
| **描述** | 記錄用戶點擊：文章、產品、課程、看診連結 |
| **驗收標準** | 後台可看各 CTA 點擊次數（為 A/B 鋪路） |

### DX-016｜免責／個人化 Disclaimer 模板

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | AI／Prompt |
| **描述** | 推價格、療程、保健品時自動附加溫和 disclaimer（依個人狀況可能不同） |
| **驗收標準** | 與初訪「溫和 Disclaimer」一致 |

### DX-017｜回覆預覽含 CTA 區塊

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 後台／UX |
| **描述** | 審核模式時，預覽區塊化顯示：正文／CTA 卡片／文章連結 |
| **驗收標準** | 客戶審核时可一眼看到「有沒有帶到銷售 loop」 |

## E3 漏斗流程

### DX-020｜「文章優先」策略開關

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 流程 |
| **描述** | 新對話或新議題時，優先推送最相關文章（非直接推課程） |
| **驗收標準** | 客戶確認的策略：「我是希望先丟文章」 |

### DX-021｜延遲主動追訊（排程）

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 流程／LINE |
| **描述** | 發送文章後，依設定（例：4h / 24h）自動發關心訊息＋下一 CTA |
| **驗收標準** | 可設定延遲時間；訊息計入 LINE 配額（客戶不介意，月購 3 萬則） |
| **依賴** | DX-020 |

### DX-022｜追訊內容模板（第二觸點）

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 流程／內容 |
| **描述** | 第二則訊息模板：關心＋課程／保健品／檢查／看診擇一或多選 |
| **驗收標準** | 至少 3 種模板可後台編輯 |

### DX-023｜免費試看 → 付費課程漏斗

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 流程 |
| **描述** | 流程：免費小講／第一章 → 產品頁 → 報名；AI 可在各節點推下一階 |
| **驗收標準** | Demo 可跑完「試看 → 付費」路徑 |
| **備註** | 客戶有獨立 landing、免費課程觀看區 |

### DX-024｜用戶 funnel 階段狀態

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 後台 |
| **描述** | 每位 LINE 用戶標記：已收文章 / 已追訊 / 已點課程 / 已點產品 等 |
| **驗收標準** | 避免對已完成轉換者重複推同一 CTA |
| **依賴** | DX-021、DX-015 |

### DX-025｜漏斗中斷再啟動

| 欄位 | 內容 |
|------|------|
| **優先級** | P2 |
| **類型** | 流程 |
| **描述** | 用戶 N 天未互動後，可觸發再關心（新文章或優惠） |
| **驗收標準** | 後台可設天數與模板 |

### DX-026｜消講／直播聊天室導購（探索）

| 欄位 | 內容 |
|------|------|
| **優先級** | P2 |
| **類型** | 整合 |
| **描述** | 預錄消講側邊聊天室：小助手自動回問題＋導購連結 |
| **驗收標準** | 需求確認即可；參考 Ken 真實消講＋客戶預錄現況 |
| **備註** | 客戶自評流程「怪怪的」但可接受預錄 |

## E4 主題路由

### DX-030｜意圖分類：可售 vs 需看診

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI |
| **描述** | 自動分類：保健建議可答、直接醫學診斷拒答、需個人化轉看診 |
| **驗收標準** | 分類錯誤率可人工標記修正 |

### DX-031｜主題 tag：多囊

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | 路由 |
| **描述** | 多囊相關問題 → 多囊文章、多囊課程、相關保健品、檢查 |
| **驗收標準** | **Demo 主情境** |

### DX-032｜主題 tag：失眠／甲狀腺／更年期／減重

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 路由 |
| **描述** | 各主題有獨立素材包；更年期等可能不導保健品但導課程／看診 |
| **驗收標準** | 後台可配置「此主題啟用哪些 CTA 類型」 |

### DX-033｜減重議題交叉銷售

| 欄位 | 內容 |
|------|------|
| **優先級** | P1 |
| **類型** | 路由 |
| **描述** | 減重問題可串課程、保健品、看診（客戶稱「裡面全部都在賣」） |
| **驗收標準** | 與 DX-012 loop 一致 |

### DX-034｜一文章多問題匹配

| 欄位 | 內容 |
|------|------|
| **優先級** | P0 |
| **類型** | AI／搜尋 |
| **描述** | 同一篇文章可被多種問法觸發（語意相似即可） |
| **驗收標準** | 客戶要求：「三個、四個問題都適合丟同一篇文章」 |
| **依賴** | DX-003 |
