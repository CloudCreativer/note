## 跨客戶共通主題

### 1. 回覆 ≠ 成交

- **大象醫師**：回答正確但不會賣 → 需銷售 funnel（文章 → 追訊 → 課程／產品／看診）
- **Yevonne**：聊天不等於名單 → 需 SOP 導向**電話**，且轉換率可量測

### 2. 「像 AI」是流失主因

- 大象：尚可接受知識型回覆，但缺**下一步 CTA**
- Yevonne：純自動／選項式 → 用戶**直接不回**；需 ChatGPT 式情緒＋真人穿插

### 3. 產品能力缺口（依優先）

| 能力 | 大象 | Yevonne |
|------|:----:|:-------:|
| 銷售 loop／多 CTA 串接 | ● | ○ |
| 文章 deep link＋主動追訊 | ● | ○ |
| 多步驟 SOP 流程 | ○ | ● |
| 情緒化對話語氣 | ○ | ● |
| 人工接手／半自動 | ○ | ● |
| 轉換率 dashboard | ○ | ● |
| 多人共用官方 LINE | ○ | ● |
| Meta／IG 整合 | ○ | ● |
| LINE 卡片 UX（誤觸） | ○ | ● |

● = 該客戶 P0　○ = 相關但非首要

### 4. 與成交流程的對應

- 兩位客戶皆已過「興奮期」試用，進入**猶豫期／具體需求期**
- 大象：興奮點在「銷售閉環補上就能付費」
- Yevonne：猶豫點在「AI 轉換率能否取代 50% 人工名單率」

## 建議產品 Roadmap 切片

### 近期（呼應 MVP 回饋）

1. **銷售／CTA 模組**：產品連結、圖片、文章、延遲追訊
2. **SOP 流程引擎**：多步驟狀態、欄位收集、人工接管
3. **Tone 2.0**：情緒價值模式（交友／療癒類）
4. **多人協作**：同一官方 LINE 多帳號登入
5. **LINE 卡片觸控區設定**

### 中期

- Meta／IG DM 整合
- 轉換 funnel 分析（電話率、導購點擊、A/B）
- 付費會員內容排程推送
