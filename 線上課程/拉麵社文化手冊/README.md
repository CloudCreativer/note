# 拉麵社文化手冊

> The Culture Handbook of RAMEN Community · 黃晨皓（阿金 KIM）· 2023.11 創立

## Insight

拉麵社（2023.11 創立，5000+ 人）文化核心：**人如拉麵機**——輸入與輸出需一致、可預測（Ray Dalio 啟發）。**RAMEN 模型**：Reflection/Radiance、Authenticity、Mutuality、Evolution、Network。三大支柱：互相幫助、樂於分享、成長學習。社群獨立經營、稱成員「老闆/同學」；**真誠是選擇不是技巧**。互動節奏可潛水、無「落後」；行為準則鼓勵真誠輸出、禁止說教與引戰。術語含：共用表單、支票、四級人、水庫/清淤泥/炸水庫、大廳/討論串/接龍。

## Translation

| 手冊概念 | 我的理解 |
|----------|----------|
| 拉麵機 | 內在運作邏輯要穩定，才能可預測地「輸出」人生 |
| 老闆精神 | 對自己人生有決策權，不是一定要創業 |
| 24 小時不打烊 | 可自由選距離：看、潛水、追連載都行 |
| 四級人 | 提問→解法→執行→接受結果→再提問 |
| 水庫隱喻 | 先顧自己能量，才能長期給出 |
| 價值觀/故事/感受/行動版 | 對外介紹拉麵社的四套話術 |

## Why

| 以前覺得 | 現在覺得 |
|----------|----------|
| 社群=聊天群 | 可設計文化手冊、行為準則、術語系統 |
| 要常互動才不落後 | 拉麵社允許潛水，沒有比賽感 |
| 真誠=完美人設 | 真誠是選擇，可以不完美 |
| 社群靠創辦人 | 從參與到貢獻，社群屬於每個人 |
