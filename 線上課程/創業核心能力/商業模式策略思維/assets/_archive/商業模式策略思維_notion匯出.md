### 單元 2 - 商業模式畫布 策略思維工具

## 🎯 一、商業模式的核心概念

### 💡 定義

商業模式（Business Model）描述一個組織如何 **創造、交付、傳遞價值（Create, Deliver, Capture Value）**。

### 🧩 功能與意義

- 說明「**企業如何賺錢**」與「**為誰創造價值**」。
- 連結產品、客戶、市場與資源之間的邏輯關係。
- 是企業競爭力的核心來源。

---

## 🧭 二、商業模式的九大元素（The 9 Elements）

| # | 元素名稱 | 定義 | 核心問題 |
| --- | --- | --- | --- |
| **1️⃣ 價值主張（Value Proposition）** | 企業提供的產品或服務的獨特吸引力。 | 我們為客戶解決了什麼問題？為何選擇我們？ |  |
| **2️⃣ 目標客群（Customer Segments）** | 企業選擇服務的具體客戶群體。 | 我們的主要客戶是誰？他們有何共同特徵？ |  |
| **3️⃣ 顧客關係（Customer Relationships）** | 建立與維護客戶關係的方式。 | 我們如何維持客戶滿意度與忠誠度？ |  |
| **4️⃣ 通路（Channels）** | 將產品／服務傳遞給客戶的方式。 | 我們透過哪些通路銷售與傳遞價值？ |  |
| **5️⃣ 收益流（Revenue Streams）** | 企業的收入來源與盈利模式。 | 我們如何產生收益？可否多元化？ |  |
| **6️⃣ 關鍵活動（Key Activities）** | 支撐商業模式的核心任務或作業。 | 企業最重要的日常運作是什麼？ |  |
| **7️⃣ 關鍵資源（Key Resources）** | 運作商業模式所需的資產。 | 我們需要哪些人力、技術、資金或品牌？ |  |
| **8️⃣ 關鍵夥伴（Key Partnerships）** | 為創造與交付價值所需的合作夥伴。 | 我們與誰合作能提升效率或市場力？ |  |
| **9️⃣ 成本結構（Cost Structure）** | 支撐整體商業模式的成本與支出。 | 哪些是主要成本？如何控制？ |  |

---

## ⚙️ 三、商業模式的結構邏輯：**右邊「價值創造」 vs 左邊「價值支撐」**

### 🔸 右側（橘色區塊）：**價值創造與收益**

包含：

1. 目標客群
2. 顧客關係
3. 通路
4. 收益流
5. 價值主張（中心）

👉 代表「**創造價值、交付價值、產生收入**」的區域。

### 🔹 左側（藍色區塊）：**價值支撐與成本**

包含：

1. 關鍵活動
2. 關鍵資源
3. 關鍵夥伴
4. 成本結構

👉 代表「**支援與投入資源**」的區域。

---

### 📊 對應邏輯關係圖

| 區塊 | 意義 | 對應功能 |
| --- | --- | --- |
| 右邊（價值創造） | 收入與市場成長面 | **加項（＋）** |
| 左邊（價值支撐） | 成本與資源投入面 | **減項（－）** |
| 🧮 利潤關鍵 | 「右邊的價值」 > 「左邊的成本」時，企業才能持續運行。 |  |

## 🧩 四、設計商業模式的建構順序（Recommended Flow）

> 依照老師的策略邏輯，設計時可依此順序進行：
> 

| 步驟 | 對應元素 | 說明 |
| --- | --- | --- |
| ① | **價值主張** | 從中心出發，明確定義企業要解決的客戶問題。 |
| ② | **目標客群** | 找出誰最需要你的價值主張。 |
| ③ | **顧客關係** | 決定如何建立與維持長期連結。 |
| ④ | **通路** | 思考如何最有效地傳遞產品與服務。 |
| ⑤ | **收益流** | 確立收入模式與金流邏輯。 |
| ⑥ | **關鍵活動** | 支撐價值主張的核心作業。 |
| ⑦ | **關鍵資源** | 提供活動所需的資源配置。 |
| ⑧ | **關鍵夥伴** | 確定合作夥伴與外部支援體系。 |
| ⑨ | **成本結構** | 整合所有活動、資源與夥伴的成本。 |

> ✅ 設計邏輯：從價值 → 客戶 → 活動 → 成本
> 
> 
> 先想「為誰創造價值」，再思考「如何實現與維持」。
>

## 🎯 一、什麼是價值主張（Value Proposition）

### 📘 定義

> 「價值主張是企業向客戶傳遞的核心價值，是讓客戶選擇你的關鍵理由。」
> 

它回答的核心問題是：

- 為什麼客戶要選擇我們而不是別人？
- 我們的產品／服務到底解決了什麼問題？
- 它能為使用者帶來什麼具體好處？

### 🌟 意義

- 是企業在市場中「**脫穎而出**」的關鍵因素。
- 決定了企業的競爭定位與品牌吸引力。
- 是整個商業模式的**中心軸**，連結左右兩側的資源與客戶。

---

## 🧠 二、策略思維：價值識別 → 創造 → 優化

| 階段 | 目標 | 核心問題 |
| --- | --- | --- |
| **識別（Identify）** | 發現市場中的真實需求與痛點。 | 客戶真正想要的是什麼？ |
| **創造（Create）** | 根據需求設計出解決方案或產品。 | 我們能提供什麼獨特價值？ |
| **優化（Refine）** | 不斷調整產品與訊息以提升認同感。 | 我們的價值有被理解與體驗到嗎？ |

> ✅ 成功的價值主張＝明確、具吸引力、可驗證。
>

## 🧩 三、案例分析：製產科技（SmartTrack）

### 🏭 背景

- 公司名稱：**製產科技**
- 產品：**「健康守護者」智能健康手環**
- 功能：心率監測、水分分析、活動追蹤

### ⚠️ 問題

- 產品技術優秀，但市場反應冷淡。
- 消費者「不明白它如何改善健康」。
- **核心問題：價值主張不清晰。**

### 💡 解決思路

創辦人林小姐重新檢視市場與定位，啟動「價值識別」流程：

| 問題 | 對應思考 |
| --- | --- |
| 用戶不理解產品價值 | → 檢查「傳遞訊息」是否貼近痛點 |
| 技術導向過強 | → 重新聚焦「實際效益」 |
| 缺乏市場對話 | → 與使用者訪談、挖掘需求與感受 |

> 🧭 結論：技術創新若沒有明確價值主張，就無法形成市場競爭力。
>

### 📋 製產科技的價值主張表（範例）

| 客戶檔案 | 對應產品價值 |
| --- | --- |
| **任務：** 希望能方便追蹤日常健康狀態。 | **產品：** 智能健康手環「健康守護者」。 |
| **痛點：** 不懂如何從數據中得出可行建議；資訊太複雜。 | **痛點緩解：** 提供簡潔的健康建議與行動提示。 |
| **收益：** 想看到「健康改善的具體變化」。 | **收益創造：** 每週健康報告與趨勢分析，讓用戶看到成效。 |

> 💬 成果：重新聚焦後，產品行銷語言更清楚、銷售表現回升。

## 🧩 五、實作指引：如何建立你的價值主張表

1. **選定明確的客戶群。**
    - 使用Persona描述典型客戶輪廓。
2. **定義任務、痛點與收益。**
    - 訪談、觀察、收集用戶故事。
3. **對應產品功能與價值。**
    - 釐清產品如何緩解痛點、創造好處。
4. **檢查對應關係是否明確。**
    - 每個痛點至少對應一個解決方案。
5. **持續測試與優化。**
    - 根據市場反饋調整語言與行銷主軸。

| 🔄 持續優化是競爭關鍵 | 隨市場變動調整產品與訊息。 |
| 🧩 工具推薦 | 使用 **Value Proposition Canvas** 進行策略檢視。 |

> 一句話總結：
> 
> 
> 沒有清晰價值主張的產品，就算技術再好，也難以讓市場買單。
> 

### 單元2 - 目標客層 | 解鎖客群定位的力量

## 一、目標客群的核心概念

### 💡 定義

目標客群（Customer Segments）指企業**選擇服務的具體客戶群體**。

透過劃分與分析，企業可對不同客群設計差異化的策略。

### 🧭 為何重要？

- 每個客群的 **需求、價值認知、購買動機** 均不同。
- 不同客群需對應不同的：
    - 行銷策略
    - 定價策略
    - 產品與服務設計策略
- **精準識別與聚焦** 是企業資源運用效率的關鍵。

---

## 二、關鍵能力：市場細分能力（Market Segmentation Competence）

### 🎓 學習目標

培養三項核心能力：

1. **識別**潛在客群
2. **劃分**差異群體
3. **制定**對應策略（產品、通路、行銷）

### 🧩 四個常用細分維度

| 維度 | 說明 | 範例 |
| --- | --- | --- |
| **地理細分（Geographic）** | 區域、氣候、城市類型 | 北部都會 vs 南部鄉鎮 |
| **人口細分（Demographic）** | 年齡、性別、收入、教育 | 高收入上班族 vs 學生族群 |
| **心理細分（Psychographic）** | 價值觀、生活型態、態度 | 環保意識強的族群 |
| **行為細分（Behavioral）** | 購買習慣、使用頻率、品牌忠誠度 | 每月固定購買清潔用品者 |

## 一、顧客關係的重要性

### 💡 為何企業需要重視顧客關係？

### 單元 4 - 通路 | 開啟客戶接觸的大門

## 一、通路的核心概念

### 💡 定義

通路（Channels）是企業用來將產品或服務 **傳遞給客戶** 的管道。

它包含從銷售、配送到售後互動的所有接觸點。

![Hahow for Business00_37.png](%E5%95%86%E6%A5%AD%E6%A8%A1%E5%BC%8F%E7%AD%96%E7%95%A5%E6%80%9D%E7%B6%AD/Hahow_for_Business00_37.png)

### 🧭 企業為何要重視通路設計？

- 影響產品到達顧客的速度與便利性
- 決定顧客體驗與購買意願
- 是價值主張落地的重要橋樑

> 🔑 關鍵思維：
> 
> 
> 「好的產品若沒有被看見，就無法產生價值。」
> 

---

## 二、通路策略的兩大挑戰

| 挑戰 | 說明 |
| --- | --- |
| **1️⃣ 多元通路選擇** | 現代企業面對多樣化通路，如實體店面、電子商務、社群媒體、合作平台等。 |
| **2️⃣ 核心通路聚焦** | 在眾多管道中，企業必須找到最能「有效覆蓋目標客群」的核心通路。 |

## 三、案例分析：香榭咖啡（Champsé Coffee）

- 位於繁忙商圈的一家在地咖啡店。

## 五、實作應用：香榭咖啡的通路重構策略

### 🎯 目標

3. 強化社群媒體經營，分享咖啡故事與製作影片。
4. 推出「線上訂購、店內自取」的混合服務模式。

### 🚀 成效

- 擴大市場觸及 → 從單一地區到全國客群。
- 維持品牌溫度 → 顧客仍可感受到手作與文化精神。
- 提升顧客滿意度與重複購買率。

### 單元 5 - 收益流 | 收益模型創新的魔法

> 「多元化收益流」是企業財務穩定與永續發展的關鍵。

## 🧩 一、收益流的重要性

- 每個企業都需要了解自己有哪些「收入來源」。
- 收益來源越單一，越容易受市場變化影響。
- 多元化收入能提升「財務穩定性」與「抗風險能力」。

## 💡 二、收益流的常見類型

講者提到幾種：

- 商品銷售收入（product sales）
- 服務收入（service income）
- 廣告收入（advertising）

## 🧭 五、實作步驟：如何建立你的關鍵活動流程圖

![Hahow for Business03_39 (2).png](%E5%95%86%E6%A5%AD%E6%A8%A1%E5%BC%8F%E7%AD%96%E7%95%A5%E6%80%9D%E7%B6%AD/Hahow_for_Business03_39_(2).png)

1. **盤點所有主要業務活動**
    - 例如：研發、生產、銷售、客戶服務。
2. **明確定義每個流程階段的責任部門**

4. **視覺化流程**
    - 可使用流程圖、泳道圖（Swimlane Diagram）或 Notion 表格呈現。
5. **持續優化與回饋**
    - 定期檢視瓶頸點，找出可自動化或簡化的部分。

## 💡 一、關鍵資源的定義

> 關鍵資源（Key Resources）是企業為了創造價值、維持運營、與客戶互動及獲利而必須投入的核心資產。

### 📊 資源類型

| 類型 | 說明 | 例子 |
| --- | --- | --- |

## 💡 一、成本結構的定義

### 📘 定義

> 成本結構（Cost Structure）是企業在運營過程中，
> 
> 
> 為提供產品或服務而發生的各項成本組成與比例。
> 

它反映了企業的**經營模式與策略重點**，

也是企業能否穩定獲利的關鍵基礎。

---

## 🧩 二、成本的主要類型

| 類型 | 說明 | 例子 |
| --- | --- | --- |
| **固定成本（Fixed Costs）** | 不隨產量變化而改變的支出 | 房租、設備折舊、薪資 |
| **可變成本（Variable Costs）** | 會隨生產或銷售量變動的支出 | 原物料、包裝、物流 |
| **半變動成本（Semi-variable Costs）** | 固定與變動兼具的支出 | 電費、維修費 |
| **隱藏成本（Hidden Costs）** | 難以量化但實際存在的開銷 | 人員流動、溝通成本 |

> ✅ 重點： 成本結構並非只是數字，而是企業運作的「鏡像」——
> 
> 
> 它揭示了企業把資源投入在哪裡，核心價值靠什麼支撐。
> 

---

- 找出主要成本來源與潛在浪費

### 🔍 為何重要

| 面向 | 說明 |
| --- | --- |
| 💰 提高利潤率 | 有效控成本能直接提升盈餘 |
| ⚙️ 維持競爭力 | 降低成本可讓價格更具市場優勢 |
| 🧭 策略調整 | 清楚的成本結構能支持精準決策 |

| 缺乏人為判斷 | ➜ AI 結果應交由人員進行策略思考與決策。 |
