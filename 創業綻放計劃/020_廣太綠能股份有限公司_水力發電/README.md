# 020 廣太綠能 小水力發電

## 一句話摘要

廣太綠能（花蓮）專注 10 kW 以下低落差、低流量 Crossflow 小水力，鎖定工廠餘壓、污水廠放流等傳統認為無經濟規模的場域；模組化安裝、不影響廠區運轉。

## 評審提問與回答

### Q1. 日月光案例的投資多久回本？機器本身也要用電吧？

（回本為評審關切，團隊以案場條件說明。）日月光單機約 **60–100 萬**（依能量大小），管內餘壓約每小時 **3.75 kWh**。廠內用電通常 **>5 元/度**、約 **20 小時/日、365 天**，不需另建水閘，回本約 **3–5 年**；水力機組壽命可逾百年。供應鏈用東元馬達、新望 Inverter 等，利於維運與併網。

### Q2. 政府／技術難度在哪？若真的能賺錢，別人會不會很快抄你們？

成功約 **90% 在選址**（九大關鍵：取水回水、不影響廠區、併網或儲能、維運等）。我們從 2023 年起用 GIS、無人機 RTK、自研算力軟體估算水頭水量與回本，不再只靠人工探路。台灣精密加工可持續降本；我們也協助同業維修，看到部分機組零件難維護的問題。壁壘是**案場 know-how + 工法 + 供應鏈**，不是單一葉輪。

### Q3. 台灣已設一些案場，進入當地有什麼挑戰？商業模式是什麼？像日月光一個廠可以用你們多少設備？

商業模式：**銷售設備** + 整廠輸出建置，目標自有電廠。已掃描約 **2,000+** 潛在案場。日月光除已裝案場外，**K7、K8、K27** 等持續評估（簡報／現場說法）。

### Q4. 廢水廠已有完整設備，你們後加設備，會不會讓整體廢水設計大變動？

我們原則是**不影響廠區運作**、不必停機即可安裝。花蓮水資源中心案即利用處理後放流約 **2 m** 水頭創能。取水／回水設計是關鍵技術之一，須符合九大條件才能取得客戶信任。
