# 020 廣太綠能股份有限公司 水力發電

## 一句話摘要

廣太綠能股份有限公司的主題聚焦於水力發電，目前重點在釐清日月光案例的回收年限與損益平衡點、技術壁壘、台灣案場痛點與可複製性，以及廢水結合設備後的技術與商業變化。

## 重點整理

### 產品與技術方向

- 水力發電方案
- 可能應用於工業用水／廢水流程中的能量回收
- 可能包含水力設備、控制系統與場域整合工程

### 初步觀察

- 回收年限與損益平衡是客戶是否採用的關鍵決策指標
- 技術壁壘不只在發電設備，還包括場域整合、穩定運轉與維運能力
- 台灣案場若痛點可標準化，才有機會形成可複製商模
- 廢水水質與流量波動大，通常會影響設備效率、壽命與維護成本

## 關鍵結論

- 這份筆記目前屬於水力發電案場經濟性與複製策略問題整理階段

- 可複製性需要建立在標準化設計、安裝與維運流程上
- 廢水場景下的設備耐受性與效能穩定度是重要風險點

## Q&A

### 日月光案例：幾年回收、損益平衡點

- 問題：日月光案例的回收年限與損益平衡點是多少？
- 可追問方向：
  - 初期建置成本（CAPEX）與年運維成本（OPEX）各是多少？
  - 回收期計算是否包含電價、補助、折舊與停機風險？
  - 在不同流量／水質條件下，回收期是否明顯變動？
  - 是否有實際運轉年化發電量與節費數據可佐證？

### 技術壁壘

- 問題：技術壁壘是什麼？
- 可追問方向：
  - 壁壘在水輪機設計、控制演算法、系統整合，還是維運能力？
  - 是否有專利、專有製程、或難以複製的案場 know-how？
  - 與傳統水力或其他能量回收方案相比，差異化指標為何？
  - 競品是否容易透過價格競爭取代？

### 台灣案場痛點與可複製性

- 問題：台灣案場的痛點是什麼？是否可複製？
- 可追問方向：
  - 痛點在法規、場地限制、流量不穩、工程介面，還是客戶決策流程？
  - 哪些模組可標準化，哪些仍需大量客製？
  - 成功案場能否快速複製到其他產業（半導體、化工、食品）？
  - 目前案場複製週期與成功率如何？

### 廢水加上設備後有什麼變化

- 問題：廢水再加上設備，會有什麼樣的變化？
- 可追問方向：
  - 廢水中的顆粒、腐蝕性、溫度是否影響設備壽命與效率？
  - 是否需要前處理設備才能穩定運轉？
  - 導入後對廢水處理流程、能源回收與維護成本有何影響？
  - 是否會新增法規合規、檢測或安全管理要求？
