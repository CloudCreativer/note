# 014 台灣微電網科技公司 微電網設計

## 一句話摘要

台灣微電網科技公司的主題聚焦於微電網設計，目前重點在釐清微電網優勢、日本合作細項，以及「中信電工有專利、公司本身無專利」所代表的技術與商業風險。

## 背景

- 主題公司：台灣微電網科技公司
- 主題方向：微電網設計
- 已知線索：日本合作；中信電工專利；公司目前未見自有專利
- 現階段目標：建立產品定位、合作結構與專利風險理解

## 重點整理

### 產品與技術方向

- 微電網整體設計
- 可能涵蓋能源管理、儲能調度、電力品質與備援架構
- 可能面向工廠、園區、商業建築或離網場景（待確認）

### 初步觀察

- 微電網價值通常來自「供電穩定 + 節費 + 減碳 + 韌性」，需看公司主打哪一項
- 日本合作若存在，通常涉及技術驗證、案場導入或市場通路
- 若公司無自有專利而關鍵專利在中信電工，需釐清授權與商業自主性

## 關鍵結論

- 這份筆記目前屬於微電網方案定位與合作/IP 風險整理階段
- 後續應優先補齊微電網優勢的量化依據與日本合作內容
- 專利不在公司名下不一定是問題，但必須確認可持續授權與商業化權利

## Q&A

### 微電網優勢

- 問題：微電網優勢是什麼？
- 可追問方向：
  - 優勢是降低電費、提高供電穩定、削峰填谷，還是減碳合規？
  - 相較傳統用電與單一儲能方案，效益差異有多大？
  - 是否有導入前後的量化數據（停電風險、電費、碳排）？

### 日本合作細項

- 問題：日本合作的細項是什麼？
- 可追問方向：
  - 合作對象是企業、研究機構、系統整合商，還是政府計畫？
  - 合作內容是技術授權、共同開發、案場建置，還是代理銷售？
  - 合作地區、案場規模、時程與 KPI 是什麼？
  - 日本合作對台灣市場或海外擴張的策略意義為何？

### 中信電工專利，公司沒有專利

- 問題：中信電工有專利，但公司沒有專利，代表什麼？
- 可追問方向：
  - 公司是否有正式授權、獨家使用權或共同開發協議？
  - 專利是否涵蓋核心技術，還是僅部分模組？
  - 若授權終止或調整，對營運與客戶交付的影響有多大？
  - 公司是否有非專利護城河（系統整合、資料、案場經驗、維運能力）？
