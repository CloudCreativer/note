# 014 台灣微網 微電網設計

## 一句話摘要

台灣微網科技以微電網控制軟體與電力系統設計能力，協助企業在停電時約 100 ms 內切換至微電網供電；已通過台電 1 MW 以上微電網標準驗證（三家之一）。

## 評審提問與回答

### Q1. 能做微電網的廠商不少，你們的特殊優勢在哪裡？

我們跟國家型計畫團隊累積十幾年實地技術（含 APEC 智慧電網獎等）。優勢不只是我們一家：我們串聯上下游設備商，知道各廠設備特性，扮演**整合者**——許多設備商不知道怎麼把系統用到微電網場域，我們負責整體設計與整合。也會帶動更多電力相關技術人員一起參與長期維運。

### Q2. 簡報提到宮古島建置案場，能否說明前因後果與具體內容？

宮古島案與創辦人過去在日本合作的 SI 有關。我們海外模式是：先在台灣做軟硬整合驗證，再系統輸出海外。電力領域台灣有很好設備、電力電子與軟體能力，但過去多賣零組件而非整套系統；我們在台灣驗證後，再依當地需求調整，導入會比較快。宮古島反映此策略：掌握台灣供應鏈與設計能力，海外需求圍繞台灣網從設計到實施。

### Q3. 團隊來自中興電工，但貴公司沒有專利；創辦人／中興電工有大量微電網專利，營運上如何處理這層關係？

與中興電工長期合作，有多項技術與授權關係。公司**近兩年**的微電網控制軟體是自主開發，在軟體定義與從上到下設備整合上是公司這兩年投入資源的成果。專利與技術授權細節以合約為準。

### Q4. （補充，非提問）技術差異與核心能力

我們偏**應用整合**，單一硬體規格未必最強，強項是過去經驗能快速理解並滿足客戶。核心能力三塊：（1）**電力系統設計**——業界多賣設備，少數能做設計；（2）從設計、整合測試到長期維運託管；（3）持續投入 IEC 61850、資安等認證，讓地端控制系統可管理更多風險場景。
