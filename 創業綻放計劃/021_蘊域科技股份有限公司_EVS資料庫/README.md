# 021 蘊域科技 EVS 跌倒偵測

## 一句話摘要

蘊域科技（逐字稿亦作「音樂科技」）以 EVS 事件式視覺感測跌倒與異常行為，硬體 EVS5 切入機構累積資料，再推 SDK 授權；已與新光保全等合作。

## 評審提問與回答

### Q1. 若很多單位、醫院都要用，其他競爭者會不會很快跟上？專利也不好申請的樣子。

我們累積約六年、全球最大規模之一的 **EVS 人類行為資料庫**（早期與 Prophesee 獨家合約）。Sony、Prophesee 做**硬體**，我們做**場域應用演算法**；是合作關係。台灣在獨家期內幾乎無同技術競爭者；大廠從談判到產品至少約**三年**時間差。

### Q2. 賽道領先者之外，國內外還有沒有其他競爭者？領先狀況如何？

同上。Sony 近兩年積極推 EVS，技術來源亦與 Prophesee 相關，但與我們非直接同業競爭。

### Q3. 有鎖定個人家用或機構市場嗎？

全台醫療／長照示範與出貨場域約 **50+**。產品可像 iCam 裝在家中，Wi‑Fi 對接手機；一台可通報多台手機。機構版則納入既有護理／照護系統（MQTT、WhatsApp 等），新光保全 SKS 亦直接整合。

### Q4. 物理感測到底用什麼？是 CMOS 換讀法，還是不同原理？

是**全新技術**：透過**光折射**成像，只記錄畫面變化，非 CMOS 連續攝影。符合歐盟 **GDPR**、美國 **HIPAA** 等隱私友善方向（現場說法）。

### Q5. 能否用 30 秒講清楚：從預防老人跌倒到通報醫療機構的完整情境？

**居家：** 裝機、插電、Wi‑Fi 對接手機即可用；跌倒等異常會通報至多支手機。**機構：** 訊號納入照護系統即時通報。**新光：** 納入 SKS，智慧城展等亦展示整合。

### Q6. 公司行政管理、人員配置，是否有勞健保或法遵相關問題？

曾發生勞健保轉換過程中的錯誤，**已釐清處理**（現場簡短說明，細節以公司公告為準）。

### Q7. 反應速度 1 ms 怎麼定義？如何防止跌倒？

技術上幀率可達很高（現場稱最高約 **10,000 fps** 量級，即毫秒級）。實際人類行為感測約用 **62 fps**；其他應用（如為人保虛擬鍵盤）約 **2.5 fps**。跌倒場景取決於硬體與算力配置，非一律跑滿 1 ms。
