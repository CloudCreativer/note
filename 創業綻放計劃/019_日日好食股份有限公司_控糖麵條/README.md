# 019 日日好食 控糖麵條

## 一句話摘要

日日好食主打控糖高蛋白快煮麵，以 D2C 掌握消費者數據，三年營收成長約 1,200%，並規劃血糖資料庫與美國布局。

## 評審提問與回答

### Q1. 這包麵賣多少錢？要煮多久？除了麵以外還會做麵包、包子等其他產品嗎？

一袋均價約 **239 元**／包，一碗約 **80 多元**，比一般乾拌麵溢價約 **25–50%**，高蛋白機能品在台灣市場可接受。現行水滾後約煮 **7 分鐘**，二代目標約 **5 分鐘**。其他品項：麵包較難發酵；**餅乾、披薩皮、水餃皮**等常溫品可行，目前專注外銷常溫、快變現品項。

### Q2. 在地生產才能贏得工廠配合，技術有 barrier 不易被 copy，但台灣很會抄，這對未來執行很重要——如何防抄？

我們與**多家工廠**合作，參數是我們的 know-how，簽**保密協議**，每次調整都以量產為前提。保護靠配方分層隔離 + 工廠 NDA，不是單一工廠獨占產線。

### Q3. 主軸是控糖高蛋白，但簡報食品添加物不少，好吃常靠添加物，你們怎麼選擇讓它還健康一點？

**麵條沒有任何食品添加物**，成分很乾淨。您看到的添加物在**醬料**：醬料非搭麵必備，為**出口**（美國航運約兩個月）穩定度，依法規添加。

### Q4. 一碗麵為何能做到一片雞胸肉蛋白、高麗菜纖維、兩片吐司碳水？有什麼專利或技術？

現有**新型專利** 1 件；實品廠主要還靠公司內部**配方分層隔離** + 生產夥伴 NDA。六年前就開始做主食控糖高蛋白，累積主動找上門、持續回購的客群與數據。
