# 028 蘇菲營養生化股份有限公司 · 產能與國際化營運

## 30 秒看懂

蘇菲營養生化聚焦精準發酵蛋白質，主打可在既有發酵工廠（含啤酒廠）落地，降低建廠與擴產成本。評審 QA 核心在：專利與保護邏輯、批次一致性控制、毛利率偏低如何改善、是否可量產更多產品，以及跨國團隊如何維持執行效率。

## 他們在做什麼

- 技術主軸：以精準發酵生產蛋白質原料。
- 生產策略：優先利用既有發酵產線（例如啤酒廠）導入，降低 CapEx。
- 商業定位：B2B，產品供應 + 技術授權/合作並行。
- 國際敘事：台灣作為技術中心，海外據點支援商務與合作拓展。

## 解決什麼問題

- 傳統蛋白來源受資源與供應鏈波動影響，且成本與永續壓力升高。
- 新蛋白技術常卡在放大量產與品質一致性。
- 國際擴張初期容易陷入「技術有、量產難、毛利低」三重挑戰。

## 怎麼賺錢

- 收入模式（逐字稿口述）：
  - B2B 原料供應。
  - 技術合作與授權。
  - 以既有發酵廠落地降低擴產門檻。
- 毛利邏輯：早期借廠生產毛利偏低，若自有產能提高，毛利有機會顯著改善。
- 擴品邏輯：以相同平台延伸多種蛋白或應用，提升產線利用率與產品組合毛利。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 已和在地啤酒廠進行合作推進，作為示範與產能放大路徑。
  - 有跨國合作與國際專案經驗，並以台灣為技術核心。
  - 已描述短中期營運與成長目標（含營收與擴產敘事）。
- 評審追問焦點：
  - 專利是否足夠支撐長期護城河。
  - 不同工廠條件下的批次一致性如何保證。
  - 目前毛利偏低是否可持續。
  - 國際團隊分工是否會拖慢決策。

## 評審提問與回答

### Q1. 專利保護到底在哪裡？只是配方還是整個平台？
- 團隊回答：保護重點不只單一配方，還包含菌種、製程與應用端的組合式策略。
- 正確性註記：`中信心`
  - 理由：方向合理，但專利號與國別布局待文件驗證。

### Q2. 為什麼一直提啤酒廠？它是客戶還是生產場域？
- 團隊回答：啤酒廠主要是可快速導入的發酵場域與合作夥伴，不等同最終唯一市場。
- 正確性註記：`中高信心`
  - 理由：商業邏輯清楚，且與成本優化敘事一致。

### Q3. 放大量產後，批次一致性怎麼控？
- 團隊回答：以製程控制與品質檢驗機制管理批次差異，並依不同場域條件做參數校正。
- 正確性註記：`中信心`
  - 理由：有流程概念，但缺具體 QC 指標與外部驗證數字。

### Q4. 毛利率大約 15% 偏低，怎麼改善？
- 團隊回答：早期借用外部產能導致毛利受限，後續透過規模化與自有/深度合作產能可提升。
- 正確性註記：`中高信心`
  - 理由：與製造業常見路徑一致，改善機制可自洽。

### Q5. 除了現在產品，能不能量產其他品項？
- 團隊回答：平台可延伸到其他蛋白與應用，具擴品潛力。
- 正確性註記：`中信心`
  - 理由：技術上可行，但商業時程與法規路徑尚未量化。

### Q6. 國際化團隊怎麼運作，不會失焦嗎？
- 團隊回答：以台灣為研發核心，海外據點負責商務拓展與合作推進。
- 正確性註記：`中信心`
  - 理由：組織方向清楚，仍需更多制度化節奏（KPI/決策機制）證據。

## 我學到的 3 件事

1. 精準發酵商業化早期，產能策略比單一技術更影響毛利。
2. 「借廠放大」是合理路徑，但要同步準備一致性與品質證據。
3. 國際化團隊的價值，在於把技術能力轉成跨國合作節點。
