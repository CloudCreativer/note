# 011 科淨能源股份有限公司 碳中和轉型

## 一句話摘要

科淨能源股份有限公司的主題聚焦於碳中和轉型，目前重點在釐清電解是否必須使用綠電、用台電是否仍能減碳、碳排大戶導入成本與轉化效率、美國技術與專利限制、商業模式（賣機器或訂閱）、客戶合作案例、日本合作策略，以及專利歸屬與設備定價。

## 背景

- 主題公司：科淨能源股份有限公司
- 主題方向：碳中和轉型
- 已知線索包含技術來自美國、可能有日本合作、專利分屬個人與公司、需釐清機器售價
- 現階段目標是建立技術、合規與商業模式理解，並整理碳排效益與 IP 相關問題

## 重點整理

### 產品與技術方向

- 碳中和轉型相關解決方案
- 可能涉及電解製程（如綠氫、碳捕捉或相關能源轉換）
- 技術來源與美國相關（細節待查）
- 面向碳排大戶的導入與數據轉化

### 初步觀察

- 「電解是否一定要用綠電」牽涉碳足跡計算方式與客戶 ESG 報告需求
- 碳排大戶導入通常有設備、整合、運維等轉換成本，需與節碳效益對照
- 美國技術若伴隨專利授權，可能限制商業化地域、模式或再授權
- 專利分屬個人與公司時，需釐清是否影響融資、授權與客戶信心
- 日本合作可能具技術驗證、市場進入或供應鏈意義

## 關鍵結論

- 這份筆記目前屬於碳中和方案與商業驗證問題整理階段
- 後續應優先釐清減碳邏輯（綠電 vs 台電）、導入成本與效率數據
- 商業模式（設備銷售 vs 訂閱）與專利結構將直接影響估值與擴張路徑
- 客戶案例與日本合作是驗證市場可信度的重要依據

## Q&A

### 電解是否一定要用綠電

- 問題：電解一定要用綠電嗎？用台電這樣有省到碳嗎？
- 可追問方向：
  - 電解產品／服務的具體用途（綠氫、廢氣處理、其他）？
  - 碳排計算採用哪種標準（電網排放係數、契約綠電、現場再生能源）？
  - 使用台電與使用綠電在碳減量上的差異為何？是否有量化比較？
  - 客戶的 ESG 或碳盤查是否要求特定電力來源？

### 碳排大戶導入成本與轉化效率

- 問題：碳排大戶導入是否有轉換成本？數據轉化效率有比別人高？
- 可追問方向：
  - 轉換成本包含設備、土建、停機、人員培訓、系統整合哪些項目？
  - 導入周期與回本假設為何？
  - 「轉化效率」的定義與對照基準（同業、傳統方案）？
  - 是否有可公開的效率提升百分比或案場數據？

### 美國技術、專利限制與商業模式

- 問題：技術來自美國，是否有專利限制？是賣機器還是訂閱？
- 可追問方向：
  - 美國技術是授權、合資、技術移轉，還是共同開發？
  - 專利限制涵蓋地域、用途、 sublicensing 或出口？
  - 收入來自一次性設備銷售、租賃、訂閱、節能分潤，還是混合模式？
  - 軟體／數據平台是否另計費？

### 實務與客戶合作案例

- 問題：有哪些實務與客戶合作案例？
- 可追問方向：
  - 客戶產業（鋼鐵、化工、電子、電廠等）與案場規模？
  - 合作階段是 PoC、試運轉、正式導入還是長期運維？
  - 可量化的節碳、節費或運轉數據為何？
  - 是否有可公開的客戶名稱或案例研究？

### 日本合作的策略意義

- 問題：日本合作基於什麼想法？或策略意義？
- 可追問方向：
  - 合作對象是企業、研究機構還是政府計畫？
  - 合作內容是技術驗證、代工、市場代理還是聯合研發？
  - 對台灣本業與海外擴張的具體幫助為何？
  - 是否為進入亞太或其他市場的跳板？

### 專利為什麼一個是個人一個是公司

- 問題：專利為什麼一個是個人一個是公司？
- 可追問方向：
  - 個人專利持有人是否為創辦人、技術負責人或美國合作方？
  - 公司是否已透過授權、轉讓或獨家實施權取得使用資格？
  - 是否影響投資人盡職調查或客戶合約中的 IP 保證？
  - 未來是否計畫將個人專利轉入公司？

### 機器一台多少錢

- 問題：機器一台多少錢？
- 可追問方向：
  - 單台容量、規格與價格區間？
  - 價格是否含安裝、維護、軟體與碳盤查服務？
  - 不同規模碳排大戶的 typical 採購台數與總專案金額？
  - 與競品或傳統方案的成本比較？
