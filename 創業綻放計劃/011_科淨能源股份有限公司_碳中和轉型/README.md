# 011 科淨能源股份有限公司 碳中和轉型

## 30 秒看懂

科淨能源主打把碳排與電解技術結合成可商業化的減碳方案，核心敘事是「減碳不只合規，還能轉成可計價價值」。評審關注焦點在：是否一定要綠電、碳排大戶導入成本與效率、技術來源與專利歸屬、以及到底賣設備還是做訂閱服務。

## 他們在做什麼

- 技術方向（依既有筆記與逐字稿可辨識片段）：
  - 以電解/碳處理技術切入碳中和；
  - 對象聚焦高排碳產業客戶；
  - 搭配系統化導入與後續運營。
- 商業敘事：從「減碳成本」轉為「碳資產與收益機會」。

## 解決什麼問題

- 碳排大戶面臨碳費與國際供應鏈減碳壓力。
- 傳統減碳方案導入周期長、資本投入高、效益不易量化。
- 企業缺可同時兼顧合規、成本與營收可能性的方案。

## 怎麼賺錢

- 可辨識商模（待二次校對）：
  - 設備/模組銷售；
  - 導入與運維服務；
  - 訂閱或處理量計費的持續收入。
- 商業關鍵：
  - 客戶導入回收期；
  - 電力來源與碳效益差異；
  - 技術授權條件是否影響擴張。

## 實績與進展（決賽現場）

- 逐字稿可辨識重點：
  - 技術主張：把二氧化碳轉為高價化學品/燃料，並搭配模組化放大。
  - 商業進展：提及中鋼、長春石化等 MOU 與海外合作接觸。
  - 收入敘事：設備/處理費/耗材與碳價值並行的 recurrent revenue 架構。
  - 國際布局：提及日本與其他海外市場拓展。

## 評審提問與回答

### Q1. 電解一定要用綠電嗎？用台電還算減碳嗎？
- 評審重點：碳效益邏輯是否成立。
- 團隊回答重點（逐字稿）：
  - 使用綠電時，減碳與綠色化學品價值更完整；
  - 若用一般電力，仍可能有減碳效果，但幅度與碳價值會受影響。
- 正確性註記：`中信心`

### Q2. 碳排大戶導入成本高嗎？轉化效率真的有優勢？
- 評審重點：客戶導入可行性與效益。
- 團隊回答重點（逐字稿）：
  - 強調可接既有場域導入，主打轉化率與能耗優化；
  - 導入仍有場域整合成本，但以回收期敘事支撐可行性。
- 正確性註記：`中低信心`（需補量化對照表）

### Q3. 技術來自美國，有沒有專利或授權限制？
- 評審重點：能否自由商業化與擴張。
- 團隊回答重點（逐字稿）：
  - 技術起源於美國，後續在台灣持續開發與專利布局；
  - 專利路徑含個人與公司，會以授權/轉入方式確保商業使用。
- 正確性註記：`中低信心`

### Q4. 你們是賣機器、做訂閱，還是混合模式？
- 評審重點：收入結構與可持續性。
- 團隊回答重點（逐字稿）：
  - 採混合模式：設備 + 處理服務 + 耗材/持續性收入。
- 正確性註記：`中信心`

### Q5. 日本合作的策略意義與實務案例？
- 評審重點：合作是否能帶來市場與驗證。
- 團隊回答重點（逐字稿）：
  - 日本合作用於驗證與海外拓展銜接，並作為國際市場敲門磚之一。
- 正確性註記：`中信心`

### Q6. 為什麼專利一個在個人、一個在公司？
- 評審重點：IP 風險與投資可行性。
- 團隊回答重點（逐字稿）：
  - 因技術演進與時序，存在個人與公司專利並行；
  - 公司會透過授權/整理讓商業權利可被檢核。
- 正確性註記：`中低信心`

### Q7. 機器一台多少錢？
- 評審重點：採購門檻與回本模型。
- 團隊回答重點（逐字稿）：
  - 價格依場域規模與模組配置而定，另有處理費模式降低初期門檻。
- 正確性註記：`中低信心`

## 我學到的 3 件事

1. 碳中和方案要成交，客戶最在意的是「真實減碳 + 財務可行」。
2. 電力來源（綠電或台電）會直接改變碳效益敘事與 ESG 認列。
3. 專利歸屬清晰度會大幅影響客戶與投資人信任。
