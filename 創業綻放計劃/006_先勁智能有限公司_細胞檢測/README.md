# 006 先勁智能有限公司 細胞檢測

## 30 秒看懂

先勁智能主打細胞檢測分析軟體平台，核心是把不同機台產生的高維訊號做標準化分析，降低人工作業門檻並提升一致性。評審 QA 焦點在：技術是不是純軟體、專利/股權與美商關係、資金與定價是否合理、以及 FDA 驗證邏輯是否成立。

## 他們在做什麼

- 產品形態（依逐字稿）：跨儀器的細胞檢測資料分析軟體/平台。
- 技術敘事：
  - 將訊號處理類比「語音辨識的降噪與辨識」；
  - 對不同儀器、不同操作者資料做標準化處理；
  - 提供快速一致的分析與報告輸出。
- 應用場景：細胞治療製程品管、生技研發、檢測流程數位化。

## 解決什麼問題

- 人工分析負擔重、訓練慢，且跨人員一致性不足。
- 不同機台/流程下，資料格式與品質不一致，難規模化。
- 在細胞治療等高要求場域，需要更高可追溯與標準化能力。

## 怎麼賺錢

- 逐字稿可辨識模式：
  - 教育版/企業版授權（按授權場景計）；
  - 按次或按批次服務（特定場景）；
  - 後續可能走更高單價的訂閱化方案。
- 逐字稿口述資料：
  - 公司與美商體系有歷史關係；
  - 累計募資級距約在千萬台幣以上（口述 2.2 千萬等說法，待正式資料確認）。

## 實績與進展（決賽現場）

- 逐字稿可辨識進展：
  - 已有跨機構合作與技術落地案例；
  - 提到與國際標準/社群合作經驗；
  - 產品已進入商業化初期定價與市場擴展階段。
- 評審關注重點：
  - 收費是否偏高、價值是否可被客戶驗證；
  - 法規有效性是「對照驗證」還是「正式核准路徑」；
  - 專利與公司架構是否影響未來商業化。

## 評審提問與回答

### Q1. 你們是軟體公司還是硬體整合公司？
- 評審重點：產品邊界與可擴張性。
- 團隊回答重點（逐字稿）：
  - 主體是可串接不同儀器的分析軟體平台；
  - 目標是標準化與自動化分析，不以自製硬體為主。
- 正確性註記：`中信心`
  - 原因：定位清楚，但仍需正式產品文件佐證交付方式。

### Q2. 專利與美商共有關係是什麼？
- 評審重點：權利歸屬與商業限制。
- 團隊回答重點（逐字稿）：
  - 公司與美商關係屬歷史架構/共同發展脈絡；
  - 現階段商業化由本地公司推進。
- 正確性註記：`中低信心`
  - 原因：逐字稿表述受 ASR 影響，需公司法與專利登記文件核對。

### Q3. 資金狀態與公司規模是否健康？
- 評審重點：是否能支撐法規與市場擴張。
- 團隊回答重點（逐字稿）：
  - 已有募資與策略資源支持，仍在成長階段。
- 正確性註記：`中低信心`
  - 原因：數字口述不夠穩定，需以正式財務資訊為準。

### Q4. 訂閱費為何這樣定價？
- 評審重點：價格與客戶價值是否匹配。
- 團隊回答重點（逐字稿）：
  - 依產品場景與使用模式分層；
  - 預計往更成熟的訂閱與授權模式上調整。
- 正確性註記：`中信心`
  - 原因：定價框架存在，但缺客戶 ROI 對照數據。

### Q5. FDA（或法規）有效性怎麼證明？
- 評審重點：合規敘事是否可落地。
- 團隊回答重點（逐字稿）：
  - 以既有核准檢測流程做對照評估（敏感度/特異度）來驗證模型有效性。
- 正確性註記：`中信心`
  - 原因：方法合理，但「法規核准」與「技術驗證」仍需清楚區分。

## 我學到的 3 件事

1. 細胞檢測 AI 的核心價值常在「標準化」而非單點準確率。
2. 生醫場景一定要把「模型效果」和「法規核准」分開說清楚。
3. 若商模同時有授權與按次收費，最終要用客戶 ROI 證明可持續性。
