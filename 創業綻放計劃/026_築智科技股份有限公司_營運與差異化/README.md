# 026 築智科技股份有限公司 營運與差異化

## 一句話摘要

築智科技股份有限公司的主題聚焦於商業競爭與營運模式，當前重點在釐清與競爭對手的差異、如何防止帳號共用、未來聚焦方向，以及公司營運是否需要實體空間。

## 背景

- 主題公司：築智科技股份有限公司
- 主題方向：競爭差異化、帳號機制、未來聚焦與營運型態
- 已知關注點：帳號共享風險與實體空間必要性
- 現階段目標：建立可執行的差異化與營運策略盤點框架

## 重點整理

### 產品與商業方向

- 需釐清核心產品型態（SaaS、平台、系統整合或混合）
- 需定義主要客群與價值主張，才能有效比較競品

### 初步觀察

- 「競爭差異」應落在可量化價值（成本、效率、留存、導入速度）
- 帳號共用是 SaaS 常見漏損，會直接影響收入與資安
- 「未來專注」若過度分散，會稀釋資源與市場訊號
- 實體空間需求需依產品交付模式決定（純軟體 vs 展示/運維/實作）

## 關鍵結論

- 這份筆記目前屬於商業模式與營運治理問題整理階段
- 後續應優先補齊競品地圖、反共享機制與聚焦路線
- 是否需要實體空間應以成本效益與客戶需求驗證，而非先入為主

## Q&A

### 與競爭對手差異

- 問題：與競爭對手差異是什麼？
- 可追問方向：
  - 差異在功能深度、導入速度、整合能力、價格，還是服務品質？
  - 目標客群與競品是否重疊？若重疊，勝出關鍵是什麼？
  - 是否有可量化證據（留存率、轉換率、導入時程、客戶案例）？
  - 差異是短期可被複製，還是長期護城河？

### 如何避免一個帳號大家通用

- 問題：如何避免一個帳號大家通用？
- 可追問方向：
  - 是否有裝置綁定、同時登入限制、異常地理登入偵測？
  - 是否導入 SSO、2FA、角色權限與稽核紀錄？
  - 價格方案是否鼓勵合法多席位，而非逼客戶共用帳號？
  - 是否有違規偵測與分級提醒/封鎖機制？

### 未來會專注在哪裡

- 問題：未來專注方向是什麼？
- 可追問方向：
  - 未來 12/24/36 個月分別聚焦哪個市場與產品線？
  - 哪些業務應加碼，哪些業務應收斂？
  - 是否以 ARR、毛利、客戶成功率作為聚焦判準？
  - 目標是否與人力與資本配置一致？

### 公司營運是否需要實體空間

- 問題：公司營運是否需要實體空間？
- 可追問方向：
  - 若有硬體展示、教育訓練、維修服務，是否需要據點？
  - 純軟體交付是否可用遠端營運降低固定成本？
  - 實體空間是否帶來客戶信任、銷售轉換或合作機會？
  - 有無「小據點 + 遠端」的混合營運模式可行？
