# 018 勝豐精密 五金零件整合平台

## 一句話摘要

勝豐精密是五金零件整合平台（非自有工廠製造商），以 2 萬種規格、採購看得懂的圖面與 48 小時出貨，從插銷切入，三年打進約 1,800 家客戶。

## 評審提問與回答

### Q1. 你是類似 B2B 的銷售平台嗎？自己也是製造商，還是純平台？除了插銷，鎖定品項很多，接下來往哪發展？

我們就是**五金零件整合平台**，整合台灣在地供應鏈，依需求配對加工方式。與工廠差異：工廠彼此競爭、資訊保留；我們是**合作**關係。我們看產品想的是「找誰做、多少量、能否共單降本」，不是只做能不能做。品項已從插銷擴到螺絲、螺帽、華司，並往**橡膠製品**前進——品類邏輯相同：台灣有生產但內銷門檻高，我們做對接。

### Q2. 五金零件要證明品質，需相關認證；你們在品質與國際／國內認證上如何說明？

台灣製造能力很強，缺的是**資訊整合**。我們跑很多工廠，科技業小單對工廠很麻煩（量測、刀具、交期），他們寧可做原本大單。我們用**大單綁小單**整合訂單，甚至多做一點變庫存，變成完整五金解決方案，所以毛利能比其他製造業高。（逐字稿未逐項列 ISO 等證照。）

### Q3. 若兩萬種零組件的 Marketplace 成功了，下一步怎麼讓後面供應商變得更強？

台灣很多特殊材料、表面處理訂單若不留台灣，材料也不會進口——我們希望打破這道牆，接軍工、航太等特殊金屬加工訂單。舉例：泰國客戶只買 150 支插銷，家庭代工不願接；我們整合其他客戶訂單一起下，甚至多做成庫存。賣一萬種插銷賣的是**解決方案**，不是單一零件。

### Q4. 客戶名單有宏碁、鴻海等一流電子廠，品質與技術加持如何談？

（與 Q2 連動）我們靠共單、吃庫存、整合供應鏈，讓工廠願意接科技業零碎、高要求訂單；三年熬過累積庫存階段，今年起毛利明顯跳升。目標是把更多訂單留在台灣，含 A286、Inconel 718 等特殊加工——台灣不是不能做，是沒人願意接小單。

### Q5. 若你們沒有製造，怎麼讓零件產業技術水準提升？

同上：我們把訂單留在台灣、帶動特殊材料與加工需求，讓供應鏈有機會升級；平台角色是媒合與共單，不是取代工廠製造。
