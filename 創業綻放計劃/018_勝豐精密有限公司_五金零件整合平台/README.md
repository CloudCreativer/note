# 018 勝豐精密 · 五金零件整合平台

## 30 秒看懂

勝豐精密**不是工廠**，而是**五金零件 B2B 整合平台**：把 **2 萬種**規格上架、用**採購看得懂的圖面**、**48 小時出貨**，從「大廠不做的」**插銷**切入，三年打進約 **1,800** 家客戶（含上太空案例）。  
商業飛輪：專注品項 → 拿單 → **共單降本** → 提高毛利（**20%→32%**）→ 再拿單；今年營收目標至少 **2,000 萬**。

## 解決什麼問題

### 台灣傳產／科技廠痛點

- 客人拿圖問「有沒有」：工程圖難懂、找不到廠、小量沒人做、價格貴。  
- 螺絲可能**台灣製造卻去美國買回來**。  
- 整機出口上升 → 科技廠需要大量**零碎零件**，但不知去哪買。

### 平台解法

- **2 萬 SKU** 規格化 + 庫存 + **48h 出貨**。  
- 圖面給**採購語言**（非工程圖）。  
- **大單綁小單**說服代工廠；吃庫存換科技業小批量訂單。  
- 把特殊材料／表面處理訂單留在台灣，帶動供應鏈升級。

## 評審提問與回答

### Q1. 你是類似 B2B 的銷售平台嗎？自己也是製造商，還是純平台？除了插銷，接下來往哪發展？

我們就是**五金零件整合平台**，整合台灣在地供應鏈。與工廠差異：工廠彼此競爭、資訊保留；我們是**合作**關係。品項已從插銷擴到螺絲、螺帽、華司，並往**橡膠製品**前進。

### Q2. 五金零件要證明品質，需相關認證；你們如何說明？

台灣製造能力強，缺的是**資訊整合**。我們用**大單綁小單**整合訂單，甚至多做一點變庫存，變成完整五金解決方案，所以毛利能更高。（逐字稿未逐項列證照。）

### Q3. 兩萬種零組件的 Marketplace 成功後，如何讓供應商更強？

台灣特殊材料、表面處理訂單若不留台灣，材料也不會進口——我們接軍工、航太等特殊加工訂單。例：泰國客戶只買 150 支插銷，我們整合其他客戶訂單一起下。

### Q4. 客戶有宏碁、鴻海等，品質與技術加持如何談？

我們靠共單、吃庫存、整合供應鏈，讓工廠願意接科技業零碎、高要求訂單；三年熬過累積庫存階段，今年起毛利明顯跳升。目標把更多訂單留在台灣（含 A286、Inconel 718 等特殊加工）。

### Q5. 若沒有製造，怎麼提升產業技術水準？

把訂單留在台灣、帶動特殊材料與加工需求，讓供應鏈有機會升級；平台角色是媒合與共單。

## 我學到的 3 件事

1. **B2B 五金可以產品化**：勝豐把「規格翻譯 + 快速交付 + 共單整合」做成可重複的服務產品。  
2. **先打小而難的品項是有效策略**：從大廠不做的小量插銷切入，反而能建立護城河與供應鏈黏著。  
3. **營運飛輪是核心資產**：訂單、庫存、成本、毛利之間的正循環，決定這類平台可否長期擴張。
