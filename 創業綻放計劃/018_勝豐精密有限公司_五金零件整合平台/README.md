# 018 勝豐精密有限公司 五金零件整合平台

## 一句話摘要

勝豐精密有限公司的主題聚焦於五金零件整合平台，目前重點在釐清公司是平台型服務還是製造商、品質認證狀態、兩萬零組件規模化後的強化路徑，以及進出口成長下如何切入上市櫃電子公司供應鏈。

## 背景

- 主題公司：勝豐精密有限公司
- 主題方向：五金零件整合平台
- 已知線索：零組件規模約兩萬項；進出口比例持續提升
- 現階段目標：建立商業定位、供應鏈能力與大客戶拓展策略理解

## 重點整理

### 產品與商業方向

- 五金零件整合平台
- 可能涵蓋供應商整合、選型配套、交期管理、品保與物流
- 需釐清是否同時具備製造能力

### 初步觀察

- 「平台還是製造商」會直接影響毛利結構、資本投入與競爭策略
- 若要供貨給上市櫃電子公司，品質系統與認證通常是最低門檻
- 兩萬零組件本身不是護城河，關鍵在資料化能力、交付穩定度與客戶切換成本
- 進出口占比提高代表市場機會擴大，但也提高合規、品質與交期管理壓力

## 關鍵結論

- 這份筆記目前屬於供應鏈平台定位與規模化策略問題整理階段
- 後續應優先補齊公司定位（平台/製造/混合）、品質認證與客戶分級策略
- 若目標上市櫃電子公司，需建立可被稽核的品質、交期與追溯機制
- 規模化重點應從「品項數量」轉向「數據與服務深度」

## Q&A

### 只是平台還是製造商

- 問題：公司只是平台還是製造商？
- 可追問方向：
  - 收入主要來自交易撮合、加值服務，還是自有製造毛利？
  - 是否有自有工廠、產線或關鍵製程能力？
  - 對客戶承諾是供應鏈整合、快速交付，還是製造品質控制？
  - 平台與製造若並行，內部資源如何分配？

### 品質是否有認證

- 問題：品質是否有認證？
- 可追問方向：
  - 是否具備 ISO 9001、IATF 16949、ISO 13485 或其他產業認證？
  - 認證範圍是公司整體、特定工廠，還是特定產品線？
  - 是否有來料檢驗、製程追溯、出貨驗證與不良率管理機制？
  - 對電子業客戶要求的稽核流程是否已具備？

### 兩萬零組件成功後如何更強

- 問題：兩萬零組件如果成功了，如何變得更強？
- 可追問方向：
  - 是否建立零件資料庫標準化（規格、替代料、可靠度、交期）？
  - 是否能用數據做採購預測、交期優化與風險預警？
  - 能否從「商品平台」升級成「工程解決方案」與技術服務？
  - 如何提高客戶黏著（API 串接、採購流程整合、客製化報表）？

### 進出口比例增加，如何賣到上市櫃電子公司

- 問題：進出口比例越來越多，如何賣到上市櫃電子公司？
- 可追問方向：
  - 目前主要出口市場與客戶型態是什麼？
  - 是否已對齊電子業採購規範（品質文件、供應商分級、ESG/碳排要求）？
  - 進入上市櫃客戶通常需要哪些前置條件（AVL、試單、稽核、認證）？
  - BD 策略是直攻大客戶、經銷商合作，還是從二階供應商切入？
