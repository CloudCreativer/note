# 005 華城電能科技股份有限公司 電動車充電

## 一句話摘要

華城電能科技股份有限公司的主題聚焦於電動車充電一條龍服務整合，目前重點在釐清一條龍服務的核心定位、建置成本與充電時間策略、專利與營運關係、研發結構、數據掌握能力、毛利提升策略，以及電動車與電動裝市場變化下的應變方式與母公司助力。

## 背景

- 主題公司：華城電能科技股份有限公司
- 主題方向：電動車充電一條龍服務整合
- 已知線索包含約 3 億營業額但仍處燒錢階段、十幾項專利、研發占比約 30% 且多為轉讓、掌握幾十萬筆數據
- 現階段目標是建立商業定位與財務結構理解，並整理可驗證的經營與技術問題

## 重點整理

### 產品與服務方向

- 電動車充電一條龍服務整合
- 可能涵蓋建置、營運、軟硬體、數據與充電效率優化

### 初步觀察

- 若營業額約 3 億仍明顯燒錢，需釐清一條龍服務中哪一段才是真正創造收入與護城河的環節
- 充電時間與建置成本通常是商業模式能否規模化的關鍵變數
- 十幾項專利若與營運脫鉤，可能更多是技術資產而非直接營收引擎
- 研發占比 30% 且多為轉讓，代表技術輸出可能是重要收入或策略手段，而非純內部研發
- 電動車需求放緩、電動裝需求增加時，公司需重新定義主力市場與產品組合

## 關鍵結論

- 這份筆記目前屬於充電服務定位與財務結構梳理階段
- 後續應優先釐清一條龍服務的主軸、燒錢原因與毛利提升路徑
- 專利、研發轉讓與數據能力需分開評估，避免混為同一種競爭優勢
- 市場結構變化（電動車 vs 電動裝）將直接影響公司成長敘事與資源配置

## Q&A

### 3 億營業額仍燒錢，一條龍服務主軸在哪

- 問題：營業額約 3 億仍很燒錢，一條龍服務的主要定位究竟在哪一段？
- 可追問方向：
  - 收入主要來自建置、設備銷售、營運分潤、軟體平台，還是能源管理？
  - 燒錢主要來自擴張建置、研發、通路，還是營運虧損？
  - 一條龍是為了提高客單價，還是為了鎖定長期營運與數據？

### 建置成本與充電時間

- 問題：建置成本是多少？為什麼要特別解決充電時間？
- 可追問方向：
  - 單站建置成本、回本周期、利用率假設為何？
  - 充電時間優化是為了提升周轉率、降低排隊、提高營收，還是技術差異化？
  - 是否有快充、功率管理或排程相關的具體方案？

### 專利十幾項與營運關係

- 問題：十幾項專利分別是什麼？與實際營運的關係為何？
- 可追問方向：
  - 專利涵蓋充電樁硬體、通訊協定、能源管理、安全機制，還是其他領域？
  - 專利是防禦性布局、授權收入來源，還是對外談合作的籌碼？
  - 哪些專利已實際用在產品或案場中？

### 研發占 30% 且多為轉讓

- 問題：研發占約 30%，且大多為轉讓，這代表什麼？
- 可追問方向：
  - 轉讓是指技術授權、專利轉讓、專案移轉，還是內部研發成果對外輸出？
  - 研發費用高是因為產品迭代，還是因為商業模式偏專案與客製？
  - 轉讓收入是否為穩定營收來源，還是一次性收入？

### 幾十萬筆數據與掌握參數

- 問題：幾十萬筆數據掌握了哪些參數？
- 可追問方向：
  - 數據來自充電行為、設備狀態、用電負載、故障紀錄，還是用戶行為？
  - 這些參數如何支援營運優化、定價、維運或產品設計？
  - 數據是否形成可複製的演算法或營運優勢？

### 毛利提升策略

- 問題：毛利提升的策略是什麼？
- 可追問方向：
  - 是透過提高利用率、降低建置成本、軟體加值，還是調整客戶結構？
  - 哪些業務線毛利較高，哪些仍在補貼擴張？
  - 是否有從建置導向營運導向的轉型計畫？

### 電動車下降、電動裝新增，如何應變

- 問題：若電動車需求下降、電動裝需求增加，公司如何應變？
- 可追問方向：
  - 產品與案場是否可從乘用車充電轉向商用、物流、工業或特殊載具？
  - 一條龍服務是否需調整硬體規格、功率配置與客戶類型？
  - 是否已有電動裝相關客戶、試點或營收占比？

### 母公司華城電機的助力

- 問題：母公司華城電機與目前公司的助力是什麼？
- 可追問方向：
  - 助力來自品牌、通路、製造、機電整合能力，還是客戶資源？
  - 母子公司之間是資源共享、訂單導流，還是技術與供應鏈協同？
  - 是否存在集團內部轉單、成本分攤或策略協同安排？
