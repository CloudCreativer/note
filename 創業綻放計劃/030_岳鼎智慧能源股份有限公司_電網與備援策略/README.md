# 030 岳鼎智慧能源股份有限公司 電網與備援策略

## 一句話摘要

岳鼎智慧能源股份有限公司的主題聚焦於電網與備援策略，當前重點在釐清其方案是否可完整取代 UPS、以全漢案例定義獨特價值與北美 5% 佔比邏輯，以及電網互動與降載機制的實際運作方式。

## 背景

- 主題公司：岳鼎智慧能源股份有限公司
- 主題方向：智慧能源、備援方案、電網互動與降載管理
- 已知關注點：UPS 替代性、全漢案例、北美市佔說法、降載運作機制
- 現階段目標：建立技術可行性與商業說服力的驗證框架

## 重點整理

### 產品與技術方向

- 備援電力與能源管理方案
- 電網互動（可能涉及需量反應、削峰填谷、調度控制）
- 降載機制與場域運行策略

### 初步觀察

- 「能否取代 UPS」需分場景，通常取決於切換時間、可靠度與合規要求
- 「北美 5%」若作為對外主張，必須清楚定義分母（哪個市場區間）
- 案例價值需從痛點-方案-結果三段式敘事，避免只講客戶名稱
- 降載不只是技術控制，也涉及電價機制、契約容量與客戶營運限制

## 關鍵結論

- 這份筆記目前屬於能源備援方案與市場驗證問題整理階段
- 後續應優先補齊 UPS 替代邊界、案例量化數據與市佔定義
- 電網/降載運作須拆成技術、商業、法規三層解釋

## Q&A

### UPS 是否能完整取代

- 問題：方案是否能完整取代 UPS？
- 可追問方向：
  - 取代範圍是短時斷電保護、長時備援，還是混合場景？
  - 切換時間（ms 級）、穩定度與關鍵設備兼容性是否達標？
  - 是否符合資料中心、工廠、醫療等不同場景的合規要求？
  - 更可能是「部分替代 + 互補」還是「全面替代」？

### 全漢案例：獨特問題與北美 5%

- 問題：以全漢為例，解決了什麼獨特問題？為什麼能占北美 5%？
- 可追問方向：
  - 全漢案例中的具體痛點是停電風險、能源成本、需量罰金，還是 ESG 壓力？
  - 解法與傳統方案相比改善了哪些指標（成本、穩定性、回收期）？
  - 「北美 5%」是指哪個細分市場（UPS、儲能、特定應用）？
  - 此市佔數據來自自有統計、第三方報告，還是單一客群滲透率？

### 電網、降載運作方式

- 問題：電網與降載是如何運作的？
- 可追問方向：
  - 系統如何監測負載並觸發降載策略（規則/AI/即時控制）？
  - 降載優先順序如何設定，如何避免影響關鍵生產流程？
  - 是否整合台電費率、需量反應或合約容量管理？
  - 實際成效如何量化（電費下降、罰金降低、供電穩定）？
