# 030 岳鼎智慧能源股份有限公司 · 電網與備援策略

## 30 秒看懂

岳鼎主打資料中心機櫃層級的動態電力協調方案，核心是結合電網訊號、機櫃電源與電池管理，讓 AI 負載在新法規下仍可穩定運行。評審 QA 核心在：是否取代 UPS、與硬體夥伴合作邏輯、電網降載如何實際運作，以及市場佔比敘事是否可被驗證。

## 他們在做什麼

- 產品方向：機櫃層級電力協調器 + 上層協調 Agent。
- 技術主張：在 OCP/OAI 類架構下，透過機櫃內電池與電源模組協調，提升韌性與有效功率。
- 目標場景：高密度 AI 資料中心，特別是電網波動與法規要求嚴格場域（逐字稿脈絡以北美為主）。

## 解決什麼問題

- 新型資料中心在特定電力架構下，前端未必配置傳統 UPS，導致電網擾動時韌性壓力升高。
- AI 高峰負載與電網調度要求衝突，容易產生降載與穩定性難題。
- 純硬體備援成本高、效率損耗高，需更細粒度的機櫃層電力協調。

## 怎麼賺錢

- 收入模式（逐字稿口述）：
  - 每機櫃部署硬體/控制節點。
  - 年度軟體服務費（協調與管理功能）。
  - 透過規格導入與夥伴生態放大出貨規模。
- 市場邏輯：先切北美高密度資料中心場景，再擴到更多電網規範要求嚴格區域。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 已有與電源/伺服器生態夥伴合作與 PoC 推進。
  - 嘗試進入 OCP 規格路徑，放大生態採用機會。
  - 以機櫃層協調方式提升可用功率與穩定度。
- 評審追問集中：
  - 方案到底是替代 UPS 還是補充 UPS。
  - 和既有電源廠合作時，權責邊界與價值分工。
  - 電網異常下實際降載與回復流程是否可驗證。

## 評審提問與回答

### Q1. 你們是要取代 UPS 還是補強 UPS？
- 團隊回答：在目標架構下重點不是傳統 UPS 取代敘事，而是機櫃層動態協調與電池協作，處理特定新型資料中心痛點。
- 正確性註記：`中信心`
  - 理由：方向清楚，但「可替代邊界」仍需更精準定義。

### Q2. 為什麼全漢（或電源夥伴）會跟你們合作？
- 團隊回答：團隊提供控制與協調能力，夥伴提供既有硬體與通路，形成互補。
- 正確性註記：`中高信心`
  - 理由：合作分工敘事一致，符合產業合作常態。

### Q3. 電網異常時，降載是怎麼被觸發與執行？
- 團隊回答：上層 Agent 感知電網狀態並下發策略，機櫃層控制器協調電源與電池完成瞬時調節與回復。
- 正確性註記：`中高信心`
  - 理由：技術流程描述完整，與前後回答一致。

### Q4. 北美 5% 這個說法如何定義？
- 團隊回答：以目標細分場景作為切入推估，而非整體市場全面占比。
- 正確性註記：`中低信心`
  - 理由：口徑在逐字稿中不夠精確，需補第三方市場模型。

## 我學到的 3 件事

1. 能源新創要說服評審，必須把「技術流程」轉成「可驗證運行指標」。
2. 在資料中心電力場景，取代與補強 UPS 是兩個不同命題，不能混用。
3. 生態合作的關鍵是權責清楚：誰做硬體、誰做控制、誰拿客戶入口。
