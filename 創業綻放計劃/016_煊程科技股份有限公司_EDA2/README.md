# 016 煊程科技股份有限公司 EDA2

## 一句話摘要

煊程科技股份有限公司的主題聚焦於 EDA2，目前重點在釐清輸入輸出定義與價值鏈分工、工具廠商採用情況、專利與競品差異化、商業模式，以及公司願景與目標如何具體落地。

## 背景

- 主題公司：煊程科技股份有限公司
- 主題方向：EDA2
- 目前關注點：I/O、分工、採用現況、專利與競品、商模、願景落地
- 現階段目標：建立 EDA2 在產業鏈中的實際位置與可擴張性理解

## 重點整理

### 產品與技術方向

- EDA2（具體定義待公司說明）
- 可能位於晶片設計流程中的特定環節（前端、後端、驗證、簽核或整合層）
- 可能面向工具廠、IC 設計公司或系統整合夥伴

### 初步觀察

- 「EDA2 分工在哪裡」是最核心問題，先釐清流程位置才有辦法談商模與差異化

- 專利與競品比較需同時看技術深度、導入門檻、整合成本與客戶鎖定能力
- 願景如果過大但缺少階段性 KPI，容易造成市場訊號混亂

## 關鍵結論

- 這份筆記目前屬於 EDA2 定位與商業化驗證問題整理階段
- 後續應優先補齊 I/O、工作流節點與目標客群

## Q&A

### 輸入、輸出是什麼？EDA2 分工在哪裡

- 問題：輸入、輸出是什麼？EDA2 分工在哪裡？
- 可追問方向：
  - 輸入是 netlist、layout、PDK、rule deck、測試向量，還是其他資料格式？
  - 輸出是報告、優化建議、簽核結果、可回灌檔案，還是 API 結果？
  - 位於設計流程哪一段：front-end、back-end、verification、DFT、sign-off？
  - 與現有 EDA 工具是替代關係還是補強模組？

### 工具廠商使用情況

- 問題：工具廠商使用的情況如何？
- 可追問方向：
  - 已合作或試用的工具廠商有哪些（若可公開）？
  - 使用層級是 PoC、內部驗證、正式商用，還是共同銷售？
  - 客戶導入障礙在技術、流程、授權模式還是支援服務？
  - 採用後是否有量化成效（時間縮短、錯誤率下降、成本節省）？

### 專利、競爭對手優勢、差異化、商模

- 問題：專利？競爭對手優勢？差異化在哪裡？商模在哪裡？
- 可追問方向：
  - 是否有自有專利與核心演算法保護？涵蓋哪個技術點？
  - 主要競爭對手是國際 EDA 大廠、區域新創，還是內部自研工具？
  - 差異化是速度、準確度、自動化程度、易整合，還是價格／授權彈性？
  - 商模是授權費（seat/license）、訂閱制、用量計費，還是專案導入？
  - 是否可形成高切換成本與長期客戶黏著？

### 公司願景與目標過多，如何細緻化對手及差異化

- 問題：公司願景與目標看起來很多，如何細緻化對手及差異化？
- 可追問方向：
  - 願景可否拆成 12 個月、24 個月、36 個月的明確里程碑？
  - 每一階段對應的目標客群、產品功能與收入模型是什麼？
  - 各目標對應哪一類競爭者，核心差異化指標是什麼？
  - 哪些目標應優先聚焦，哪些可延後以避免資源分散？
