# 003 安瑟樂威股份有限公司 · 智慧能源管理

## 30 秒看懂

安瑟樂威（`Ancillary Power`）主打「分散式能源聚合 + 雲端電力調度」的虛擬電廠平台，訴求同時解決企業端電費優化與電網端調度需求。  
簡報可辨識其核心敘事是：以 VPP 聚合資源提供低延遲調度，並把成長引擎拆成電力交易平台、綠電售電與海外市場三軌，目標在 2028 年衝到約 `31億` 營收規模。

## 他們在做什麼

- 公司定位（依逐字稿）：台灣能源智能服務商，以 VPP 為核心。
- 解決方案：
  - 聚合分散式資源（用電負載、發電設備、儲能、電動車）；
  - 透過平台做調度與結算，滿足電網需求；
  - 提供企業端能源管理與負載預測。
- 商業敘事：把「用電成本」轉成可管理、可產生收益的能源資產。

## 解決什麼問題

- 電網尖峰與突發事件時，需快速調度分散資源以穩定供電。
- 用電大戶缺乏即時可操作的負載預測與調度能力。
- 傳統能源管理導入慢，跨場域部署與維運成本高。

## 怎麼賺錢

- 逐字稿可辨識三類營收：
  - 核心業務：虛擬電廠/電力交易相關收入。
  - 售電或轉供相關收入（口述占比上升）。
  - 軟體/顧問等服務收入（占比較低）。
- 逐字稿口述數據：
  - 去年營收約 3 億級距；
  - 核心業務占比約 80%（前一年約 90%），顯示收入結構多元化中。

## 實績與進展（決賽現場）

- 逐字稿可辨識進展：
  - 平台能力頁揭露高聚合、低延遲與多能源接入架構（儲能、太陽能、充電樁等）。
  - 團隊頁揭露人員由 `8 -> 37`（2022~2025）與跨域 DEI 組織配置。
  - 成長頁揭露 2025~2028 年營收階梯（約 `3.48億 -> 31億`）與 `22.6% CAGR` 敘事。
  - 資金頁揭露獎金/資金用途分配與 KPI（容量增長、節能績效、新增服務）。
- 評審關注重點：
  - 「節能/省錢」到底是節能量、電費節省還是調度收益。
  - 與台電/市場規則對接後，客戶端實得價值如何計價。
  - 技術門檻是否可複製到海外市場制度。

## 評審提問與回答

### Q1. 幫用電大戶「省多少」？怎麼收費？與台電如何對接？
- 評審重點：客戶價值是否可量化、商業模式是否成立。
- 團隊回答重點（逐字稿）：
  - 核心不只節能，而是「聰明用電 + 參與調度」；
  - 參與台電調度可形成可計入績效的價值；
  - 另提供能源管理平台與預測能力。
- 正確性註記：`中信心`
  - 原因：邏輯完整，但節能百分比與收費公式未完整量化公開。

### Q2. 去年營業成績成長，收入來源是什麼？
- 評審重點：成長是否來自單一業務，風險是否過高。
- 團隊回答重點（逐字稿）：
  - 核心業務占比約 80%，較前一年 90% 下降；
  - 第二、第三收入來源正在成長，收入結構變多元。
- 正確性註記：`中信心`
  - 原因：有占比敘述，但缺審計口徑財報佐證。

### Q3. 技術門檻是什麼？能否向海外擴張？
- 評審重點：護城河是否足以避免同質化競爭。
- 團隊回答重點（逐字稿）：
  - 平台主打全雲端、高聚合、低延遲部署；
  - 可在短時間內完成站點接入與調度能力上線；
  - 海外擴張方向已列為中期策略。
- 正確性註記：`中信心`
  - 原因：技術方向合理，但海外法規與市場進入路徑尚未具體化。

### Q4. 為何先深耕台灣，再談海外？
- 評審重點：台灣模式能否外溢。
- 團隊回答重點（逐字稿）：
  - 先在台灣把模型與規模做實，再複製到海外市場。
- 正確性註記：`中信心`
  - 原因：符合能源新創常見路徑，但需補國別策略與在地夥伴清單。

## 我學到的 3 件事

1. VPP 的核心價值通常不是單點節能，而是「可調度能力」的市場化。
2. 成長期最關鍵的是把收入來源從單一業務轉成可持續的多元結構。
3. 能源平台要出海，技術之外更難的是電力制度、法規與在地商業夥伴。
