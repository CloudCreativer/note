# 003 安瑟樂威股份有限公司 智慧能源管理

## 一句話摘要

安瑟樂威股份有限公司的主題聚焦於減碳與電力需求管理，核心切入點包含智慧能源艦隊管理平台與 AI 分析用電，目前重點在釐清節能成效、營收模式、能源調度管理門檻，以及海外布局與募資策略。

## 背景

- 主題公司：安瑟樂威股份有限公司
- 主題方向：減碳、電力需求管理、智慧能源平台
- 已知切入點包含智慧能源艦隊管理平台、AI 分析用電、能源調度管理
- 現階段目標是建立產品與商業模式理解，並整理可驗證的經營問題

## 重點整理

### 產品與技術方向

- 減碳
- 電力需求管理
- 智慧能源艦隊管理平台
- AI 分析用電

### 初步觀察

- 產品主張應聚焦於協助用電大戶或能源資產持有者優化用電與調度效率
- 若平台同時處理需求面管理與電廠調度，代表技術與商業模式可能跨越能源管理軟體與能源服務
- 若宣稱可省電 45%，需要確認是特定情境、單一案場，還是可普遍複製的成效
- 若公司切入電廠轉型與能源調度管理，門檻可能來自資料整合、即時控制能力、法規與在地市場機制

## 關鍵結論

- 這份筆記目前屬於能源平台定位與商業驗證問題整理階段
- 後續應優先補齊節能成效依據、營收模式與能源調度管理的實際落地案例
- 海外布局與是否募資，將直接反映公司成長策略與資本密集程度

## Q&A

### 用電大戶省電 45%

- 問題：用電大戶可省電 45% 的說法依據為何？
- 可追問方向：
  - 45% 是尖峰用電削減、總用電降低、電費節省，還是碳排減量？
  - 這個數字來自單一客戶案例、特定產業，還是多個案場平均值？
  - 是否有導入前後比較、期間長度、基準條件與第三方驗證？

### 營業成績與營收模式

- 問題：公司的營業成績如何？營收模式與業務樣態為何？
- 可追問方向：
  - 收費是 SaaS 訂閱、專案導入、節能分潤，還是平台加顧問服務？
  - 客戶是工廠、商辦、用電大戶、電廠，還是能源服務商？
  - 目前營收來源以軟體平台為主，還是整合服務與顧問專案為主？

### 電廠轉型能源調度管理

- 問題：從電廠轉型到能源調度管理，商業模式的門檻是什麼？是否可以擴展海外？
- 可追問方向：
  - 核心門檻在於演算法、資料接入、控制權限、法規資格，還是客戶信任？
  - 若要複製到海外，是否需要因應各國電力市場制度與調度規則重做？
  - 平台是否已具備多案場、多資產、多地區的擴展能力？

### 電廠如何海外布局

- 問題：若切入電廠或能源調度場景，海外布局會如何進行？
- 可追問方向：
  - 是透過技術授權、在地合作夥伴、系統整合商，還是直接落地案場？
  - 優先市場會是東南亞、美國、日本，還是其他再生能源需求較高地區？
  - 海外布局主要挑戰是法規、市場准入、設備整合，還是當地維運能力？

### 為什麼沒有對外募資

- 問題：為什麼公司沒有對外募資？
- 可追問方向：
  - 是因為目前現金流足夠、策略上不想稀釋股權，還是商業模式仍在驗證中？
  - 若屬於能源平台與調度管理業務，資本需求通常不低，未募資是否代表偏輕資產模式？
  - 是否曾接受政府計畫、策略投資或其他非典型資金來源？
