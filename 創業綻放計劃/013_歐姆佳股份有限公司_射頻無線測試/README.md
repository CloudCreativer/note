# 013 歐姆佳股份有限公司 射頻無線測試

## 一句話摘要

歐姆佳股份有限公司的主題聚焦於半導體與相控陣列場景下的射頻晶片與無線測試，目前重點在釐清產品是否強調速度與 CP 值而缺乏量測準確度論述、專利佈局、DPM 與 RD 使用權、插卡式架構與速度需求，以及營業額接近 1 億時是否還有其他產品線。

## 背景

- 主題公司：歐姆佳股份有限公司
- 主題方向：半導體、相控陣列、射頻晶片、無線測試
- 已知線索：營業額接近 1E（推測為約 1 億新台幣，待確認）
- 現場觀察（使用者筆記）：對外強調速度快、CP 值高低，但並未看到量測準確度相關說明
- 現階段目標：建立產品技術與商業理解，並整理驗證與 IP 相關問題

## 重點整理

### 產品與技術方向

- 射頻晶片測試
- 無線測試
- 相控陣列相關應用（待釐清具體產品形態）
- 可能涉及插卡式測試架構

### 初步觀察

- 若行銷訴求偏重速度與 CP 值，投資人會關注準確度、可重現性與校準能力是否被省略
- DPM（需確認全名：Data / Design / Device Performance Management？）與 RD 能否取用，牽涉資料資產與研發協作模式
- 「插卡式」可能代表模組化測試平台，速度需求需對應客戶製程節拍
- 營收約 1 億時，需釐清是單一主力產品還是多產品組合

## 關鍵結論

- 這份筆記目前屬於射頻測試設備／方案與技術驗證問題整理階段
- 後續應優先補齊量測準確度指標、專利地圖與產品線結構
- DPM 定義與 RD 使用權將影響技術護城河與客戶黏著度
- 插卡式與速度規格需綁定目標客戶（封測、IDM、設計公司）場景

## Q&A

### 速度快、CP 值，但沒有量測準確度

- 問題：對外強調速度快、CP 值高低，但似乎沒有量測準確度相關說明？
- 備註：此為使用者現場觀察，需進一步查證公司簡報或規格書是否有相關數據。
- 可追問方向：
  - 準確度指標（例如量測誤差、重複性、校準週期）為何？
  - 與國際大廠（或同業）測試機在速度 vs 精度上的取捨？
  - 客戶驗收標準是否以準確度為前提，還是先看 throughput？
  - CP 值高低的對照基準（競品、舊機台、客戶自製方案）？

### 專利佈局

- 問題：專利佈局狀況如何？
- 可追問方向：
  - 專利數量、主要技術領域（射頻、相控陣列、測試方法、硬體架構）？
  - 台灣、美國、中國等地域佈局？
  - 專利是防禦性、授權收入，還是客戶導入的技術背書？
  - 是否有核心演算法、校準方法或插卡式架構的獨家專利？

### DPM？RD 可以拿去研發？

- 問題：DPM 是什麼？RD 是否可以拿去研發？
- 備註：DPM 全名與定義待現場或文件確認。
- 可追問方向：
  - DPM 是平台、資料庫、流程管理，還是客戶交付物？
  - RD 團隊能否使用 DPM 內數據／模組做二次研發？授權範圍為何？
  - 客戶資料是否涉及保密、歸屬與再使用限制？
  - 若 RD 可沿用，對產品迭代與護城河的實際幫助為何？

### 插卡式與速度需求

- 問題：是否為插卡式架構？多快的需求？
- 可追問方向：
  - 插卡式指模組化儀器、PXI、自研背板，還是可熱插拔測試頭？
  - 客戶要求的測試時間、UPH、或頻段切換速度為何？
  - 速度瓶頸在硬體、軟體、演算法還是整合？
  - 不同射頻／相控陣列產品線是否共用同一平台？

### 營業額接近 1 億與其他產品

- 問題：營業額接近 1E，產品還有其他的嗎？
- 備註：「1E」推測為約 1 億（新台幣），需確認幣別、年度與是否為合併營收。
- 可追問方向：
  - 營收主力是單一測試設備、授權、維護，還是專案整合？
  - 除射頻／無線測試外，是否有相控陣列專用方案、軟體、校準服務？
  - 客戶集中度（前五大客戶占比）？
  - 未來 1–2 年產品路線圖？
