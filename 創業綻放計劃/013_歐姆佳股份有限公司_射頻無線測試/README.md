# 013 歐姆佳 射頻無線測試

## 一句話摘要

歐姆佳以相控陣列快速量測與板卡化射頻測試方案，服務 Wi‑Fi 6、衛星、半導體等客戶；2026 年募 1,000 萬推板卡化與 EPA 工具包。

## 評審提問與回答

### Q1. 產品在商業上的定位如何？簡報提到速度快，但比較表有 CP 值、測試速度，卻沒有量測準確度，這部分跟其他商品有什麼不同？

我們主要客戶是研究單位（含相控陣列、辦公室測試設備等）。**速度**已達客戶要求；**準確度**目前符合業界規範。相控陣列控制上，只能做到前端 beam IC 可控制的最小角度（例如能做 5° 就做到 5° 精度），已在一比一精度上被市場驗證，無法再更細。

### Q2. 你手上那一塊（逐字稿作「新產的 DPM」，應為 **DPA**）請再說明：對客戶 RD 而言，是可以直接用軟體做實驗研發，還是你們自己要拿去現場比賽？

DPA 可組合（例如 16 顆拼成離軌衛星應用），核心是 **Control IC**。客戶需求很多：希望我們協助 design，架構是 open 的，只要跟我們模組、MCU 對接就能快速完成後段軟體開發；對廣達等缺 RF 人力的廠特別需要。也能支撐晶片設計公司：產品賣給廣達，對接技術可一致，並支援低軌衛星、Open RAN 等協議。

### Q3. 插卡式機制貢獻很好，但頻率只到 14.2 GHz，這個頻段能應付產業需求嗎？未來往高頻走，有什麼期待？

板卡標示 14.2 GHz（十字機）鎖定現有主力客戶 **Wi‑Fi 6（802.11ax）**，在簡報所述數百億美元市場中占多數；毫米波短期較難做大。下一季推聲量比模組，目標延伸至約 **50 GHz**；**200 GHz 以上**與日本 NICT、羅姆合作（含 **500 GHz** 成像模組等）。今年軍方、漢翔等案場需求增加，模式是「建置案場 + 快速校正技術」，縮短客戶原本需一、兩個月的校正時間。
