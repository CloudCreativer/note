# 010 原子精製股份有限公司 ALD製程

## 一句話摘要

原子精製股份有限公司的主題聚焦於高階 ALD（原子層沉積）製程機器，目標是讓 ALD 提升產能，目前重點在釐清光學與車用應用可行性、設備成本與募資規劃、探針鍍膜相關數據，以及相對競品的競爭優勢。

## 背景

- 主題公司：原子精製股份有限公司
- 主題方向：高階 ALD 製程機器、提升 ALD 產能
- 已知線索包含募資約 300（單位待確認，可能為百萬或千萬新台幣）、探針鍍膜應用
- 現階段目標是建立技術應用與商業化理解，並整理市場、財務與驗證數據相關問題

## 重點整理

### 產品與技術方向

- 高階 ALD 製程機器
- 提升 ALD 產能（吞吐量、周期時間或批次效率）
- 可能應用於探針鍍膜等精密鍍膜場景

### 初步觀察

- ALD 設備通常資本密集，客戶決策周期長，設備成本與 ROI 是關鍵
- 若募資規模相對不大（使用者筆記：只募 300），需釐清單位、用途與下一輪計畫
- 光學、車用為潛在延伸市場，但需驗證製程要求、認證門檻與客戶導入路徑
- 探針鍍膜的片數、探針數與成效數據，可作為早期商業驗證的重要指標

## 關鍵結論

- 這份筆記目前屬於 ALD 設備與商業驗證問題整理階段
- 後續應優先補齊應用場景、定價／募資結構與探針鍍膜實證數據
- 競爭優勢需從產能、成本、精度、維護或客製化能力等維度具體化
- 「募 300」需確認幣別與單位，避免誤判公司資本規模

## Q&A

### 光學與車用應用

- 問題：是否可用到光學應用？車用應用？
- 可追問方向：
  - 光學應用指哪類鍍膜（抗反射、濾光、鏡頭模組、雷射元件等）？
  - 車用是否涉及感測器、功率元件、車載晶片封裝或其他部件？
  - 進入這些市場需滿足哪些膜厚均勻性、溫度、材料與認證要求？
  - 是否已有試用、送樣或合作案場？

### 設備成本與募資規劃

- 問題：設備成本多少錢？募資為什麼只有募 300？未來打算募多少？
- 可追問方向：
  - 「300」是指 300 萬、3000 萬、3 億，還是其他單位？
  - 單台 ALD 設備售價、毛利率與交期為何？
  - 本輪募資主要用途（研發、量產、通路、認證）？
  - 下一輪募資的目標金額、時間點與里程碑為何？

### 探針鍍膜：片數、探針數與數據

- 問題：探針鍍膜？有多少片？有多少探針？數據？
- 可追問方向：
  - 探針鍍膜是現有主力應用，還是驗證場景之一？
  - 「片」指 wafer、基板批次，還是鍍膜腔體的裝載單位？
  - 單次可處理探針數量、良率、膜厚一致性、寿命提升等數據為何？
  - 是否有客戶名稱、案場規模或重複訂單（若可公開）？

### 競爭優勢

- 問題：競爭優勢是什麼？
- 可追問方向：
  - 相對國內外 ALD 設備商，優勢在產能、價格、精度、占地面積還是維護？
  - 是否有專利、獨家製程 know-how 或關鍵模組自製率？
  - 客戶選擇公司的主要原因（導入速度、客製化、在地服務）？
  - 與「提升產能」直接相關的量化指標（例如 throughput 提升 %）？
