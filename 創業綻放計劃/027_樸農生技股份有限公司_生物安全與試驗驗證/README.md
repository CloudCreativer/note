# 027 樸農生技股份有限公司 · 生物安全與試驗驗證

## 30 秒看懂

樸農生技聚焦昆蟲病毒型生物農藥與量產平台，核心是「高安全性 + 可規模生產 + 冷鏈限制降低」。評審 QA 核心在：病毒來源與人體/環境安全證據、海外試驗是否可商業化、合格驗證標準，以及海外機會如何持續取得。

## 他們在做什麼

- 技術方向：以昆蟲病毒（核多角體病毒脈絡）開發生物防治產品。
- 產品定位：替代部分化學農藥，降低抗藥性與殘留風險。
- 平台主張：把高門檻養蟲與生產流程數位化/自動化，提高產能與一致性。
- 商業路徑：台灣先行驗證，搭配海外場域試驗與在地合作商進行導入。

## 解決什麼問題

- 化學農藥減量政策下，現場缺少兼顧效能與安全的新方案。
- 生物農藥常見痛點是生產成本高、供應不穩、保存條件嚴格。
- 海外市場導入需要兼顧在地法規、環境驗證與商業可行性。

## 怎麼賺錢

- 收入模式（逐字稿口述）：
  - 自有產品銷售。
  - 代工/委託生產（平台產能服務）。
  - 海外合作商導入後的持續供應。
- 商業放大點：把技術平台化，讓不同病毒株或不同市場可重複複製。
- 時程敘事：先拿本地藥證與量產，再擴到海外市場。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 第一支產品推進藥證流程，並提及第二支產品規劃。
  - 有海外場域合作接觸與試驗推進（含歐洲場域脈絡）。
  - 強調常溫保存能力與量產可擴張性。
- 評審追問焦點：
  - 病毒來源是否合法、可追溯，且是否對人體與其他生物安全。
  - 海外試驗是否只是驗證，還是已有收費/商業模式。
  - 合格環境驗證標準與持續監測機制是否完備。

## 評審提問與回答

### Q1. 病毒從哪裡來？是不是基改？為何能說對人體安全？
- 團隊回答：病毒株來自自然篩選，非基因改造；並引用長期使用文獻與監管脈絡說明安全性。
- 正確性註記：`中高信心`
  - 理由：回答邏輯完整且重複一致；仍需第三方檢測文件佐證。

### Q2. 海外場域試驗有實績嗎？有收費嗎？
- 團隊回答：已有海外合作方在談場域測試，流程通常先小規模驗證，再依結果啟動登記與商業導入。
- 正確性註記：`中信心`
  - 理由：商業流程說明合理，但收費模式與已簽約程度未完全量化。

### Q3. 所謂合格環境驗證，標準是什麼？
- 團隊回答：以各國法規與在地登記要求為主，會按國家要求補齊資料與驗證。
- 正確性註記：`中信心`
  - 理由：方向正確，但缺明確標準名稱與驗證機構資訊。

### Q4. 海外機會是怎麼拿到的？可複製嗎？
- 團隊回答：來源包含參展與國際接觸，先由在地夥伴提供場域試驗，再進入當地法規與商業流程。
- 正確性註記：`中信心`
  - 理由：BD 路徑可理解，仍需更多已成交案例支持「可複製」。

## 我學到的 3 件事

1. 生物農藥公司最先要回答的是安全性證據鏈，而不是市場故事。
2. 海外試驗要有價值，關鍵在能否從 PoC 轉成可收費供應。
3. 技術平台化能放大規模，但前提是法規文件可快速在各國重用。
