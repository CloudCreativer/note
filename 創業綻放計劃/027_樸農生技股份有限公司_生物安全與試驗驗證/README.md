# 027 樸農生技股份有限公司 生物安全與試驗驗證

## 一句話摘要

樸農生技股份有限公司的主題聚焦於生物安全與試驗驗證，當前重點在釐清病毒來源與人體安全證據、海外場域試驗經驗與收費模式、合格環境驗證標準，以及海外試驗機會取得管道。

## 背景

- 主題公司：樸農生技股份有限公司
- 主題方向：病毒相關技術應用、生物安全、場域驗證
- 已知關注點：來源可追溯性、人體無害證明、海外試驗落地
- 現階段目標：建立安全性、法規合規與商業可行性的驗證框架

## 重點整理

### 產品與技術方向

- 病毒相關技術（具體用途待確認）
- 可能涉及場域導入、環境驗證與實地試驗
- 可能面向農業、生技或環境治理情境（待確認）

### 初步觀察

- 「病毒哪裡來」與「對人體無害」是最核心的信任門檻
- 海外試驗經驗若能量化，會直接提升商業可信度
- 合格環境驗證需明確指出標準、單位與流程，避免口號化
- 海外機會來源可反推公司 BD 能力與合作網路成熟度

## 關鍵結論

- 這份筆記目前屬於生物安全與海外驗證問題整理階段
- 後續應優先補齊病毒來源、風險評估與安全證據鏈
- 海外試驗若具收費模式，代表從驗證走向商業化的關鍵轉折
- 合規與驗證要件需可被第三方查核

## Q&A

### 病毒哪裡來？如何確保對人體無傷害

- 問題：病毒來源為何？如何確保對人體沒有傷害？
- 可追問方向：
  - 病毒株來源是自研、授權、公共菌種庫，還是合作單位提供？
  - 是否有完整來源追溯與批次管理（traceability）？
  - 人體安全證據來自哪些試驗（毒理、暴露、環境風險）？
  - 是否有第三方檢測、法規審查或倫理審查支持？
  - 對操作人員與周邊環境是否有 SOP 與防護規範？

### 海外場域試驗經驗？有收費嗎

- 問題：海外場域試驗是否有經驗？有收費嗎？
- 可追問方向：
  - 已完成哪些國家、哪些場域、哪些規模的試驗？
  - 試驗屬 PoC、示範案還是正式商業導入？
  - 收費是試驗服務費、成果分潤，還是後續合約綁定？
  - 試驗成功率、轉單率與客戶續約情況如何？

### 合格環境驗證

- 問題：有哪些合格環境驗證？
- 可追問方向：
  - 採用哪一套國內外標準（ISO、當地主管機關或產業規範）？
  - 驗證範圍是實驗室、場域、產品，還是整體流程？
  - 驗證是一次性還是持續稽核？週期與維持條件為何？
  - 是否有公開證書、檢驗報告或稽核紀錄？

### 海外試驗機會如何獲得

- 問題：海外試驗機會是如何取得的？
- 可追問方向：
  - 來源是政府計畫、國際合作、客戶轉介，還是通路夥伴？
  - 是否依靠既有學研合作或跨國企業供應鏈導入？
  - 獲案流程中最關鍵的門檻是法規、在地夥伴還是技術驗證？
  - 目前可複製的獲客機制是否已建立？
