# 017 蔚流生物科技股份有限公司 腫瘤環境晶片

## 一句話摘要

蔚流生物科技股份有限公司的主題聚焦於腫瘤環境晶片與 MedSelect 系統，目前重點在釐清 85-90% 數據如何測試得出、營運模式是賣晶片還是賣系統、專利屏障是否存在，以及團隊組成與規模是否足以支撐商業化。

## 背景

- 主題公司：蔚流生物科技股份有限公司
- 主題方向：腫瘤環境晶片、MedSelect 系統
- 已知線索：對外提及 85-90% 數據；目前觀察團隊公開資訊似乎僅兩位核心成員
- 現階段目標：建立技術可信度、商業模式與團隊可執行性理解

## 重點整理

### 產品與技術方向

- 腫瘤環境晶片
- MedSelect 系統
- 可能應用於腫瘤藥物反應評估、病患分層或治療決策輔助（待確認）

### 初步觀察

- 若宣稱 85-90% 成效，需先釐清指標定義與測試設計，避免誤讀
- 晶片與系統可能是同一價值鏈不同層，商模需明確界定主收入來源
- 若目前看不到專利屏障，需確認是否以 know-how、資料、流程認證建立護城河
- 團隊規模若偏小，需釐清是否有外部合作網絡補足研發與臨床資源

## Q&A

### 85-90% 是怎麼測試出來的

- 問題：數據 85-90% 是如何測試得出的？
- 可追問方向：
  - 85-90% 指的是準確率、預測一致性、命中率，還是其他指標？
  - 樣本數、資料來源、實驗設計與對照組為何？
  - 是回溯研究、前瞻研究，還是臨床合作案的結果？
  - 是否有第三方驗證、論文、海報或醫院共同發表？

### 營運模式：賣晶片還是賣系統；專利屏障

- 問題：營運模式是賣晶片還是賣系統？目前未看到明確專利屏障。
- 可追問方向：
  - 收入主軸是一次性晶片銷售、系統授權、檢測服務，還是混合模式？
  - MedSelect 是軟體平台、硬體系統，還是檢測流程品牌？
  - 若專利不足，是否有製程 know-how、資料庫、法規認證或合作網絡形成門檻？
  - 是否已有申請中專利或未公開的 IP 佈局？

### 團隊組成（目前只看到兩個人）

- 問題：團隊組成如何？目前只看到兩位成員。
- 可追問方向：
  - 兩位是否為對外揭露的創辦核心，而非完整團隊？
  - 是否另有研發、臨床、法規、BD 或顧問團隊？
  - 是否透過 CRO、醫院、學研單位或代工夥伴補足能力？
  - 現階段組織是否能支撐產品驗證、商務拓展與法規進程？
