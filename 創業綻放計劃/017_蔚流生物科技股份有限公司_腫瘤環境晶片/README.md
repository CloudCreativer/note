# 017 蔚流生物 腫瘤環境晶片

## 一句話摘要

蔚流以 MedSelect 腫瘤微環境晶片，在 AI 篩選後、GLP 動物實驗前驗證 ADC／雙抗等功能性；核心銷售設備與晶片，並以 CRO 平行測試推動市場轉換。

## 評審提問與回答

### Q1. 準確度 85% 到 92% 是根據多少次測試、多久得到的數據？

我們跟新竹、台中、廣州、台大平鎮等地醫院合作：收集患者手術檢體，在醫師治療**之前**對晶片上的腫瘤模型做測試，再對照病人實際治療後結果。晶片上有效、病人身上也有效，就定義為相關。分母約 **30–40 例**，分子落在 **85–92%**。意義是：新藥開發經此層測試後，對下一階段臨床試驗較有信心。

### Q2. 你們到底要賣晶片、帶檢驗，還是銷售檢驗器？也沒看到專利品狀。

我們定義自己是**設備開發商**，核心目標是銷售**設備 + 晶片**。先做 CRO／平行測試服務，是因為藥廠數十年慣用動物實驗，需用平行方式降低轉換阻力；服務是市場教育策略，長期仍以設備銷售為主。專利佈局在**晶片設計與架構**（缺氧、纖維化、血管床等模擬及方法學），簡報建築物章節有提及。

### Q3. 請再說明團隊組成。

對外兩位為主要經營角色（含現場秀君，具 MBA／財務背景）；其餘多為科學家與早期行銷合作夥伴。此階段最重要的是說服藥廠採用平行、非動物替代方案，科學家網絡是核心資產。

### Q4. 前面案例選了親和性很低的兩個藥，親和性跟這個測試有什麼關係？

那是**胰臟癌**案例：胰臟癌有強缺氧與纖維化，治療困難。我們要回答的是：選出的藥在某劑量下 OK，超過則可能產生**免疫介導的全身性肝毒性**。晶片可培養肝臟相關細胞，驗證肝毒性與合理劑量窗口，不只是「藥有效」，還要劑量與安全性可往下推。
