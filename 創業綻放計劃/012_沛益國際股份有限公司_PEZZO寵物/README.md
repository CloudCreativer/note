# 012 沛益國際 · Petzzo

## 30 秒看懂

沛益國際（品牌 **Petzzo**）要做的是：**寵物的身分證 + 健康檔案平台**。  
用臉部辨識、晶片號碼與就醫紀錄整合成 **Pet ID**，讓飼主在 App 管理病歷，讓動物醫院用 SaaS 接診，並讓產險更快、更準地核對「這隻毛孩是不是保單上的那一隻」。  
技術源自工研院孵化，現以 **B2C 訂閱** 切入（早期曾走 B2B 產險，因台灣寵物險市場太小而轉向）。

## 解決什麼問題

### 飼主端

- 看獸醫前不知道要花多少、病歷分散、不敢常帶毛孩就醫。
- App 可累積**所有寵物的病歷**，提高黏著度（團隊稱「病歷在裡面就不易卸載」）。

### 產險端

- 台灣寵物險產品線少（現場稱全台約 **4 家**），產險常虧損仍須推廣。
- 理賠依賴紙本、耗時；**身分難核對**（僅晶片號、缺飼主／臉部綁定），有道德風險（詐保、替換寵物）。
- Pet ID 可提供較完整資訊，縮短理賠與核保負擔（團隊承認無法单靠辨識完全防詐）。

### 醫院端

- 需能對接投保客戶、開立電子診斷書、配合理賠流程（官網有合作醫院與 PetID 理賠流程說明）。

## 產品怎麼運作（Pet ID）

1. 飼主在 App 為寵物建檔（臉部照片、晶片等）。  
2. 就醫或申請理賠時，以臉部特徵（上百～上千維）比對相似度。  
3. **臉部 alone 不能當唯一身分依據**；正式身分 = 臉部圖資 + 晶片號碼 + 醫院病歷 → **Pet ID**。  
4. 醫院／產險透過 Pet ID 核對同一隻寵物，加速電子診斷書與理賠。

（與早期東詣／Petzzo 新聞敘事一致：臉部辨識 + 區塊鏈／雲端病歷縮短理賠；現法人為新設的沛益國際，技術脈絡來自工研院移轉。）

## 評審提問與回答

### Q1. 你們的 AI／模型是怎麼服務的？另外簡報上的 3D 臉部辨識，技術內容與應用範圍能否再具體說明？

我們做的是寵物臉部辨識。在注射、就醫等情境，用不同照片計算臉部特徵值（可達上百到上千個維度），用來確認寵物相似度。但這**不能**單獨作為身份的唯一依據。我們真正做的是：臉部圖資 + 晶片號碼 + 醫院病歷，整合成 Pet ID，用這個 ID 輔助所有寵物身份辨識。

### Q2. 衛福部（逐字稿誤聽為「地院處」）對寵物相關規定，是否有必要做你們所說的這套機制？

（此題在逐字稿中未聽到團隊直接回答。）

### Q3. 公司 114 年 12 月 20 日設立，專利原為工研院移轉的「快速保險與理賠系統」——對公司而言，移轉目的是什麼？

最早寵物辨識技術來自工研院（前公司孵化）。在前公司做孵化時發現，只做寵物店無法變現，必須找應用；最早與明台產險等合作，但全台寵物險產線僅四家、市場太小，因此改為 B2C、面向全台與海外飼主的訂閱模式，並獨立成新創公司。專利移轉是為了在保險／理賠場景落地這套技術。

### Q4. 經濟部資料顯示您投資 150 萬、公司資本額 620 萬約占 24%，但簡報寫創辦人持股 43.92%，差異在哪？

簡報所列 43.92% 是**注資之後**的持股。目前已獲得注資（逐字稿為「五感集團 2,000 萬」，投資方全名以簡報為準）。另含特別股等表決權安排，合計表決權約 74%（現場說法）。

### Q5. 你拿日本寵物保險數據比較，但日本寵物飼養成本高才有投保需求；台灣為何有急迫性？又如何預防道德風險（如寵物詐保）？

亞洲各地投保率：日本最高（現場稱約 14%），香港、新加坡等也超過 5%；台灣目前低於 3%，我們認為有成長空間。日本投保時常規定必須同保、辨識要求高；台灣多只有晶片號碼、缺乏飼主資訊。我們用 Pet ID（臉部 + 醫療地理資訊 + 晶片）補足，讓產險有較完整資訊，有助降低理賠風險；但團隊也承認，無法单靠辨識完全杜絕詐保，仍須搭配其他機制。

### Q6. 你們串聯技術平台與保險，若平台業者也能切入，你們的技術優勢與商業模式是什麼？

我們從醫療切入：飼主下載 APP 後可累積病歷，不易卸載。醫院端有免費版／收費版 SaaS，依健康管理、醫療服務分級收費，並可搭配健檢或醫療附加險。產業痛點上，我們要解決的是：飼主就醫費用不透明、怕看醫生；產險端產品線虧損卻仍需推廣——希望透過訂閱與 Pet ID 機制，讓飼主願意使用，並降低產險回覆／理賠負擔。

### Q7. 產業痛點是針對金融機構還是飼主？服務對象除了貓狗，是否包含其他寵物？

痛點同時在**飼主**（就醫、費用、病歷）與**產險**（理賠、風險）。現場回答以貓狗情境為主，未細述其他物種。
