# 025 艾斯創生醫股份有限公司 手術系統擴張

## 一句話摘要

艾斯創生醫股份有限公司的主題聚焦於手術系統商業化與擴張，當前重點在釐清美國健保使用與給付機制、海外拓展地區、資本擴張計畫、手術系統外的產品延伸，以及從阿基里斯腱場景往其他適應症擴張的策略。

## 背景

- 主題公司：艾斯創生醫股份有限公司
- 主題方向：手術系統相關醫療應用
- 已知關注點：美國給付、海外市場、資本擴張、產品線與適應症擴張
- 現階段目標：建立法規/給付路徑與商業成長策略的驗證框架

## 重點整理

### 產品與商業方向

- 手術系統（具體功能與適用術式待確認）
- 目前主要聚焦阿基里斯腱相關場景
- 可能延伸到其他骨科或軟組織手術應用（待確認）

### 初步觀察

- 醫材商業化關鍵在「法規核准 + 給付路徑 + 臨床採用」
- 美國市場成功與否，常取決於 CPT/DRG、保險支付與醫院採購邏輯
- 若目前聚焦單一術式，需明確擴張路線才能支撐長期估值
- 資本擴張計畫應對應臨床、法規、商業里程碑，而非單純募資金額

## 關鍵結論

- 這份筆記目前屬於醫材市場進入與規模化問題整理階段
- 後續應優先補齊美國給付策略、海外市場優先順序與適應症擴張藍圖
- 手術系統外的第二成長曲線需及早定義，降低單一產品風險
- 資本計畫需綁定可驗證的階段成果與現金流需求

## Q&A

### 美國健保使用與給付

- 問題：在美國健保體系如何使用？給付狀態如何？
- 可追問方向：
  - 對應的是 Medicare、Medicaid 還是商業保險？
  - 是否已有對應的 CPT/HCPCS/DRG 代碼或既有代碼可套用？
  - 給付是全額、部分，還是需醫院自付資本支出？
  - 臨床經濟證據（縮短手術時間、降低併發症）是否足以支持給付談判？

### 海外拓展：除了美國還有哪裡

- 問題：海外拓展除了美國還有哪裡？
- 可追問方向：
  - 優先市場是日本、歐洲、東南亞，還是中東？
  - 各市場的法規門檻（FDA/CE/PMDA 等）與進入時程為何？
  - 擴張模式是直營、經銷商、醫療集團合作，還是授權？
  - 各市場是否已有臨床合作或試點醫院？

### 資本擴張計畫

- 問題：資本擴張計劃是什麼？
- 可追問方向：
  - 下一輪募資金額、用途（臨床、法規、商業化、製造）如何配置？
  - 現金 runway 與達成下一里程碑所需資金缺口？
  - 是否有策略投資人（醫療器材、醫院體系、通路）導入？
  - 資本策略是快速擴張還是分階段驗證？

### 手術系統外是否有其他發展

- 問題：手術系統之外還有其他發展嗎？
- 可追問方向：
  - 是否有軟體服務、術前規劃、術後追蹤或耗材型收入？
  - 是否開發周邊模組形成完整解決方案？
  - 第二產品線是否已進入研發/驗證/上市階段？
  - 新產品與現有系統是否共用通路與客戶基礎？

### 為何只在阿基里斯腱，是否有其他擴張

- 問題：為什麼目前只在阿基里斯腱？還有其他擴張嗎？
- 可追問方向：
  - 阿基里斯腱是否作為法規與臨床切入點（narrow indication strategy）？
  - 下一步會延伸到哪些術式或部位？時間表是什麼？
  - 擴張到新適應症需要重新臨床驗證或法規送件嗎？
  - 跨適應症後，市場規模與單位經濟如何改變？
