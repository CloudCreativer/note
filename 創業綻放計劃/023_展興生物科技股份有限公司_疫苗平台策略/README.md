# 023 展興生物科技股份有限公司 · 疫苗平台策略

## 30 秒看懂

展興生物聚焦動物疫苗平台，核心敘事是以昆蟲桿狀病毒系統切入豬隻疫苗，並延伸到多疾病管線。評審 QA 的焦點在：專利歸屬與保護策略、平台與產品關係、藥證與食安安全性、與進口疫苗競爭差異、為何優先做 PED 而非非洲豬瘟、量產可行性，以及公司近年具體里程碑。

## 他們在做什麼

- 定位：動物疫苗研發公司，主打平台型技術 + 產品化。
- 技術主軸：以桿狀病毒/表面抗原展示概念建立疫苗系統（逐字稿口述）。
- 首要產品：以豬流行性下痢（PED）疫苗為先行產品，並提及後續 PRRS、非洲豬瘟等方向。
- 團隊敘事：研究背景強，並強調與藥廠/生產端合作。

## 解決什麼問題

- 傳統動物疫苗在成本、免疫保護與安全性上常有取捨。
- 豬場重大傳染病（如 PED、PRRS、非洲豬瘟）需要更穩定且可規模供應的方案。
- 國內高度依賴進口疫苗，供應與自主性風險高。
- 研發到量產之間存在「法規 + 製造 + 通路」三重門檻。

## 怎麼賺錢

- 商業模式（逐字稿口述）：
  - 產品銷售（自有疫苗產品）。
  - 技術授權（平台能力授權）。
  - 平台合作（與大型藥廠共同開發/導入）。
- 收入邏輯：先以先行產品打開市場，再用平台擴增適應症與合作案。
- 量產策略：前端原液/原包由團隊掌握，後段量產與有經驗疫苗廠協作。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 先行產品已進入藥證審查相關階段。
  - 提到第三方驗證與期刊資料作為技術背書。
  - 提及已吸引國際藥廠合作興趣。
  - 國內外通路/生產合作（含代工與養殖端合作）正在推進。
- 評審追問集中：
  - 專利歸屬與保護力是否足夠。
  - 為何先做 PED 而非非洲豬瘟。
  - 安全性與食安關聯是否完整驗證。
  - 量產落地是否已具體可行。

## 評審提問與回答

### Q1. 你們說的專利到底是什麼？是否只有在中研院端？
- 團隊回答：核心專利與中研院相關，並說明公司已取得可執行/使用安排；另有多國佈局（逐字稿口述含三國等字樣）。
- 正確性註記：`中信心`
  - 理由：回答方向一致，但專利號與法律關係需文件核對。

### Q2. 為什麼不是先做非洲豬瘟，而是先做 PED？
- 團隊回答：曾評估非洲豬瘟方向，但受主管機關對實驗開放條件限制，先以可執行且市場明確的 PED 推進。
- 正確性註記：`中高信心`
  - 理由：回覆有明確政策限制脈絡，邏輯完整。

### Q3. 藥證核可與食安如何確保？
- 團隊回答：申請藥證流程包含安全性相關檢測（例如局部反應、殘留/內臟規範等），通過後才可進入上市。
- 正確性註記：`中信心`
  - 理由：說法合理，但各項檢測指標與結果未展示。

### Q4. 國產疫苗如何跟國外進口競爭？
- 團隊回答：現況進口占比高，團隊主張以新技術路徑提升效果與差異化，並以本土化供應提升競爭力。
- 正確性註記：`中低信心`
  - 理由：方向正確，但缺頭對頭效價/成本/供應穩定性數據。

### Q5. 平台與產品之間如何分工？營業秘密如何保護？
- 團隊回答：透過「專利 + know-how」雙軌保護；產品可銷售，平台可授權，形成雙收入模型。
- 正確性註記：`中信心`
  - 理由：商業邏輯清楚，仍需補 SOP、權限與合作 NDA 的制度細節。

### Q6. 量產規劃可行嗎？是否已有合作方？
- 團隊回答：已有具多年經驗的疫苗廠合作量產，團隊掌握前段原液，後段由合作方支援規模生產。
- 正確性註記：`中信心`
  - 理由：有明確分工描述，但尚缺產能、良率、交期指標。

### Q7. 你們早期獲獎在 2012，近五年有什麼新突破？
- 團隊回答：補充近年仍有新計畫/獎項與資金支持，並非只有早期成果。
- 正確性註記：`中低信心`
  - 理由：逐字稿對年份與名詞辨識噪音高，建議以正式資料表再對齊。

## 我學到的 3 件事

1. 生技簡報要同時說清楚「可做」與「可量產」，缺一不可。
2. 疫苗公司的評審攻防點通常在法規可行性與量產供應鏈，不只技術本身。
3. 先行產品的選擇常由政策開放度決定，而非技術興趣決定。
