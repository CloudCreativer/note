# 023 展興生物科技股份有限公司 疫苗平台策略

## 一句話摘要

展興生物科技股份有限公司的主題聚焦於疫苗與生技平台策略，目前重點在釐清多國專利內容與豬以外研究、平台與產品分階段邏輯、營業秘密保護、藥證與食安關係、疫苗自主與國際競爭、董事長專利與 PVB 路線、產業經營策略、量產規劃，以及 2012 獲獎後的重要突破。

## 背景

- 主題公司：展興生物科技股份有限公司
- 主題方向：生技／疫苗相關平台與產品策略
- 已知關注點：多國專利、非洲豬瘟專利、PVB 方向、藥證/食安、量產與競爭
- 現階段目標：建立技術、法規、商業化與里程碑的驗證框架

## 重點整理

### 產品與技術方向

- 疫苗自主與相關技術開發
- 可能同時存在平台能力與單一產品線
- PVB（具體全名與產品定義待確認）

### 初步觀察

- 生技公司常見「平台＋產品」雙軌策略，需避免敘事混淆
- 多國專利需拆解為「保護範圍、地域、商業可執行性」
- 藥證核可與食安合規屬不同監管框架，必須清楚分流
- 若董事長有 ASF（非洲豬瘟）相關專利但主做 PVB，需解釋資源配置邏輯
- 獲獎年份與公司成立／突破時間差可作為成熟度判斷線索

## 關鍵結論

- 這份筆記目前屬於生技公司技術路線與商業化驗證問題整理階段
- 後續應優先補齊專利地圖、平台/產品分期與法規路徑
- 量產規劃與國際競爭策略是決定估值與成長性的關鍵
- 里程碑需串起：2012 獲獎 → 後續突破 → 現階段商業落地

## Q&A

### 多國專利是什麼？豬以外研究

- 問題：多國專利是哪些？除了豬以外有什麼研究？
- 可追問方向：
  - 專利涵蓋哪些國家、技術領域與用途？
  - 是核心製程、抗原平台、佐劑、遞送系統，還是應用專利？
  - 豬以外是否已布局其他物種或人用醫療方向？
  - 各國專利是否已核准或仍在申請中？

### 平台？產品？分什麼階段區分

- 問題：是平台還是產品？如何分階段區分？
- 可追問方向：
  - 平台層（技術能力）與產品層（適應症/物種）如何切分？
  - 現階段 KPI 是科研驗證、法規送件，還是銷售導入？
  - 每階段的商業模式是否不同（授權、共同開發、自有銷售）？
  - 對投資人如何溝通「平台價值」與「短期收入」平衡？

### 營業秘密如何保護產品

- 問題：營業秘密如何保護產品？
- 可追問方向：
  - 哪些核心 know-how 不適合申請專利而改用營業秘密保護？
  - 是否有完整權限控管、NDA、內部流程與技術切片管理？
  - 與外部合作（學研、代工、臨床）時如何避免外流？
  - 若離職或合作終止，如何確保技術不被帶走？

### 藥證核可是否有符合食安

- 問題：藥證核可是否符合食安？
- 可追問方向：
  - 藥證（動物用藥／疫苗）與食品安全規範的適用邊界為何？
  - 產品屬於藥品、疫苗、飼料添加，還是其他法規分類？
  - 是否已有主管機關審查、檢測與上市後監測機制？
  - 對客戶端（農場／通路）是否提供合規文件與追溯資料？

### 疫苗自主與國外競爭

- 問題：疫苗自主，如何與國外競爭？
- 可追問方向：
  - 競爭優勢是成本、供應穩定、在地需求匹配，還是技術效果？
  - 與國外大廠相比，速度、效價、安全性與可得性差異？
  - 市場策略是進口替代、利基市場，還是共同開發輸出？
  - 是否有國際驗證或跨國合作提升可信度？

### 董事長有非洲豬瘟專利，為何做 PVB

- 問題：董事長有非洲豬瘟專利，為什麼反而做 PVB？
- 可追問方向：
  - PVB 與 ASF 之間是技術延伸、商業轉向，還是資源優先順序調整？
  - 是市場時機、法規障礙、成本報酬比，還是策略聚焦所致？
  - 現在 PVB 是否是短期收入引擎，ASF 是中長期布局？
  - 兩條路線是否共用平台能力，避免研發重複投入？

### 產業特性的經營策略

- 問題：生技產業特性下，公司的經營策略是什麼？
- 可追問方向：
  - 如何管理長研發周期與現金流壓力？
  - 合作策略偏重授權、聯盟、代工，還是自建通路？
  - 在法規不確定與技術風險下，如何做產品組合與資源配置？
  - 是否建立里程碑式募資與分階段商業化計畫？

### 量產規劃

- 問題：量產規劃為何？
- 可追問方向：
  - 量產是自建廠、委外生產，還是混合模式？
  - 產能目標、良率、品質一致性與放大量產節點為何？
  - 是否已完成 GMP 或相關製造標準準備？
  - 上市後供應風險（原料、冷鏈、交期）如何管理？

### 獲獎 2012 與 5 年內公司規定，後續突破

- 問題：2012 年獲獎，且比賽規定為 5 年內公司，後續有什麼重要突破？
- 可追問方向：
  - 獲獎後關鍵里程碑（技術、法規、商業、募資）有哪些？
  - 是否從原型走到驗證，再到商業化落地？
  - 重要突破是否對營收、合作或市場地位產生實質影響？
  - 是否存在「早期高光、後續停滯」的風險信號？
