# 002 皓準科技股份有限公司 · 自行車齒輪創新

## 30 秒看懂

皓準科技主打「非正圓客製化齒盤」，以人體力學與騎乘數據做個人化設計，宣稱可提升踩踏效率（評審追問 2.7% 效益）。決賽 QA 核心在：效益是否可驗證、客製化能否規模化、海外與 ToB 進展是否足以支撐成長。

## 他們在做什麼

- 產品定位：以使用者左右腳力量差異與踩踏相位差，設計客製化齒盤。
- 技術敘事（依逐字稿）：
  - 以感測器/功率數據建立模型，將傳統圓盤改為個人化輪廓。
  - 結合資料分析與流程化製造，縮短客製交期。
- 應用客群：一般騎乘者、競技選手、車店通路；後續有復健運動延伸構想。

## 解決什麼問題

- 騎乘時左右腳輸出不均，導致效率下降與疲勞累積。
- 長距離騎乘（如長途活動）常見膝部/肌群負擔。
- 高度客製產品常見交期長、教育成本高、消費者難理解效益。

## 怎麼賺錢

- 逐字稿可辨識模式：
  - 產品銷售（單片售價口述約 USD 300，約台幣 1 萬）。
  - 通路合作（與車店合作導流、量測、交付）。
  - 海外銷售（北美通路口述已有合作）。
- 逐字稿口述營運數據：
  - 去年營收約新台幣 6,000 萬。
  - 毛利率現況約 82%，擴張期目標約 70%（因教育市場與海外拓展投入）。

## 實績與進展（決賽現場）

- 可辨識實績（逐字稿）：
  - 北美已有合作車店（口述約 3 家）。
  - 產品線已涵蓋不同車種場景。
  - 交付流程主打「量測快、交付快」（口述台灣可當天交付）。
- 評審關注重點：
  - 2.7% 效益的測試設計、樣本與可重現性。
  - 客製化數據是否能沉澱為可規模化能力。
  - 海外市場是否已形成穩定 B2B 成長引擎。

## 評審提問與回答

### Q1. 2.7% 效益怎麼驗證？售價與收費模式為何？
- 評審重點：效益的量測方法與商業可行性。
- 團隊回答重點（逐字稿）：
  - 以 input/output 概念與人體實測做比對，主張非正圓盤可改善輸出效率；
  - 單片售價口述約 USD 300（約台幣 1 萬）。
- 正確性註記：`中低信心`
  - 原因：缺少標準化實驗設計、樣本量、統計顯著性。

### Q2. 海外（尤其美國）ToB 進展如何？
- 評審重點：是否有穩定通路與放大量能。
- 團隊回答重點（逐字稿）：
  - 目前北美合作車店口述約 3 家；
  - 未來希望擴展日本與其他市場。
- 正確性註記：`中信心`
  - 原因：有具體口述，但缺合作合約與區域營收占比。

### Q3. 客製化數據非通用，如何規模化？
- 評審重點：個人化是否阻礙放大。
- 團隊回答重點（逐字稿）：
  - 透過資料累積持續優化模型；
  - 以車店流程標準化降低導入成本。
- 正確性註記：`中信心`
  - 原因：方向合理，但缺模型版本管理與再訓練制度說明。

### Q4. 使用者怎麼知道真的比較快/比較省力？
- 評審重點：終端體驗與可驗證性。
- 團隊回答重點（逐字稿）：
  - 使用者可感受踩踏更順、力量浪費減少；
  - 兼具體感與數據驗證敘事。
- 正確性註記：`中低信心`
  - 原因：偏主觀體感敘述，缺第三方客觀量測報告。

### Q5. 為何毛利從 82% 目標降到 70%？
- 評審重點：財務品質是否惡化。
- 團隊回答重點（逐字稿）：
  - 非產品本質惡化，而是擴張期投入市場教育、海外法規/稅務與通路成本。
- 正確性註記：`中信心`
  - 原因：邏輯一致，但需未來損益表驗證投入產出比。

### Q6. 人的肌力會變，舊客製結果是否失效？
- 評審重點：客製產品生命週期。
- 團隊回答重點（逐字稿）：
  - 選手會依賽段與體能變化更換多片齒盤；
  - 產品有再量測與再客製需求。
- 正確性註記：`中信心`
  - 原因：符合競技情境，但一般消費者情境資料不足。

## 我學到的 3 件事

1. 運動科技產品若主打「效率提升」，必須先有可重現的測試框架。
2. 客製化不是不能規模化，但一定要流程標準化與資料產品化。
3. 海外擴張初期毛利下修常見，關鍵在後續能否換到穩定成長曲線。
