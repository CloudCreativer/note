# 007 炬智科技 金屬檢測

## 30 秒看懂

炬智科技主打精密製程的金屬檢測方案，核心敘事是以高靈敏檢測能力切入半導體/高階製造場景。現階段重點是驗證三件事：客戶願不願意導入、設備成本是否有經濟性、以及技術在半導體流程的實際風險與可複製性。

## 他們在做什麼

- 聚焦精密製程中的金屬成分/污染檢測。
- 目標客群推測包含高階製造與半導體供應鏈。
- 產品可能包含檢測設備 + 分析流程 + 導入服務。

## 解決什麼問題

- 傳統檢測在速度、靈敏度、或線上化程度可能不足。
- 客戶痛點通常是良率風險與異常發生後追查成本高。
- 若可提早檢出關鍵金屬偏差，可降低報廢與停線風險。

## 怎麼賺錢

- 可推定商模：
  - 設備銷售（一次性）；
  - 維護/耗材/校正（持續性）；
  - 導入驗證與客製服務（專案型）。
- 關鍵商業變數：
  - 單台售價與客戶回收期；
  - 客戶採購台數（單線/單廠/多廠）。

## 實績與進展（決賽現場）

- 逐字稿可辨識重點：
  - 儀器已進入台灣高階載板相關產線做中程驗證。
  - 已與多家載板廠、藥水供應商與新製程合作方接觸/合作。
  - 商模含整機銷售、耗材、年度維保，後續規劃資料服務。
  - 團隊口述目前已售出 7 台設備（學研與產業場域皆有）。

- 建議優先補件：
  - 技術原理頁；
  - 客戶驗證/試用頁；
  - 成本與回收期試算頁；
  - 半導體應用流程頁。

## 評審提問與回答

### Q1. 哪些大廠在試用？目前到哪個階段？
- 評審重點：客戶是否已從驗證走向導入。
- 團隊回答重點（逐字稿）：
  - 切入高階載板最後一道關鍵製程；
  - 多家目標客戶正在驗證中，已有設備進場。
- 正確性註記：`中信心`

### Q2. 設備成本多少？客戶回本邏輯怎麼算？
- 評審重點：設備成本、售價與回收期。
- 團隊回答重點（逐字稿）：
  - 口述機台成本約百萬級距；
  - 預計募資約 2,500~3,000 萬用於推進驗證與商業化。
- 正確性註記：`中低信心`（數字受 ASR 影響，需簡報原圖覆核）

### Q3. 為什麼還沒有外部投資？水位與募資規劃？
- 評審重點：資金來源與下一步融資。
- 團隊回答重點（逐字稿）：
  - 早期以創辦團隊自籌為主；
  - 下一步以外部募資放大產業導入。
- 正確性註記：`中信心`

### Q4. 半導體應用在哪個流程？風險是什麼？
- 評審重點：是否只適用單一場景。
- 團隊回答重點（逐字稿）：
  - 目前聚焦載板濕製程/表面處理關鍵環節；
  - 後續延伸封測、半導體相關流體監測場景。
- 正確性註記：`中信心`

### Q5. 這項檢測對 TA 的影響到底大不大？
- 評審重點：是否直接影響良率與報廢。
- 團隊回答重點（逐字稿）：
  - 目標是把檢測從離線慢回饋，拉到現場即時監控；
  - 可降低異常滯後造成的報廢與重工。
- 正確性註記：`中信心`

## 我學到的 3 件事

1. 精密檢測新創要拿到訂單，關鍵是「客戶回本模型」而不只技術性能。
2. 半導體場景導入難點常在長驗證週期與跨部門決策。
3. 評審最在意的是可規模化的商業證據，而非單一技術亮點。
