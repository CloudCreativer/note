# 007 炬智科技 金屬檢測

## 一句話摘要

炬智科技的主題聚焦於精密製程中的金屬檢測，核心技術為水下微電漿光譜法，目前重點在釐清試用大廠、設備成本、投資與募資計畫、典型客戶需求、晶圓半導體應用與風險、對目標客戶的影響程度，以及經濟部相關資訊的可見性。

## 背景

- 主題公司：炬智科技
- 主題方向：精密製程、水下微電漿光譜法、金屬檢測
- 應用場景可能涵蓋半導體／晶圓製程等需高精度金屬污染或成分檢測的環節
- 現階段目標是建立技術與商業驗證理解，並整理市場、財務與合規相關問題

## 重點整理

### 產品與技術方向

- 精密製程（使用者筆記亦寫「精密製成」）
- 水下微電漿光譜法
- 金屬檢測
- 可能延伸至晶圓／半導體製程場景

### 初步觀察

- 若已有大廠試用，代表技術可能進入驗證或導入前期，但尚未等同於量產訂單
- 設備／分析儀成本將直接影響客戶採購台數與回本邏輯
- 尚未有外部投資時，需同時釐清公司現金水位與未來募資計畫
- 半導體應用機會大，但法規、驗證周期與客戶認證風險通常也高
- 「經濟部沒有看到公司」可能涉及補助、計畫、登錄或公開資訊可查性，需進一步查證

## 關鍵結論

- 這份筆記目前屬於技術驗證與商業化問題整理階段
- 後續應優先補齊試用客戶、設備定價與典型客戶採購邏輯
- 投資與募資敘事需與公司水位、里程碑綁定
- 半導體應用需分開評估「技術適用性」與「導入風險」

## Q&A

### 哪一些大廠在試用

- 問題：目前有哪些大廠在試用？
- 可追問方向：
  - 試用階段是 PoC、驗證案、小量導入，還是已進入採購評估？
  - 客戶產業是半導體、面板、化工、水處理，還是其他？
  - 試用是否已產生可公開的案例或成效數據？

### 設備分析儀成本

- 問題：設備／分析儀成本多少錢？
- 可追問方向：
  - 單台售價、租賃方案、耗材與維護費如何計算？
  - 與傳統檢測方式相比，總持有成本（TCO）差異為何？
  - 價格是否因模組、檢測精度或客製化而大幅浮動？

### 為什麼還沒有外部投資

- 問題：為什麼還沒有外部投資？公司水位如何？未來募資計畫為何？
- 可追問方向：
  - 是策略上暫不募資、估值未談攏，還是商業驗證仍不足？
  - 現金 runway 可支撐多久？主要燒錢項目為何？
  - 下一輪募資的時間點、金額、用途與里程碑為何？

### 典型客戶 30 秒說明

- 問題：能否用 30 秒說明一個客戶為什麼需要這台設備？他需要多少台？
- 可追問方向：
  - 客戶痛點是速度、精度、成本、合規，還是產線瓶頸？
  - 單一廠區通常需要 1 台、數台，還是每條產線一台？
  - 採購決策者是品管、製程、設備還是採購部門？

### 晶圓半導體技術用在哪一塊

- 問題：晶圓／半導體技術具體用在哪一塊？風險在哪裡？
- 可追問方向：
  - 應用在製程哪個節點（例如清洗、蝕刻、CMP、廢液、進料檢驗）？
  - 風險是誤判、法規認證、客戶驗證周期，還是與既有設備商競爭？
  - 是否需通過客戶端的長期穩定性與再現性測試？

### 檢測對 TA 公司影響大不大

- 問題：這項檢測對目標客戶（TA）公司的影響大不大？
- 可追問方向：
  - 是「可有可無的優化」還是「影響良率／停線的關鍵檢測」？
  - 若導入失敗，客戶損失為何？若導入成功，效益如何量化？
  - 客戶是否願意為此調整既有 SOP 或產線配置？

### 經濟部沒有看到公司

- 問題：為什麼在經濟部相關管道「沒有看到」這家公司？
- 可追問方向：
  - 是指補助案、計畫申請、公司登記查詢，還是公開獎項／輔導名單？
  - 公司正式登記名稱是否與對外品牌名稱不同？
  - 是否透過其他法人、母公司或合作單位申請計畫？
