# 029 星相科技股份有限公司 · 晶片與天線策略

## 30 秒看懂

星相科技（`Phasetrum`）聚焦低軌衛星相控陣射頻解決方案，主張以 `AIP` 天線 + 相控晶片 + 量測校正，一次解決終端「成本高、良率低、量產慢」三大痛點。  
簡報可辨識其路徑是先切入衛星終端與供應鏈合作，再延伸到多元高成長通訊場景。

## 他們在做什麼

- 技術方向：相位陣列晶片架構設計，搭配天線與校正平台。
- 應用場景：低軌衛星終端相關設備（含地面終端/特定高頻通訊場景）。
- 商業模式：以晶片為核心，結合參考設計與整合能力，協助客戶落地量產。

## 解決什麼問題

- 傳統相位陣列系統常面臨成本高、校正複雜、量產不易。
- 單賣晶片難以解決客戶端「可量產」痛點，需要整合天線/校正流程。
- 台灣設備商切入低軌市場時，需要可本地協作的方案商以降低導入風險。

## 怎麼賺錢

- 收入來源（逐字稿口述）：
  - 晶片與關鍵模組出貨。
  - 量產校正與整合服務（含 ATE/量產導入支持）。
  - 與既有天線/設備廠協作，放大出貨機會。
- 核心邏輯：不只比單點晶片成本，而是比客戶整體量產效率與良率。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 技術敘事：從第 3 代晶片完成架構驗證，進入第 4 代量產化與商務導入。
  - 指標敘事：頁面揭露 `1/2` 尺寸功耗、`10x` 良率與校正速度提升、`+30dB` 模組化增益與 `$38B` 終端市場。
  - 商務敘事：合作鏈條含衛星營運商、軟體/Modem 夥伴、系統設計與 ODM/製造端。
- 評審追問焦點：
  - 相比國際大廠，差異是否足以驅動客戶更換。
  - 公司是賣晶片還是賣整套，邊界是否清楚。
  - 研發團隊是否足以支撐晶片與系統雙線交付。

## 評審提問與回答

### Q1. 你說晶片是心臟，和國際大廠差在哪？
- 團隊回答：差異在整體架構與量產導入能力，不只單點晶片性能；重點是可協助客戶解決量產校正與系統整合。
- 正確性註記：`中信心`
  - 理由：方向合理，但需更多量化 benchmark 支撐。

### Q2. 客戶憑什麼要換你們方案？
- 團隊回答：誘因在總體成本下降與導入速度提升，並可借助本地供應鏈協作降低專案風險。
- 正確性註記：`中信心`
  - 理由：商業邏輯清楚，仍缺公開轉換案例數據。

### Q3. 你們到底是開晶片賣晶片，還是做整合服務？
- 團隊回答：以晶片為核心出貨，同時提供整合與量產支援，讓客戶能真正落地。
- 正確性註記：`中高信心`
  - 理由：回覆一致，且符合產業常見 go-to-market 模式。

### Q4. 研發團隊有多少人？能同時撐住兩條線嗎？
- 團隊回答：團隊由學研與業界背景組成，並透過合作夥伴補足量產與系統端能力。
- 正確性註記：`中低信心`
  - 理由：逐字稿對人數辨識不穩，需正式資料核對。

### Q5. 晶片加天線的競爭優勢是什麼？
- 團隊回答：優勢在可共同優化，降低客戶整體導入難度，而非只賣單一元件。
- 正確性註記：`中信心`
  - 理由：具策略一致性，但待補場域驗證數據。

## 我學到的 3 件事

1. 在高頻通訊市場，客戶買的是「可量產方案」不是「單顆晶片規格」。
2. 台灣新創要對抗國際大廠，最有效的是系統整合與在地供應鏈速度。
3. 量產校正能力若能產品化，會是長期護城河。
