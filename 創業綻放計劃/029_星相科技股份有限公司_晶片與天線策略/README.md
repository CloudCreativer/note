# 029 星相科技股份有限公司 晶片與天線策略

## 一句話摘要

星相科技股份有限公司的主題聚焦於晶片與天線策略，當前重點在釐清「晶片是心臟」的差異化來源、客戶換用誘因、開發與出貨責任邊界、研發團隊規模，以及晶片搭配天線的整體競爭優勢。

## 背景

- 主題公司：星相科技股份有限公司
- 主題方向：晶片與天線相關產品與服務
- 已知關注點：競品差異、客戶轉換動機、出貨範圍、團隊規模與技術優勢
- 現階段目標：建立技術定位與商業化路徑驗證框架

## 重點整理

### 產品與商業方向

- 晶片開發與商業化
- 天線整合能力（可能是系統級差異化關鍵）
- 可能同時有晶片出貨與設計服務模式

### 初步觀察

- 「晶片是心臟」代表核心價值在效能、功耗、穩定性或可整合性
- 客戶是否換用，取決於遷移成本與實際效益差距
- 出貨與協助設計的邊界需清楚，否則營收與毛利判讀容易失真
- 晶片加天線若能整體優化，可形成系統級護城河

## 關鍵結論

- 這份筆記目前屬於半導體產品定位與商業模式問題整理階段
- 後續應優先補齊競品對照、轉換誘因與收入結構
- 研發團隊規模與組成將直接影響迭代速度與交付能力
- 晶片與天線應以組合價值而非單點性能評估競爭力

## Q&A

### 晶片是心臟，差異在哪裡

- 問題：晶片是心臟，與競爭對手差異在哪裡？
- 可追問方向：
  - 差異是性能、功耗、成本、穩定性，還是可量產性？
  - 是否有關鍵 IP、架構或製程優勢可量化？
  - 與主流競品相比，核心指標提升幅度為何？
  - 差異是否已在實際客戶場景中被驗證？

### 客戶為什麼願意換

- 問題：誘因是什麼？為什麼客戶願意換？
- 可追問方向：
  - 客戶切換可獲得哪些明確收益（成本、效能、供應穩定）？
  - 導入門檻與切換成本（軟硬體改版、認證、驗證）多高？
  - 是否提供 PoC、參考設計或遷移工具降低導入風險？
  - 目前換用案例是否能提供轉換周期與效益數據？

### 開晶片與賣晶片：出貨 vs 協助設計

- 問題：開晶片、賣晶片哪些是公司出貨，哪些是協助設計？
- 可追問方向：
  - 收入分為 IC 出貨、NRE、設計服務、授權，比例如何？
  - 產品型（標準品）與專案型（客製設計）如何分工？
  - 協助設計是否會轉化為後續長期出貨？
  - 哪一類收入毛利較高、可擴展性較強？

### 研發團隊成員有多少

- 問題：公司研發團隊成員有多少個？
- 可追問方向：
  - 團隊人數、職能分布（IC 設計、RF、天線、驗證、韌體）如何？
  - 是否有關鍵人才集中風險？
  - 目前人力是否能支撐產品路線圖與客戶專案節奏？
  - 是否有外部合作（學研/代工/顧問）補足缺口？

### 晶片、天線競爭優勢

- 問題：晶片與天線整體競爭優勢是什麼？
- 可追問方向：
  - 優勢是系統整合、RF 效能、尺寸、功耗，還是部署便利性？
  - 晶片與天線是否共同優化，形成競品難以複製的組合能力？
  - 是否有場域驗證數據（訊號品質、穩定度、耗能）？
  - 優勢是否可延伸到多產業應用場景？
