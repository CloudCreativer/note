# 王子娘 · 政府政策性貸款

> 阿金 ! 人生進化中 · 王子娘 · 不代辦、不抽成，教學正確申請政府低利資金

## Insight

政策性貸款 = **政府方案 + 有統編的企業貸款**：利率 0–2.295%、可接受內帳、技能導向（非人脈）、一生可重複申請（如青創）。三大方案：**青創（18–45、最長 20 年）**、**鳳凰（男女皆可、前 2–3 年零利率）**、**中小微（100 萬內信保 100%、多公司可分開申）**。實戰：**Q1 黃金期**、先貸款再補助、**沒店面反而較易過**、現場訪查約最忙時段、營收勿暴衝、資金用途合法。避開代辦詐騙（7–8 萬手續費 + 人頭戶風險）。

## 各章筆記

## 一、核心觀念與講師背景

### 1. 講師背景：王子娘

- **資歷**：擁有 16 年銀行企金審核實戰經驗、28 張金融證照。
- **理念**：堅持「不代辦、不抽成」，只做教學。因為代辦公司通常收取高額手續費（7-8萬以上），甚至可能讓創業者淪為詐騙人頭戶。
- **初衷**：因摯友在疫情期間資金周轉不靈，誤信詐騙代辦導致悲劇，決心出來教育社會大眾如何正確跟政府申請資金 。

### 2. 什麼是「政策性貸款」？

- **定義**：指「政府的方案」加上「有統一編號的企業貸款」。
- **資格**：需有商業登記（商號、行號、企業社、店面、有限公司、股份有限公司皆可） 。
- **特點**：
    - **公益性質**：利率極低（0% ~ 2.295%），甚至有政府利息補貼。
    - **銀行態度**：對行員來說這類貸款「沒有業績」，屬配合政府政策。
    - **意義**：通過政策性貸款代表政府認可你的事業至少能存活 6 年（貸款年限通常最少 6 年）。

## 二、政策性貸款 vs. 一般企業貸款

這張表非常重要，決定了你的資金成本與申請難度：

| **比較項目** | **政策性貸款 (推薦)** | **一般企業貸款** |
| --- | --- | --- |
| **利率** | **0% ~ 2.295%** (不超過 3%)  | **3% ~ 14%**  |
| **額度** | 安全槓桿，額度合理 | 利率越高，額度可能越高  |
| **費用** | **低** (100萬約1萬內，含信保與聯徵費)  | **高** (開辦費約 5~8 萬)  |
| **財報要求** | **可接受無報稅/內帳**  | 必須有正式報稅財報  |
| **過件關鍵** | **技能導向** (靠自己送件，全台皆可)  | **人脈導向** (靠關係，難以複製)  |
| **違約金** | **基本無違約金**  | 前一年違約金通常很高  |
| **利息補助** | **有** (最長可達 5-7 年)  | 無  |
| **資本門檻** | **最低 10 萬** 即可申請  | 基本門檻通常需 100 萬  |
| **營收比** | 可達 **50%** (法規上限)  | 通常僅 30% ~ 35%  |
| **新興產業** | **支持** (如電商、AI、網拍)  | 較保守，看分行經理喜好  |

## 三、熱門貸款方案詳解

### 1. 青年創業貸款 (青創)

- **年齡限制**：18 歲至 45 歲。
- **年限優勢**：中央（經濟部/文化部）青創貸款年限最長可達 **20 年**（比照房貸規格）。
- **申請次數**：一生**不限一次**，只要還款正常可重複申請 。
- **台北市青創 (特殊)**：福利極好，但要求戶籍在台北市滿一年、公司登記在台北市滿一年。

### 2. 微型創業鳳凰貸款 (鳳凰)

- **適用對象**：**男女皆可**（鳳=男，凰=女），不僅限女性。
- **特色**：審核流程較長（3-5個月），需經過層層關卡 。
- **優惠**：一般人前 2 年零利率；若為特殊身分（離島、單親、家暴受害等弱勢）則前 3 年零利率。
- **限制**：名下只能有一家公司擔任負責人。

### 3. 中小微企業多元發展方案 (中小微)

- **現狀**：目前最熱門、銀行最願意承做的方案（政府力推）。
- **優勢**：
    - **信保成數高**：100萬以內通常是 100% 保證，銀行風險低，敢放款。
    - **多間公司**：同一負責人若有兩間公司（甲、乙），可**分別**各申請一次（例如各 350 萬），額度分開計算。
    - **年齡/年資**：不限 45 歲，公司設立超過 5 年、10 年也都能申請。

## 四、申請實戰技巧與地雷

### 1. 申請時機 (Timing)

- **最佳時機：Q1 (1-3月)**。因為政策性貸款對銀行無業績，銀行傾向在年初趕快把政府交代的額度消化完，之後專心做有業績的案子。
- **順序建議**：先申請 **「創業貸款」**，再申請 **「創業補助」**。因為補助要求較高（如資本額需全額到位、員工數等），先拿到貸款資金會比較好運作。

### 2. 店面迷思

- **真相**：**沒有店面**比有店面更容易過件 42。
- **原因**：銀行與政府看重「成本控管」。有店面代表高租金、水電、裝潢與人事成本，倒閉風險較高。

### 3. 現場訪查 (Site Visit) 的秘密

- **技巧**：若銀行要來現場訪查，一定要約你 **「最忙碌」** 的時段。
- **目的**：讓銀行看到你的生意好到「排隊」、店面「太小」、人力「不足」。這傳遞出的訊號是：**「我生意很好，需要資金擴大規模」**，而不是生意不好來借錢救急 。

### 4. 財報與資金流向

- **營收暴衝是大忌**：不要為了貸款突然讓營收翻倍（例如前10個月40萬，第11個月突然100萬），銀行會懷疑真實性，要求看對話紀錄、後台訂單甚至查稅。
- **資金用途**：必須合情、合理、合法。
    - **不可**：買房、買股票、還舊債、個人消費。
    - **可以**：設備、人力、營運周轉、廣告費（有比例限制）。
- **現金收入行業**：若你是小吃店或收現行業，務必**每週將現金存入銀行**並備註（例如：XX收入），創造金流紀錄。

### 5. 負責人自我管理

- **發薪水給自己**：負責人一定要給自己編列薪資。這代表公私分明，否則銀行會認為你把公司當提款機。
- **租賃契約**：即使公司設在自己家或父母家，也要打正式的租賃契約，區分公司與個人的財務。
- **信用分數**：負責人與配偶的聯徵分數建議在 500 分以上（信貸則需更高）。

## 五、常見 Q&A 重點整理

- **Q: 我有個人信貸，還能申請政策性貸款嗎？**
    - **A: 可以。** 兩者不衝突，只要負債比（薪資的22倍）沒爆掉，且還款正常即可。目前成功學員中有 90% 身上都有信貸。
- **Q: 公司剛成立不久（如3個月）可以申請嗎？**
    - **A: 可以。** 有學員成立3個月即申請通過。甚至尚未營業登記前若有實際經營事實，也能透過教學呈現出來。
- **Q: 一人公司（無員工）可以申請嗎？**
    - **A: 可以。** 一人公司現在反而單純，政府支持新創。但注意一人公司不能立刻幫自己保勞保（需有員工後）。
- **Q: 之前有申請過青創還沒還完，可以再申請嗎？**
    - **A: 可以。** 不需要還完舊的才能借新的，只要信用紀錄正常，可以「堆疊」申請（一生不限一次）。
- **Q: 資本額要多少？**
    - **A: 建議至少 10 萬。** 不要設定幾萬塊這種太低的數字。
- **Q: 房東/包租代管業可以申請嗎？**
    - **A: 不可以。** 只要牽扯到房地產投資、包租、整理房子出租，受限於打房政策，皆無法申請。

## 總結

想要順利拿到政府的低利資金，關鍵在於**「真實經營」**與**「正確呈現」**。不要試圖欺騙（銀行會查核），也不要過度美化（暴衝營收）。善用 Q1 黃金申請期，準備好你的**企劃書**與**財務觀念**，沒有店面、剛創業、甚至有一點信貸，都有很大的機會過件。
