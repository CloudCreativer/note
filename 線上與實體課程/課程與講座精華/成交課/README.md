# 成交課

> 阿金 ! 人生進化中 · 黃晨皓（阿金 KIM）· 不教 Sales，教 Closing

## Insight

成交 ≠ 推銷恐懼，而是讓人**感覺良好、願意靠近**的可複製流程。核心是：**成交的是人**、流程要**穩定且有效**（十字象限剔除運氣）、**人群+場景+痛點**精準定位、**服務流程**重感受而非只重效率。進階工具：深杯理論、小成交鏈、情緒七階段（不可跳關）、GOOD 模型 + 飽和攻擊——真誠利益 + 迫切感，持續做到最後一刻。

## 1. 開場：成交的定義

![IMG_4564.HEIC](assets/images/IMG_4564.heic)

- **課程目的**：這堂課不教銷售 (Sales)，教的是成交 (Closing)。
- **本質區別**：
    - **銷售**：往往販賣的是「恐懼」、「擔心」與「不確定性」（例如：不買會生病、不買會後悔）。
    - **成交**：是打造一個讓人們「感覺良好」的體驗流程，目的是讓人們想要**靠近**，因此會一買再買 。
- **核心觀念**：**成交的永遠是「人」**，而不是產品或服務 。

![IMG_4568.HEIC](assets/images/IMG_4568.heic)

## 2. 穩定性理論 (The Stability Theory)

- **問題意識**：你的成交流程是「穩定」還是「不穩定」？
- **十字象限操作**：
    - 畫一個十字，區分「穩定/不穩定」與「有效/無效」。
    - **剔除不穩定**：把流程中依賴運氣、心情（例如看心情才打招呼）的部分剔除。
    - **保留有效**：確認哪些動作是「有效」的（能增加信任、推進關係），只保留這些動作。剩下的就是可複製、可量化的印鈔機 。

## 3. 信任與客戶感受

- **信任的本質**：人們喜歡跟「信任的人」買東西，而不是跟推銷員買。如果客戶覺得你是推銷員，他會想逃跑 。
- **檢核方式**：你怎麼知道客戶現在信不信任你？
    - **不要瞎猜**：直接問 L1、L2 的問題（例如：「目前為止覺得還好嗎？」、「有什麼想問的嗎？」）。
    - **目的**：不是急著銷售，而是**確認對方「感覺是否良好」** 。

## 4. 精準定位：人群、場景、痛點

- **黃金公式**：不要問客戶在哪裡（Where/How），要先問**是誰（Who）**。
- **定義架構**：**人群 + 場景 + 痛點 = 解決方案**。
    - *案例*：不是「30歲女性」，而是「有三個月大嬰兒的媽媽（人群），在小孩半夜不睡覺時（場景），產生的焦慮痛點（痛點）」 。
- **應用**：一旦定義清楚，你就知道去哪裡找他們（例如去月子中心、公園），不用再問「該怎麼接觸」 。

## 5. 流程 vs. 服務流程

![IMG_4583.HEIC](assets/images/IMG_4583.heic)

- **區別**：
    - **流程**：講求效率、邏輯、標準化（如：退貨快）。
    - **服務流程**：講求**感受**、**體驗**、**人味**（Human Touch） 。
- **誤區**：很多人用自動化問卷擋在前面，以為流程順暢，其實客戶體驗很差（覺得無情），導致流失 。

## 6. 實戰案例一：健身房的服務流程設計

- **微笑 (Smile)**：櫃檯人員微笑不是因為禮貌，是因為數據證明這個**動作「有效」**，能增加成交率，所以被標準化保留 。
- **分類引導**：第一句話問「第一次來嗎？」，確認**屬性後進入專屬劇本（分類）**，而非對所有人講一樣的話 。
- **經典三選一**：提供 3 種方案（如：輕量、標準、豪華）。讓客戶思考「要哪一個」，而不是「要不要」。**超過三種會選擇障礙，少於三種沒有比較 。**
- **解決猶豫**：當客戶猶豫時，主動提出「體驗課」，降低決策門檻 。
- **新手健身筆記 (Newbie Note)**：
    - **動作**：離開前給客戶一本實體筆記本。
    - **目的**：1. **離場不空手**（建立連結）；2. **收集癖**（告訴他下週上課可以帶來記，為了填滿筆記，他會傾向於再次回來） 。

## 7. 實戰案例二：銀行開戶的流程優化

- **痛點**：客戶進銀行抽號碼牌後進入「等待期」，心情下墜，此時被推銷信用卡會反感 。
- **解法：開戶導覽單**：在抽號碼牌時遞給他一張單子，寫明接下來的步驟。
    - **邏輯**：**提升控制感**。客戶知道流程，焦慮感下降
- **順勢成交**：在辦手續（開放期）時順便介紹產品，而非在等待時推銷。
- **成效**：芬蘭某銀行靠此儲蓄率提升了 **4000%** 。

## 9. 大絕招：無止盡的想像

- **心法**：**「永遠幫助你面前的用戶，無止盡的想像當他擁有你的產品時，他的生活會如何改變。」** 。
- *案例*：精品廣告（Cartier, Bentley）不講規格，只講場景與氛圍。

## 10. 深杯理論 (Deep Cup Theory)

- **概念**：每個客戶對於信任的需求量（杯子的深度）不同。
- **真相**：客戶沒買（水沒滿）只是因為你**倒得還不夠多**。不要因為一次沒成交就覺得對方不想買。持續做對的動作，直到杯子滿溢 。

## 12. 實戰案例三：麥當勞自助點餐機

![IMG_4617.HEIC](assets/images/IMG_4617.heic)

![IMG_4618.HEIC](assets/images/IMG_4618.heic)

- **斜的畫面**：點餐機畫面靜止時，漢堡圖片是**斜的**。這是模擬**人類手拿漢堡準備吃下去的視角**，刺激大腦想像美味 。

## 13. 實戰案例四：Uber 募資簡報

![IMG_4620.HEIC](assets/images/IMG_4620.heic)

- **故事行銷**：不只講功能，而是講「舊金山下雨天的加成費率」（獲利場景），讓投資人想像這是一個自動運轉的賺錢機器 。
- **信任設計**：司機評分、照片，這些是為了建立信任（成交流程），而不只是功能 。

## 14. 客戶情緒七階段

![IMG_4624.HEIC](assets/images/IMG_4624.heic)

不可跳關，依序推進：

1. **防備**：不銷售，問問題，取得「小成交」（讓客戶願意開口說真話/心聲） 。
2. **觀望**：觀察中。
3. **開放**：肢體放鬆。**對策：立刻開始成交！** 。
4. **共鳴**：用**邏輯**去創造共鳴（因為你跟某某很像...） 。
5. **興奮**：情緒高點。
6. **猶豫**：理性回歸。**對策：給予務實框架**（報價、保證），解決理性恐懼 。
7. **成交**：完成交易。

## 17. 執行心法：GOOD的模型與飽和攻擊

- **GOOD 模型**：這是一個循環優化的過程 。
    - **G - Genuine (真誠)**：對產品有信心，真心想幫對方。
    - **O - Observe (察覺)**：觀察對方的狀態與需求。
    - **O - Open (開放)**：保持開放心態，不急迫，給予空間。
    - **D - Design (設計)**：根據觀察重新設計你的成交話術。
- **飽和攻擊**：引用《葬送的芙莉蓮》。不需要學高深魔法，用**最簡單、最有效**的動作（基本功），進行**大量且重複**的執行，就能擊敗對手 。

## 18. 結語：真誠利益與迫切感

- **迫切感 (Urgency)**：不停止成交不是因為「善良」，是因為**「太迫切」**希望對方變好。如果你相信產品能救他，你就**沒有資格停下來** 。
- **真誠利益**：你真心認定這產品能解決他的問題，因此你決定做你現在在做的事 。
- **無視外部**：不管天氣、股市如何，永遠勇於成交直到最後一刻，因為你管不了別人，只能管好你的介紹 。
