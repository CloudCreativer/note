# 看懂財報與資金管理

> 十堂線上課程 · 張珍綺 · 投資人視角的財務與資金

## Insight

創業者讀財報不是為了記科目，而是回答投資人三件事：**能不能賺、錢夠不夠活、成長假設站不站得住**。流程是：收入認列原則 → 營收/成本/毛利率預測 → 固定與變動費用及資本化 → 現金流與 runway → 估值方法；數字必須能回溯假設，並與 Pitch / DD 一致。

## Translation

| 課程說法 | 我的理解 |
|----------|----------|
| 投資人三件事 | 獲利能力、現金流、成長可信度——不是只看營收曲線 |
| 收入認列 | 何時算「賺到」會直接改變財測與估值敘事 |
| 固定 vs 變動成本 | 規模化時毛利率與 break-even 怎麼變，要先建模 |
| 資本化 vs 費用化 | 研發/軟體開支怎麼進表，影響短期損益與現金 |
| 新創估值 | 早期常混合可比法、DCF、里程碑法，關鍵是假設透明 |

## Why

| 以前覺得 | 現在覺得 |
|----------|----------|
| 財測是給投資人看的漂亮圖 | 是管理 runway 與募資時機的工具 |
| 認列等會計處理再說 | 認列原則會改變「何時有收入」與 KPI |
| 估值越高越好 | 要與里程碑、同業、第二輪稀釋一起想 |
| 現金流表最不重要 | 新創陣亡常死在現金，不是死在損益 |

## 🧩 二、商業模式與財務數據的關聯

> 「數字藏在商業模式裡。」
> 

### 🔁 流程圖邏輯

1️⃣ **營業收入（Revenue）**

↳ 由目標客戶與價值主張決定。

2️⃣ **成本與費用（Cost & Expense）**

↳ 來自關鍵活動、資源與合作夥伴。

3️⃣ **營運決策（Decision）**

↳ 受三大因素影響：

- 外部環境

- 團隊經營理念

- 企業體質

4️⃣ **財務結果（Result）**

↳ 最終反映在會計報表（損益表、資產負債表、現金流量表）。

---

## 📚 三、會計（Accounting）的角色

> 「會計不是學問，而是溝通語言。」
> 

### ✅ 功能定位

- 透過規則化的數據記錄，
    
    將企業活動轉換為**投資人能理解的量化資訊**。
    
- 讓投資人看見：
    
    > 「技術 → 數據 → 營運成果」的轉換。
    >

## ⚙️ 四、企業活動的三大類型

| 類型 | 說明 | 財務報表位置 |
| --- | --- | --- |
| 🏭 營業活動 (Operating) | 企業日常營運行為（銷售、採購、人事） | 損益表／現金流量表 |
| 💰 投資活動 (Investing) | 資源配置（購置固定資產、金融商品） | 現金流量表 |
| 🏦 籌資活動 (Financing) | 融資行為（借款、增資、募資） | 現金流量表＋資產負債表 |

---

## 📊 五、會計的五大要素

| 要素 | 定義 | 範例 |
| --- | --- | --- |
| 🪙 資產（Assets） | 企業擁有、能創造價值的資源 | 現金、設備、應收帳款 |
| 💳 負債（Liabilities） | 過去事項形成的義務 | 應付帳款、借款 |
| 🧾 權益（Equity） | 資產－負債＝業主權益 | 股本、保留盈餘 |
| 💵 收益（Revenue） | 因商品或服務帶來的資產增加 | 銷售收入 |
| 📉 費損（Expense） | 因成本或費用造成的權益減少 | 進貨成本、管理費用 |

> 會計恆等式：
> 
> 
> 資產 ＝ 負債 ＋ 業主權益
>

## 🧮 六、三大報表的核心差異

| 報表名稱 | 說明 | 觀察重點 | 時間性 |
| --- | --- | --- | --- |
| 🧱 資產負債表 (Balance Sheet) | 一個時間點的資產與負債狀況 | 穩定性／償債能力 | 📅 單一時間點（例：12/31） |
| 📈 損益表 (Income Statement) | 一段期間內的收入與支出 | 盈虧狀況／經營效率 | 🗓 一段期間（例：1/1～12/31） |
| 💧 現金流量表 (Cash Flow Statement) | 現金流入與流出變化 | 流動性／資金健康度 | 🗓 一段期間（例：季度或年度） |

## 財務預測與收入認列原則

## 🧭 一、為什麼投資人重視財務預測

![Hahow for Business00_23 (1).png](assets/images/Hahow_for_Business00_23_(1).png)

> 「財務預測不是算命，是策略的驗證。」
> 

投資人希望從財測中看到：

- 你是否理解市場的現實？
- 你的成長假設是否合理？
- 你是否有計畫與依據地使用資源？

---

## 🧩 二、財務預測的起點：市場與資源盤點

### 🔍 三個前置分析

1. **市場規模與滲透率**
    - 你的市場總量（TAM, SAM, SOM）是多少？
    - 你的戰略市場區塊與目標占有率？
2. **現有資源盤點**
    - 可動用的人力、資金、通路、品牌能量。
3. **銷售與成本目標設定**
    - **目標**：銷售極大化、費用極小化
    - **原則**：每一分錢都要花在刀口上

---

## 🧮 三、財務預測的七步驟流程

![Hahow for Business01_37.png](assets/images/Hahow_for_Business01_37.png)

| 步驟 | 說明 | 目的 |
| --- | --- | --- |
| 1️⃣ | 盤點市場與資源 | 確定市場機會與可用資源 |
| 2️⃣ | 訂定銷售目標 | 明確設定營收期望值 |
| 3️⃣ | 分析外部與內部環境 | 辨識潛在風險與影響因素 |
| 4️⃣ | 擬定行動方案 | 規劃具體執行策略 |
| 5️⃣ | 成本效益分析 | 找出「最小成本達最大目標」方案 |
| 6️⃣ | 輔助計畫規劃 | 包含人員訓練、設備採購等支援項目 |
| 7️⃣ | 數據化推算 | 轉化為價格、銷量、成本、費用的預估模型 |

## 💡 四、財務預測的核心原則

![Hahow for Business04_57.png](assets/images/Hahow_for_Business04_57.png)

### 1️⃣ 組成分析（Decomposition）

- 不要只給出「總營收成長10%」這種模糊預測。
- 應細分為不同收入來源的組成項：
    - 📦 硬體銷售
    - ☁️ SaaS 服務
    - 🔑 授權收入
    - 💼 顧問或維運費
- 為每一項設定不同的成長率與依據。

### 2️⃣ 關鍵動因（Key Drivers）

- 每一個數字都要能被「推論出來」。
    
    > 例：營收從 10% → 20%，為什麼？
    > 
    - 通路數增加？
    - 新產品推出？
    - 價格上調？
    - 市場擴展？
- 這些假設會在募資會議中被投資人**反覆挑戰**。

## 🧠 五、營收類型與預測基礎

| 收入類型 | 說明 | 常見產業 |
| --- | --- | --- |
| 🏷️ 商品銷售 (Sales) | 一次性商品銷售收入 | 電商、硬體 |
| ⚙️ 勞務收入 (Services) | 提供專業服務的報酬 | 顧問、教育 |
| 🧾 租金收入 (Rental) | 資產出租所得 | 不動產、設備租賃 |
| 💡 權利金收入 (Royalty) | 技術授權或專利授權費 | 生技、IP 授權 |
| 💰 利息收入 (Interest) | 投資或貸款收益 | 金融、平台型業者 |

> 📌 每種類型的收入都有不同的「認列時點」與「預測依據」，
> 
> 
> 務必釐清主要營業項目與各項佔比。
>

## ⚖️ 六、收入認列的五步判斷法（IFRS 15）

> 「收入不是你想認就能認。」
> 

| 步驟 | 內容 | 實例說明 |
| --- | --- | --- |
| 1️⃣ 辨識合約 | 任何正式或非正式合約（書面、訂單、Email）皆屬 | 客戶下單或簽約 |
| 2️⃣ 辨識履約義務 | 合約中公司需完成的具體任務 | 賣硬體＋安裝服務＝2 項義務 |
| 3️⃣ 決定交易價格 | 客戶總支付金額 | 例：總價 NT$100 |
| 4️⃣ 分攤交易價格 | 依各履約義務的「公允價值」比例分配 | 硬體60%、安裝40% |
| 5️⃣ 認列收入 | 當履約義務完成後才能認列 | 硬體交付時認列60%，安裝完成後認列40% |

### ⚠️ 常見誤區：

- 認列過早（尚未履約完畢）。
- 忽略分期／時間性義務。
- 未依公允價值拆分複合合約。

## 財務預測的實作：營收、成本與毛利率分析

![Hahow for Business00_12.png](assets/images/Hahow_for_Business00_12.png)

🧭 一、財務預測的核心：從銷售預測開始

> 「預測的準確度，決定整份財測的可信度。」
> 

財務預測的第一步，是 **根據過去的業績與市場預測，推算未來的銷售額**。

### 📌 主要依據：

- 過去營運表現（歷史銷售資料）
- 市場趨勢與產業成長率
- 產品定價策略
- 通路與促銷規劃
- 預算投入（行銷、人力、資金）

---

## 🧩 二、銷售預測的四大構面

![Hahow for Business00_42.png](assets/images/Hahow_for_Business00_42.png)

| 構面 | 問題 | 實務舉例 |
| --- | --- | --- |
| 產品策略 | 產品定位與差異化 | 主打高毛利技術產品？或平價擴量？ |
| 定價策略 | 採成本加成法 or 市場導向定價？ | 成本價＋期望毛利率＝售價 |
| 通路策略 | 銷售管道與覆蓋率 | 線上商城、自營官網、經銷商 |
| 促銷策略 | 預計的推廣活動與折扣政策 | 廣告費用、試用活動、KOL合作 |

## 📈 三、營業收入預估原則

![Hahow for Business01_29.png](assets/images/Hahow_for_Business01_29.png)

![Hahow for Business02_16.png](assets/images/Hahow_for_Business02_16.png)

1️⃣ **價與量分開預測**

- 不僅預估營收總額，也要分開看「價格變動」與「銷量成長」。

2️⃣ **收入類型拆分預估**

- 若公司同時有多種收入（硬體、軟體、服務、授權等），應個別計算。

3️⃣ **市場合理性檢核**

- 你的預測是否符合整體產業成長速度？
- 若產業平均成長5%，你預測50%，需提出合理說明。

4️⃣ **趨勢一致性**

- 銷售預測需與整體市場與消費趨勢方向一致。

5️⃣ **彈性與情境模擬**

- 應保留樂觀（Best）、基準（Base）、悲觀（Worst）三種版本，因應市場變化。

---

## ⚙️ 四、營業成本的估算邏輯

![Hahow for Business03_15 (1).png](assets/images/Hahow_for_Business03_15_(1).png)

> 「成本預測是財測可信度的第二根支柱。」
> 

### 🧾 基本公式（買賣業常用）：

```
期初存貨 + 當期進貨 – 期末存貨 = 營業成本

```

📍範例：

若公司年初有庫存100萬，全年進貨400萬，年末庫存為80萬，

則營業成本＝100 + 400 − 80 ＝ **420萬**

---

## 🧪 五、製造業的成本組成

![Hahow for Business05_21.png](assets/images/Hahow_for_Business05_21.png)

| 類別 | 說明 | 舉例 |
| --- | --- | --- |
| 🧱 直接材料 (Material) | 生產中實際使用的原料 | 鋼材、包裝、電子元件 |
| 👷‍♀️ 直接人工 (Labor) | 直接參與製造的人工薪資 | 操作員、技術人員 |
| ⚙️ 製造費用 (Overhead) | 間接支出，如折舊、水電、維修等 | 工廠租金、機器折舊、維護費 |

### 🧮 固定與變動成本

- **固定成本**：不隨產量變動（例：折舊、廠租）
- **變動成本**：隨生產量變化（例：原料、包裝、運費）

> 🔸產能利用率越高 → 固定成本分攤越低 → 單位製造成本下降。
>

## 💰 六、營業毛利與毛利率

![Hahow for Business05_48 (1).png](assets/images/Hahow_for_Business05_48_(1).png)

### 📊 計算方式

```
營業毛利 = 營業收入 − 營業成本
毛利率 (%) = 營業毛利 ÷ 營業收入 × 100%

```

### 📍解析

![Hahow for Business06_12.png](assets/images/Hahow_for_Business06_12.png)

| 狀況 | 代表意義 |
| --- | --- |
| 毛利率高 | 成本控制良好、產品具競爭力 |
| 毛利率低 | 成本偏高或銷量未達經濟規模 |
| 毛利率下降 | 可能出現價格戰或成本上升壓力 |

## 🧾 七、營業費用（Operating Expenses）

![Hahow for Business06_55.png](assets/images/Hahow_for_Business06_55.png)

分為三大類：

| 類別 | 說明 | 範例 |
| --- | --- | --- |
| 📣 銷售費用 (Selling) | 推廣與行銷支出 | 廣告費、樣品、展覽旅費 |
| 🧑‍💼 管理費用 (G&A) | 公司營運與後勤支出 | 租金、辦公費、行政人員薪資、壞帳 |
| 🧠 研發費用 (R&D) | 新產品開發投入 | 研發人員薪資、測試材料、儀器折舊 |

### 💡 補充：

- 研發費用投入高 → 可能代表公司創新力強，但短期獲利壓力大。
- 投資人會觀察研發費用佔營收比重（R&D/Sales Ratio）。

## ⚖️ 八、成本 vs 費用：會計上的差異

![Hahow for Business08_55.png](assets/images/Hahow_for_Business08_55.png)

| 類別 | 特徵 | 是否直接與營收相關 |
| --- | --- | --- |
| 成本 (Cost) | 為產生營收而發生，與商品或服務直接相關 | ✅ 是 |
| 費用 (Expense) | 維持企業日常營運所需，非直接創造營收 | ❌ 否 |

📍例子：

- 生產線員工薪資 → 成本
- 會計人員薪資 → 費用

## 費用預估與資本化邏輯

![Hahow for Business00_03.png](assets/images/Hahow_for_Business00_03.png)

## 🧭 一、費用預估的起點：釐清分類

> 「在做任何財測之前，先問自己：這筆支出是成本，還是費用？」
> 

### 📌 兩個基本步驟

1. **明確區分成本 vs 費用**
    - 成本：直接與收入生成相關（例：生產、服務提供）
    - 費用：為營運支撐支出，與收入關聯間接
2. **確認支出性質是否應歸屬營業項目**

---

## ⚖️ 二、毛利率的反思：因還是果？

> 毛利率不只是結果數字，而是一種經營「體質檢查」。
> 

### 💡 分析邏輯

- 若實際毛利率與預期落差過大，代表：
    - 成本估算錯誤（漏算、分類錯誤）
    - 成本結構變動（原料漲價、產能利用率下降）
- 因此：
    - **毛利率高** → 成本控制良好、產品具競爭力
    - **毛利率低** → 定價錯、成本高、規模未達

---

## 🧮 三、標準成本制度（Standard Cost System）

> 用來做預估與實際比較，找出偏差來源。
> 

### 概念：

- 料、工、費以「標準單位成本」為基礎編製預測
- 實際發生後再「差異分析」調整至合理水準

📍舉例：

> 標準單位成本 NT$100，實際生產成本 NT$120
> 
> 
> → 成本差異 20%，需回溯原因（原料上漲？人工效率？）
> 

---

## 💡 四、營業費用預估：銷管研三分法

![Hahow for Business01_29 (1).png](assets/images/Hahow_for_Business01_29_(1).png)

| 類別 | 說明 | 內容範例 |
| --- | --- | --- |
| 📣 銷售費用 (Selling) | 推廣與銷售相關支出 | 業務薪資、獎金、廣告費、展覽費 |
| 🧑‍💼 管理費用 (General & Admin) | 經營後勤支出 | 租金、辦公室開銷、行政薪資、壞帳 |
| 🧠 研發費用 (R&D) | 新產品或技術開發支出 | 研發人員薪資、材料、測試費、折舊 |

### 🧾 實務建議

- 明確分攤來源（避免銷管研混用）
- 避免錯列造成誤判，如：
    - 將行銷費放入管理 → 看起來銷售費用太低
    - 研發支出誤列為行銷 → 看起來研發投入不足

## 🧩 五、費用預估的四大原則

| 原則 | 說明 | 實務舉例 |
| --- | --- | --- |
| ① 完整性 (Completeness) | 所有公司相關支出都要列入 | 創業者個人支出不可混入公司帳 |
| ② 合理性 (Reasonableness) | 預估須依據市場規模、成長趨勢 | 不可突然大增 200% 無依據 |
| ③ 配比性 (Matching) | 支出與收入應屬同一期間 | 十二月獎金應記入當年，非明年 |
| ④ 一致性 (Consistency) | 計算方法與過往保持一致 | 否則投資人無法追蹤趨勢 |

## 🧾 六、權責發生制（Accrual Basis） vs 現金制（Cash Basis）

![Hahow for Business02_16 (1).png](assets/images/Hahow_for_Business02_16_(1).png)

> 「會計不看錢何時付，而看義務何時發生。」
> 

| 制度 | 原則 | 實務範例 |
| --- | --- | --- |
| 💰 現金制 | 收到錢或付錢時才認列 | 1月付12月薪資 → 記入1月 |
| 🧾 權責發生制 | 費用在「發生時」即認列 | 1月付12月薪資 → 應記入12月 |

📍提醒：

> 財測與會計報表都應使用「權責發生制」，
> 
> 
> 不可因延遲付款而低估費用。
>

## 🧠 七、研發費用預估重點

![Hahow for Business04_50.png](assets/images/Hahow_for_Business04_50.png)

> 研發預算應與研發進度、專案階段緊密對應。
> 

### 範例：

若公司預計進入「臨床二期」：

- 應估算：
    - 招募病人數與成本
    - 各實驗階段所需時間與設備
- 對應會計處理：
    - 可依據法規（如台灣稅法）列為研發支出項目

## 🧩 八、資本化 vs 費用化

![Hahow for Business06_21.png](assets/images/Hahow_for_Business06_21.png)

> 「資本化」就是把支出變成資產；「費用化」是當期損益。
> 

| 項目 | 費用化 (Expense) | 資本化 (Capitalize) |
| --- | --- | --- |
| 定義 | 當期直接認列支出 | 視為資產，分期折舊 |
| 對獲利影響 | 當期大幅減少淨利 | 攤銷後分年影響淨利 |
| 適用情境 | 行銷活動、租金、水電 | 設備購置、軟體開發、專利取得 |
| 風險 | 當期報表獲利波動大 | 長期折舊需合規判定 |

📍範例：

> 公司投入研發費用 NT$100萬
> 
> 
> 若 **費用化** → 當期費用 +100萬，獲利驟降
> 
> 若 **資本化（10年攤提）** → 每年折舊10萬，報表穩定
> 

---

## ⚠️ 九、資本化決策注意事項

1. 是否帶來**長期經濟效益**？
2. 是否可**可靠衡量成本**？
3. 是否**法律上歸公司所有**？
4. 必須由**會計師審慎評估**，確保符合準則（IAS 38、IFRS）。

---

## 🏁 十、重點總結

✅ 成本與費用分類正確，是財報可信度的基礎。

✅ 毛利率偏差是經營警訊，要反查成本結構。

✅ 預估費用要考慮「權責發生制」，非現金制。

✅ 研發與資本化需合乎準則與專業判斷。

✅ 財測需能真實反映企業「經濟實質」。

## 🧮 三、估值基礎公式：Pre-Money 與 Post-Money

![Hahow for Business03_30 (1).png](assets/images/Hahow_for_Business03_30_(1).png)

| 名稱 | 定義 | 計算方式 | 範例 |
| --- | --- | --- | --- |
| 💰 Pre-Money Valuation | 投資前公司估值 | Post-Money − 投資金額 | 若投資後估值1000萬、投資100萬 → Pre = 900萬 |
| 💰 Post-Money Valuation | 投資後公司估值 | 投資金額 ÷ 股權比例 | 投100萬換10%股權 → Post = 100 / 10% = 1000萬 |

## 📊 五、國際公認三大估值法

![Hahow for Business07_56.png](assets/images/Hahow_for_Business07_56.png)

| 類型 | 核心概念 | 適用階段 | 優缺點 |
| --- | --- | --- | --- |
| 🧱 **資產法 (Asset Approach)** | 以公司資產負債淨值為基礎，調整潛在損益 | 成熟期、資產型產業 | 穩定但低估創新潛力 |
| 📈 **市場法 (Market Approach)** | 以可比較公司（Comps）或交易倍數推估 | 成長期 | 直觀但需找到相似公司 |
| 💵 **收益法 (Income Approach / DCF)** | 以未來現金流折現（Discounted Cash Flow）求值 | 成長期 / 成熟期 | 精準但假設多、需專業模型 |

> 📘 DCF 關鍵概念：
> 
> 
> 「明天的一塊錢，不等於今天的一塊錢。」
> 
> 必須以**折現率（Discount Rate）**將未來現金流折回現值。
>

## 🧰 六、早期新創常用估值模型

![Hahow for Business09_13.png](assets/images/Hahow_for_Business09_13.png)

![Hahow for Business09_39.png](assets/images/Hahow_for_Business09_39.png)

### 1️⃣ VCM法（Venture Capital Method）

![Hahow for Business10_58 (1).png](assets/images/Hahow_for_Business10_58_(1).png)

- 以投資報酬率（ROI）推估目標估值
- **公式：**
    
    ```
    投資人ROI = 退出時企業價值 / 投資金額
    
    ```
    
- **步驟：**
    1. 預估5年後企業退出價值（依同業倍數或營收估算）
    2. 設定期望ROI（如10倍）
    3. 推算當前估值

---

### 2️⃣ Cayenne Method（卡宴法）

![Hahow for Business11_29.png](assets/images/Hahow_for_Business11_29.png)

### 💡 方法概念

以創業顧問或投資顧問的**專業經驗判斷**為基礎，

針對企業的多個面向（團隊、產品、市場、策略、競爭力、外部合作關係等）

進行「定性＋定量」評估，給予加權分數，

最後根據**參考交易倍數（Valuation Multiple）**計算出估值範圍。

### 🧩 評估項目範例

| 評估項目 | 權重範例 | 評估重點 |
| --- | --- | --- |
| 團隊與執行力 | 25% | 創辦人經驗、互補性、決策效率 |
| 產品與技術成熟度 | 20% | 原型完成度、技術壁壘 |
| 市場策略與商業模式 | 20% | TAM/SAM/SOM、通路策略 |
| 競爭優勢 | 15% | 抗模仿性、獨特定位 |
| 外部資源與合作關係 | 10% | 政府補助、加速器、策略夥伴 |
| 財務健全與現金流 | 10% | 現金存量、資金管理能力 |

📘 **計算公式：**

```
企業估值 = 參考市場估值 × 綜合加權係數

```

> 🔹 綜合加權係數通常介於 0.8～1.5 之間
> 
> 
> 🔹 評價人（通常是輔導業師、創投顧問）會依經驗做調整
> 

---

### 🧠 實務應用建議

- **結合 Scorecard 法使用**：可先用 Scorecard 建立基準，再用 Cayenne 法微調。
- **需專業參與**：建議邀請外部顧問共同評分，避免主觀偏差。
- **適合創業中期**：有營運紀錄、已進入市場，但尚未穩定獲利的階段。

---

### 3️⃣ Berkus Model（伯克斯法）

> 用「定性」評估早期價值，適合無財報的新創。
> 

| 評估項目 | 最高估值貢獻（USD） |
| --- | --- |
| 創意與市場機會 | $500K |
| 團隊素質與執行力 | $500K |
| 原型產品（Prototype） | $500K |
| 戰略合作或早期用戶 | $500K |
| 銷售與成長潛力 | $500K |
| **合計上限** | **$2.5M–$3M** |

---

### 4️⃣ Scorecard 法（評價卡法）

![Hahow for Business12_48.png](assets/images/Hahow_for_Business12_48.png)

> 適合尚未上市、無財報的新創團隊。
> 

| 評估面向 | 權重範例 | 評估指標 |
| --- | --- | --- |
| 團隊與經驗 | 30% | 創辦人背景、執行力 |
| 產品與技術 | 25% | 創新性、技術壁壘 |
| 市場機會 | 20% | TAM / SAM / SOM |
| 競爭優勢 | 10% | 市場壟斷性、護城河 |
| 行銷策略 | 10% | 通路布局與推廣力 |
| 財務策略 | 5% | 現金流、資金管理 |

📊 **最終估值 = 交易市場平均估值 × 綜合得分係數**

## 🧩 十、影響估值選擇的三大關鍵
