## 記者會筆記

## 核心隱喻與計畫定位

- **火星計畫**：代表極致、艱難、沒有退路的環境。火星很遠，但「人心更遠」。此計畫旨在跨越心理距離，登陸人們心中塵封的夢想。
- **阿提米斯（Artemis）**：月亮女神。象徵在黑暗中引導光芒，代表「在未知之中依然能夠選擇前往的勇氣」。
- **場地選擇（台北小巨蛋）**：刻意選擇一個「不安全」、象徵巔峰與不確定性的場地。因為領導者若選擇安全保守，追隨者就會失去相信夢想的理由。

## 板塊一：核心團隊架構與「高壓專案」執行心法

| **職位** | **擔當者** | **核心任務與準則** |
| --- | --- | --- |
| **總指揮官** | **Kim 黃震浩** | 確保方向正確、關鍵決策、維持初衷，成為團隊的領袖。 |
| **副指揮官** | **Peter 陳偉誠** | 穩定器，負責執行、節奏協調與資源分配，確保全員平安抵達。 |
| **太空員領隊** | **小鯨魚 蕭景宇** | 關注團隊默契與心理狀態，透過「過度溝通」與「5秒專注」穿越風暴。 |
| **風險評估專家** | **Haley 廖盈雅** | 分析盲點與代價。主張**「願景可以留白，但決策與責任不能模糊」**。 |
| **視覺結構專家** | **Yvonne 李其鋒** | 將抽象概念具象化。視覺非裝飾，而是為了**建立秩序、降低焦慮、發現問題**。 |

在高壓、極度緊湊（倒數小於100天）的任務中，團隊不靠意志力撐，而是靠**系統與紀律**。

### 1. 視覺結構專家 Yvonne（李其鋒）：視覺是秩序，不是裝飾

- **核心價值**：在混亂與高壓中最先壞掉的是「資訊秩序」。視覺設計的目的是建立架構。
- **合格視覺的三大標準**：
    1. **定位清晰**：讓每個人在複雜節點中，一眼看出「自己在哪裡」。
    2. **降低協作成本**：對內外部建立統一的視覺語言與邏輯，避免誤解像滾雪球般擴大。
    3. **暴露問題**：好的視覺不該用來「包裝」或掩蓋缺點，而是讓流程的斷點、角色的缺口被清楚看見，進而回頭修正。

### 2. 風險評估專家 Hailey（廖盈雅）：把熱情的火焰裝進玻璃罐

- **核心價值**：不阻止任務前進，而是讓團隊在推進時保有理性，確保「代價」在可控範圍。
- **情緒與風險剝離法**：遇到問題時，將「情緒反應」與「風險事實」拆開。逐條寫下現狀、發生時間、驗證項目與推估結果，再進行反向辯論。
- **留白 vs. 含糊**：
    - **可以留白**：願景、達成的方法。
    - **絕對不能留白**：決策責任、風險界線、出問題時誰接手。把模糊當彈性，會導致效率下降與代價升高。

### 3. 太空員領隊 Ofish 小鯨魚（蕭景宇）：團隊的呼吸與微觀管理

- **核心價值**：監控團隊心理狀態與默契，確保團隊在動盪中「有在呼吸」。
- **防錯機制（過度溝通）**：高壓下的小失誤會致命。無論多小的事，都在所有節點進行微小確認，防止資訊落差。
- **微觀推進法（五秒專注度）**：將龐大任務拆解到最小單位，只要能專注執行「5秒」，就能順利推進。
- **心態錨定**：「當你覺得有瓶頸的時候，你就有一個瓶頸。」利用太空筆記本記錄每日完成的小事，累積成就感。

### 4. 副指揮官 Peter（陳偉誠）：任務的穩定器與底線守護者

- **核心價值**：協調支援，確保指揮官的願景能平穩、安全地落地。
- **三大潛在危機**：時間壓力、系統結構不夠清晰、突發狀況。
- **執行底線**：絕不允許任何人以「時間不夠」為藉口去妥協品質。確保每一個環節都有目的性、有意識地執行，而非盲目趕工。

## 板塊二：總指揮官 Kim 的領導力進化與決策

### 1. 領導者的自我克制與穩定

- **從「奔放」到「穩定」**：過去 Kim 習慣靠個人魅力與臨場發揮（甚至上台前一刻還在改簡報），但面對百人協作的火星計畫，任何微小更動都會引發災難。
- **覺悟**：「領導者的成功代表整個幕後團隊的大成功。」個人的自由奔放不能再成為藉口，必須穩定下來處理複雜事務。

### 2. 資源的極致取捨

- **砍掉的**：純粹為了「帥」但無意義的事物、不必要的媒體曝光。
- **重金投入的**：音樂、視覺、以及確保觀眾「身心靈安全」的體驗（如何進入狀態，如何平穩返場）。

### 3. 無路可退的決心（照飛不誤）

- 即便在滑雪時摔傷肋骨，仍確保計畫推進。
- 買足了能賠償小巨蛋違約金的意外險——「就算我掛了，不在了，火箭依然會飛。」
- **領導者的定義**：不是準備到完美才開始，而是一旦坐上那個位置，就該做出與之匹配的行為。

## 板塊三：阿提米斯重生演講 —— 突破「平庸地心引力」的底層邏輯

這是一場從心智底層解除限制的演說，核心在於打破「不可能」。

### 1. 失敗的重新定義：是「分號」不是「句號」

- **現象**：我們常把失敗視為「人格判決」（證明我不行），因此連第一步都不敢跨出。
- **真相**：失敗只是一個「分號」，一個用來提示你修正軌道的數據。它提醒你停下來檢視方向，而不是提早宣布人生結束。

### 2. 拆解「思想監獄」與「意志力陷阱」

- **思想監獄**：腦子飛快，身體卻停在原地。過度分析風險與代價，把力氣都花在「想像」，最後身體學會了「只要不動就不會受傷」。
- **意志力陷阱**：意志力是一種會消耗的燃料。當你完全靠意志力硬撐，總有歸零的一天。行動不能靠硬撐，需要「系統」的推動。

### 3. 行動的真正「資格」

- 真正的勇敢，不是站在安全的地方喊口號；而是雙手顫抖、腳步站不穩、心裡害怕失去控制，但**依然沒有停下來**。
- 當你敢把名字寫上去，敢下注在自己身上，預期之外的資源與支持才會啟動。

### 4. 人生的終極目標：成為他人的「證據」

- 不要讓從未走過那條路的人，用他們的安全感來決定你的夢想。
- 你的前進不是一個人的事，你的目標不只是抵達終點，而是**讓你的存在本身成為一個清晰的訊號——告訴身邊的人「夢想可以成真」**。
