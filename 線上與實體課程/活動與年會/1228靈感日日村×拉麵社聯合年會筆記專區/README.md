## 📌 Q&A 彙整

Ｑ：正在做品牌圈，有做了許多問卷並有針對不同 TA 做問卷，但不確定這樣做是否正確？

先執行，做問卷有分對錯對錯循環。

Ｑ：是否賺錢，如果不賺錢是否想做？

如果不賺錢仍想繼續做，則帶給人們幸福感。

沒有任何想要賺錢、財務的理由，本身就會帶來給人幸福感。

企圖用作品的高尚，來躲在後面躲避不懂商業邏輯；並不是自己閉門造車，把作品賣出去，就會有答案。

Ｑ：作品低級的賽道，在脆有流量，但想轉換到 IG？

如果在脆如果沒有流量，會怎樣？如果會死掉，那其實等於沒有護城河

五大流量：社群、直接、社群、推薦、廣告

IG = 脆，都是社群流量，這是同樣的流量，建議改換做其他領域，例如私域。但 LINE OA 是一對一 =/= 社群流量，一對一的效率差，要一對多才有效率。

在這之前，先問：

- 你假設自己再也不會進步？
- 你的用戶不會變得比你更聰明？

如果答案是「否」，你說護城河高這件事不成立，也就是你的護城河很低。

有很多人會害怕被攻擊到自己被藏起來的地方，但這才是真正要解決的問題，如果不真實說出問題點，那只會一直循環回到問題本身，並不會真正解決問題，我們必須真實面對真正的問題，才能解決它。

關於泰式料理、經營五年、過往經驗、生意好，擴充三家店。但是正職員工帶不起來，已經嘗試過以下做法，但都失敗：

- 過去長官經驗，失敗
- Youtube 知識分享，失敗
- 獎金誘因，失敗
- 入股，失敗

Kim 舉例：給女兒一千元，並告訴他可以買各種東西，又另外拿了一百塊，跟他說可以買各種東西，最後，問女兒他想要哪個？

最後女兒選擇了一百元，原因僅只是：因為他喜歡紅色。

之所以失敗，是因為你沒有告訴他們做這個選擇的價值在哪裡？對他們哪裡有價值？如果他們不知道這個價值是什麼，你必須告訴他們價值在哪？也包含你的路徑。

Kim 舉例：有一天和老婆在爭執開一間店，如果可以，為什麼不拓展、開很多、賺很多，人都要成長⋯⋯，而老婆回應：「我又不是你，為什麼要跟你一樣？」

所以並不是每個人要的都跟你一樣，你要去找到他要的是什麼，給他他想要的，他才會接受。如果你不知道他想要什麼，你可以直接問他。

什麼才稱之為是完美的護城河？

你可以直接聯繫上、可以叫得來的客戶，才可稱之護城河，每個人都要有自己的名單池。

原因是：透過媒體社群（Instagram、Facebook、Threads）上認識我們的用戶，都受到演算法的影響，會被所打的廣告素材、時間、順序不同，導致對用戶來說其實並不完全認識我們，這是因為演算法的關係導致無法深度認識我們，因為用戶被打到廣告是破碎、沒有邏輯規則的。

這也是為什麼我們有做電子報，因為電子報可以自己控制，每封電子報都可以持續堆疊、可以建立用戶認識我們的順序與邏輯，進而深度一步步認識我們。

穩定的流量 PxQ

## 📌 Q&A 彙整

Q：關於自我質疑，是什麼可以讓你從演員到顯化老師的過程中不被干擾？

A：

- 不要害怕（臉）的提問。
- 自我懷疑是持續的練習： 九粒分享她在好萊塢當演員時，曾面臨各種打擊（如試鏡騙局或因外貌被拒絕），這些經歷磨練了她的心志。
- 具象化你的情緒**：** 將負面情緒想像成一個物件或卡通人物，心理學研究顯示，具象化有助於陪伴與處理情緒！
- 行為造就現實： 「我沒有自信」 >> 把他變成「那有自信的人會怎麼做？」

Q：我剛創立IG兩周，想要詢問對NEWNEW生心靈創作者的建議？

A：

- 一定要做自媒體，很棒！你會更認識自己。
- 不要被數字綁架： 不要用數字定義自己，就像小時候用成績定義優劣一樣，反而框架了自己是個「100追蹤的人」。
- 專注於「想傳遞的價值」： 如果你想分享你的生活讓人感同身受，你就不要忘記自己的初心，不要去專注在數字。
- 不要為自己算命：「你說你做不到，你就做不到。」，你在用默認定義自己。
- 設定停損點並勇於轉向： 如果你真的撐不下去，請不要告訴自己說：「我就是懶、我就是爛。」，代表告訴你，你一定有事情必須要改變。老師分享自己曾做了一年「英文教學」後感到不快樂，勇於承認「需要改變」並轉向自己興奮的「音樂」與「顯化」領域，反而獲得了成功。
- 如果你真的有一件事要撐不下去了：問自己說：「5~10年後你還看不看得到這件事的未來？」。

Q：提問者表示他是做業務的，業務員如何面對被拒絕的恐懼？

A：

- 練習「不要臉」： 九粒分享自己曾在美國沒有綠卡時，靠著「毛遂自薦」爭取到經紀公司的職位。練習被拒絕能累積故事與經驗。你需要不斷的練習被拒絕，被拒絕是一件多幸福的事情。
- 重新定義拒絕： 拒絕不代表你不好，而是為你的未來增加故事素材。

Q：**可以幫別人顯化嗎（例如希望弟弟改變）？**

A：

- 老師反問：「你想要改變的是你的人生還是他的人生？」，如果後者當然是NO！
- 專注於自己： 我們無法改變任何人。與其糾結特定對象，不如顯化「我想要什麼樣的合作關係」或「我想要能與我同頻率的伴侶」。
- 祝福優於強迫： 可以透過祈禱（Coherence Healing）給予祝福，但前提是對方也想要這份改變。

Q：我是做餐飲的，我要如何讓我的品牌是有記憶點且不被取代？

A：人與故事是核心： 這個世界最不會被取代的就是“人”。如果你只是介紹你的餐點很好吃，那對於別人不會有記憶點。但如果你介紹你的餐點之於你有什麼故事，那沒有人能取代你的故事。

Q： **如果陷入負面情緒、卡點很久該怎麼辦？**

A：

- 命名法（Name it to tame it）： 先不要定義該情緒是不好的，先問自己你為什麼不喜歡這個情緒。覺察並命名你的情緒（如「失望」、「自責」），一旦命名，你就能駕馭它而非被它控制。如果你很難說出是什麼情緒，那代表你沒有從來沒有正視他。
- 心理諮商如同健身： 諮商是為了看見內在故事並處理它，這不是病，而是「心理健身」。
- 自我安撫： 如果你下次有情緒，練習把手放在心上。告訴自己：「我是安全的。有這個情緒很正常。」情緒其實是中性的，他是為了保護你。

Q：自己是一個工作上情緒化的人，所以開始練習理性的決策，但當理性過度壓抑感性，覺得不一致時該如何選擇？

A：

- 覺察身體訊號： 如果做出理性的決定卻感到不舒補，那是靈魂在說「不」。
- 修復內在小孩（Reparenting）： 父母或外界認為的「好」不一定適合你，要學習重新教育並對待自己的內在小孩。

Q：我想要做戲劇結合療癒。想詢問老師課程如何設計？及能否能邀請你來我的podcast？

A：

- 系統化整理： 太多人詢問如何顯化，課程能比 90 秒的 IG 影片更完整地傳遞知識體系。
- 分享顯化回憶錄：什麼是心意的方向>>「你一定要先知道山頂的風景有多美，你才會有動力到山頂。」
- 結合直覺與工具： 她會持續向不同大師進修，並利用 ChatGPT 協助梳理邏輯，最終結合自己的直覺來開發內容。

#老師補充，通常你顯化的東西不是你想要的，而是你的大腦習慣的。

- 釐清你想要的感受是什麼
- 不要寫的太多，分區塊：事業、金錢、財務，選1~2個就好
- 用圖像方式分享如何做年度顯化，看到、聽到、聞到、摸到…用5感的方式讓他更明確，讓自己進入那個畫面中，許多運動員也都用這個方式。

Q：去年老師許了31個顯化目標，只成功了一半，我們一般人要用什麼心態面對顯化失敗這件事。

A：

- 時機未到：人生當然不是一路順遂，遊樂園都有鬼屋。看看自己過往的顯化目標，跟現在比起來一定有很大的差距（surprise），但如果當時我在意那一年沒有達到，然後我就放棄顯化，那就不用有今天這個結果。相信他不是不到，只是時候未到。
- 條條大路通羅馬：不要單戀一條路。如果你覺得這條路成功機率很低，就換一條路走。老師分享他想成為好萊塢明星，但不是很真的順遂，他就選擇果斷換賽道嘗試。

Q：我有上老師的顯化課，第1想詢問老師如何構想影片及課程？第2我的「工程師腦」不擅長視覺化想像該怎麼辦？第3即使我已經有防護罩，我該如何不受到外在負面能力的干擾？

A：

- GPT是我的好朋友：我會多元吸收所有網路上關於這件事（顯化）的內容，並且吸收自己相信的，然後彙整出自己的。然後問自己。
    
    ：「我自己有做過什麼很幫助到我自己？」
    
    ：「這些東西可以怎麼幫助到別人？」
    
- 允許想像： 很多人沒有想像力的原因是因為：「你覺得想像別的事情太危險。」，你需要把那個開關打開，去允許自己有想像的空間之所以卡住，通常是因為兒時被限制「不准做白日夢」，要先告訴自己「想像也是安全的」。
- 每一次被觸發都是一個禮物：你應該從自己每一次的觸發去發現、去認識自己為什麼當下有那個感受，這不是一個讓你變完美的過程，而是改變你思維的過程。

Q：我本身不是一個追求完美的人，但我的生活中有很多追求完美的人，我本身是一個鋼琴老師，身邊的人（如學生或家長）追求完美而不敢嘗試，如何幫他們？

A：尊重他人的選擇：你已經盡力了。人生本來就是一個追求體驗的過程，如果他們去迴避那些不好的體驗，那也是他們的人生選擇。你不用執著說這件事一定要發生，這樣只會把緊繃的能量給到他。推薦練習「金錢存錢筒」，每擔心一次別人怎麼看，就存一塊錢，讓人意識到擔心他人的眼光非常浪費生命。

Q： 冥想過程如何清空，有沒有具體的方法吧？

A：放鬆！你的大腦預設值是一直思考。

- 睡前練習： 睡前「半睡半醒」的狀態最容易放空。
- 想像無限空間（Void）： 想像自己沒有身體，只是一個光點或靈魂，待在一個無限可能的黑色或白色空間中。

        - 有可能是老師方法覺得特別有幫助？ 作為引導者、卡牌引導思考，引導者的思考特別讓學員覺得被療癒，是任何一個，後續宣傳銷售可以放大這個點，不用特別為自己做卡排、瑜伽的定義
- Q3-1: 在詢問學生時，會不會讓學生覺得過於商業？
    - 預算許可，可以找一個不是你的人來幫你訪問（不是老師自己去訪問學生），個人創作者會有難度 -> （自己去訪問學生的時候）可以問的細一點，你覺得我已經很棒了，那你覺得我好在哪裡？ 你說我的工作方帶的很好，你可以跟我分享三個你覺得好的地方？ 若他講的很虛，那就有可能是他在捧你（建立親和力）
- Q4：業界有聲量的老師第一次做課，完全依照老師想法給很多內容？做市場研究，依照市場研究的結果回饋給老師？
    - 做市場研究，邀請老師的鐵粉（6-8人）去訪談，優先說服鐵粉再往外擴圈
    - 第一次開課是最好轉換的時候
    - 可以依照以往的架構，主要參考鐵粉的聲音，鐵粉若是之前上過課的人，可能會敲碗進階課，第一堂課以初階為主，
- Q5：古典音樂教育，以前學生是音樂科班的學生，現在想要從實體轉成線上且受眾擴大，音樂體驗＋療癒，能否推給以前不會聽古典音樂的人？
    - 競品，江老師，已經有在用這個角度有在做小白市場推廣的音樂家，可以多看知識衛星
    - 市場調查
    - 成功做音樂相關，推廣給小白，目前都是有自體流量，不一定要開課，也可以導流到現有的服務，經營成功的話就代表有鐵粉，再去訪談粉絲，要線上課、實體課、直播課？ 先嘗試是否有可能養得起流量？
- Q6：身體放鬆的線上課程，偏手法與每個人體感問題不同，如果變成線上課程（單項知識傳遞），不確定對方的手法是否正確？可能影響到學員身體的問題？
    - 線上課程影片真的沒辦法，需要有一個嚴格檢核，可以用混成學習
    - 可以在某個單元設定節點，雙周、每月一次的直播，直接演繹一次，可以確保大家是否有犯錯，可以整理成 QA ，再上傳變成補充包
    - 或請學員錄製一堂課，用 AI 來做回饋
- Ｑ7：縫紉線上課比較偏興趣課，如何在非剛需（nice to have）的領域做出平價小白市場的區隔？
    - 不是剛需與否的差異，非剛需的角度、訴求去做包裝，要去確認競品切角，回到訪談，花錢願意去學縫紉相關要的目的是什麼？他是要做出一個作品還是只是喜歡做作品的過程療癒、放空的感覺？
    - 會不會大家需要的不是線上課？ 有可能大家買才料包更輕鬆？
- Q8：線上價格定定要參考什麼？
    - 損益兩平，成本、機會成本（時間成本）、行銷成本、廣告預算、經營電子報的成本、平台抽成
    - 競品定價
    - 市場願付價格
    - 如果算完發現做線上影片不符合損益，可以轉個想法，講義、線上工作坊

## 📌 Q&A 彙整

上半場—

Q1.心理諮商現實面的問題考量（地方政府、法令歸責等）

A1.Cary-外國（日本）學者認為諮商為個人服務，符合大眾服務導向，但心理方面屬醫療行為，現今法令、醫療單位及地方政府部門仍有不一樣的狀況，心理諮商是人文和社會性有高度相關的專業。

Q2.2017創立的目標和契機？（觀眾為醫療消費的CRM相關人員）

A2.更普遍推廣心理健康的契機，使用狀況隨時做調整，並多與使用者、諮詢教練等溝通，回饋更直覺。

Q3. 要怎麼找到適合的諮商師？

A3.

(1)理解自己需要的問題（議題）、是否了解自己。

(2)儘可能將問題（議題）如實告訴對方。

(3)諮詢師的行為、談吐及長相等。

Q4.遠距抱抱的詢問、覺察回饋不錯，但下一步行為方式或回答有時會遇到「卡住」的狀況，如何克服？

A4.可以根據《原子習慣》一書，詢問問題的動機取向，才能做出行為改變。

Q5.遠距抱抱的費用問題，是否能提供免費諮詢的次數？（ICF教練詢問）

A5. 認同提供免費諮詢，但目前考量可行性。

Q6.遇到身邊的人有產生心理問題時，有無適當的溝通方式？如何釐清界線及引導？怎麼與被溝通者建立信任關係？

A6.CHAT GPT、遠距抱抱都是很好的工具。界線及引導建議如實告知對象，心理諮詢師的角度來說，以心靈動力及精神分析為主要察覺方向，進一步建立關係。

Q7.如何判斷自己的心理狀況？

A7.先了解自己的痛苦來源，釐清自我成長時的狀況（行為結果導向等）。
