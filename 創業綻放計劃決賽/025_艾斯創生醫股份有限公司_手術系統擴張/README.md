# 025 艾斯創生醫股份有限公司 · 手術系統擴張

## 30 秒看懂

艾斯創（`Aztron Medtech`）主打阿基里斯腱微創修復系統 `TendMIN`，強調小切口、低併發症與縮短恢復期。  
簡報可見其商業化主軸是「美國給付路徑 + 醫院/經銷擴張 + 適應症延伸」，並已揭露實際臨床與營運指標。

## 他們在做什麼

- 產品定位：以小傷口完成肌腱縫合的手術器械/系統。
- 首要適應症：阿基里斯腱斷裂修復（逐字稿口述）。
- 差異主張：縮小傷口、降低併發症風險、縮短復健期。
- 商業化主軸：美國市場先行，搭配經銷與醫院臨床導入。

## 解決什麼問題

- 傳統阿基里斯腱修復多為較大傷口，病患復原時間長，且併發症風險較高。
- 醫師端需要更一致、學習曲線較短的微創工具。
- 醫材公司要跨過法規核准後，仍須解決給付、訓練與醫院採用門檻。

## 怎麼賺錢

- 商業模式：B2B 醫材銷售（賣給經銷與醫療院所），並由術式擴張帶動使用量。
- 收益驅動：器械本體 + 後續臨床擴點 + 新適應症擴展。
- 美國路徑：逐字稿提及已有保險/給付代碼可用，降低採用阻力。
- 募資用途：擴充適應症、擴大市場滲透、增聘團隊與國際拓展。

## 實績與進展（決賽現場）

- 逐字稿可辨識主張：
  - 產品頁標示 `FDA / TFDA cleared`，並有臨床病例照片與追蹤案例。
  - 科學驗證頁揭露：`5 papers`、`8+ patents granted`、`190+ clinical cases validated`。
  - GTM 頁揭露：`190+ US patients treated`、`40+ hospitals`、`37+ distributors`、`2026 run rate $900K`。
- 評審追問集中：
  - 給付真實可用性與適應症可擴張性。
  - 醫師訓練流程是否可規模化。
  - 除阿基里斯腱外，是否具備可延展的產品線與市場。

## 評審提問與回答

### Q1. 美國健保/保險到底怎麼支付？是否已經可用？
- 團隊回答：目前在美國已有可對應的保險支付路徑（逐字稿提及 CPT code 及整合方式）。
- 正確性註記：`中信心`
  - 理由：回答方向明確，但代碼與給付覆蓋範圍需正式文件核對。

### Q2. 醫師要不要受訓？訓練成本會不會太高？
- 團隊回答：有標準化訓練流程，含模型練習與進階實作；醫師可在相對短週期內導入。
- 正確性註記：`中高信心`
  - 理由：回覆有流程與時程概念，臨床導入敘述一致。

### Q3. 除了美國，還會往哪些市場走？
- 團隊回答：美國先行，並同步規劃其他國際市場（如歐洲/亞洲）擴張。
- 正確性註記：`中信心`
  - 理由：策略方向清楚，但未提供各地法規與商業時程。

### Q4. 現在募資要做什麼？資本擴張計畫是什麼？
- 團隊回答：資金主要投入擴適應症、增加商業化團隊、擴大國際市場滲透。
- 正確性註記：`中信心`
  - 理由：用途合理，仍需與分期里程碑（何時達標）對齊。

### Q5. 為什麼先鎖阿基里斯腱？其他適應症什麼時候擴？
- 團隊回答：阿基里斯腱是可快速驗證臨床價值與商業可行性的切入點，後續往膝蓋/肩膀等適應症拓展。
- 正確性註記：`中高信心`
  - 理由：切入策略與醫材路徑一致，符合先窄後廣邏輯。

## 我學到的 3 件事

1. 醫材公司要規模化，不能只講核准，還要講清楚給付與訓練。
2. 「先單一術式打穿，再擴適應症」是高效率且可驗證的路徑。
3. 募資說服力來自「資金用途對應里程碑」，不是募資金額本身。
