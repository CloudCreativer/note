# 019 日日好食 · 控糖高蛋白麵

## 30 秒看懂

日日好食做**控糖高蛋白快煮麵**——一包約等同**一片雞胸肉蛋白**、半顆高麗菜纖維、低碳水，鎖定健身、減重、糖友與銀髮族。  
以 **D2C 電商**掌握數據（ROAS 約 **500%**、回購約 **20%**），三年營收成長約 **1,200%**；若獲資源：**300 萬**深耕台灣、**400 萬**建**日常生活血糖資料庫**（與健保互補）。

| **模式** | 研發 + 品牌自控，生產多廠合作 + NDA |

---

## 解決什麼問題

- 台灣慢性病健保支出龐大；民眾知要控糖／高蛋白，但**離不開麵食**。  
- 日日好食用「**一碗麵**」降低飲食改變門檻，並用血糖數據說服（簡報／《控糖 58 週》倡議）。

## 評審提問與回答

### Q1. 這包麵賣多少錢？要煮多久？除了麵以外還會做其他產品嗎？

一袋均價約 **239 元**／包，一碗約 **80 多元**，溢價約**25–50%* *。現行煮約 **7 分鐘**，二代約 **5 分鐘**。餅乾、披薩皮、水餃皮等常溫品可行。

### Q2. 在地生產與防抄如何做？

與多家工廠合作，參數為 know-how，簽**保密協議**；保護靠配方分層 + NDA。

### Q3. 食品添加物與健康原則如何平衡？

**麵條無添加物**；添加物在**醬料**（出口航運約兩個月需穩定度）。

### Q4. 蛋白質／纖維訴求的專利或技術？

**新型專利** 1 件；主要靠配方分層隔離 + 工廠 NDA。六年前即做主食控糖高蛋白，累積主動回購客群與數據。

## 我學到的 3 件事

1. **功能食品也能走數據驅動**：日日好食不是只賣產品，而是用 D2C 指標與血糖驗證建立說服力。  
2. **產品策略重點是「不改習慣」**：比起教育消費者完全戒澱粉，用替代方案更容易擴散。  
3. **早期品牌可先用商業數據建立信任**：ROAS、CPC、回購率與三年成長，是比口號更有力的溝通語言。
