# 007 炬智科技 金屬檢測

## 30 秒看懂

炬智科技（`PLASMIC`）主打「水下微電漿」線上即時金屬離子分析儀，目標把高階 PCB/IC 載板濕製程從離線抽樣改成秒級連續監控。  
簡報可辨識其核心賣點是：`30+` 元素同步、`0.1 ppm` 靈敏度、免氬氣、可整合製程槽；商模則以整機 + 耗材 + 維保建立長期 LTV。

## 他們在做什麼

- 聚焦精密製程中的金屬成分/污染檢測。
- 目標客群推測包含高階製造與半導體供應鏈。
- 產品可能包含檢測設備 + 分析流程 + 導入服務。

## 解決什麼問題

- 傳統檢測在速度、靈敏度、或線上化程度可能不足。
- 客戶痛點通常是良率風險與異常發生後追查成本高。
- 若可提早檢出關鍵金屬偏差，可降低報廢與停線風險。

## 怎麼賺錢

- 簡報可辨識四大收入來源：
  - 整機銷售：`400萬/台`（線上即時金屬離子檢測設備）；
  - 耗材收入：`35萬/年`（電極、標準液等）；
  - 年度維護：校正與維護支援；
  - 資料服務：未來加值（跨製程分析、擴充測項）。
- 通路策略：透過 PCB 濕製程設備商與藥水供應商合作導入，再服務高階 PCB/IC 載板廠。

## 實績與進展（決賽現場）

- 逐字稿與簡報可辨識重點：
  - 技術指標：可同步檢測 `30+` 金屬元素、靈敏度 `0.1 ppm`、秒級線上監控、免氬氣導入。
  - 產業進度：Tier-1 載板廠產線 PoC 進行中，並規劃 2026 完成 `2` 家載板產線驗證與 V1.0 工業機台。
  - 財務路徑：2026~2029 營收預估由 `800萬` 成長至 `1.39億`（簡報口徑，維護/資料服務尚未全納入）。
  - 資金規劃：爭取 `900萬` 支持金（45% 客規儀器、35% 產線驗證、20% 立案商拓），目標取得首份採購意向書。
  - 逐字稿補充：口述已售出 `7` 台（學研與產業場域），需與正式出貨/驗收清單再核對。

## 我學到的 3 件事

1. 精密檢測新創要拿到訂單，關鍵是「客戶回本模型」而不只技術性能。
2. 半導體場景導入難點常在長驗證週期與跨部門決策。
3. 評審最在意的是可規模化的商業證據，而非單一技術亮點。
