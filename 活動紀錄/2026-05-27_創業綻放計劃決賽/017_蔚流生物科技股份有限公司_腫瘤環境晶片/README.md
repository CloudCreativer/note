# 017 蔚流生物 · MedSelect 腫瘤環境晶片

## 30 秒看懂

蔚流生物的 **MedSelect**（簡報亦作 Massaline／NASA-Lite）是在**新藥進臨床動物實驗之前**，用**腫瘤微環境晶片**驗證 ADC、雙抗等**功能性療效**的平台——比傳統動物實驗更快、並可測區域／即時反應。  
商業路徑：**CRO 服務**打市場 → 銷售 **MSI Ultra 設備 + 晶片** → 累積資料做**數位雙生**訂閱。FDA 接受非動物替代方案，為切入時機。

---

## 他們在做什麼

| 面向 | 說明 |
|------|------|
| **產品** | 腫瘤環境晶片 + **MSI Ultra**（培養、控制、顯微整合） |
| **流程** | 患者腫瘤／檢體 → 前處理 → 移植晶片 → 重塑微環境 → 藥物測試 |
| **差異** | 可測 **ADC、雙抗** 功能；非僅培養類器官；晶片可模擬缺氧、纖維化、血管床等 |
| **切入點** | **AI 篩選 → 類器官 → 【蔚流】→ GLP 動物**（與純類器官競品錯位） |

---

## 解決什麼問題

- 新藥（尤其 ADC、雙抗）用動物／培養盤**難評估功能性**，開發常拖 **1–2 年**；約 **20%** 臨床失敗與臨床前驗證不足相關（簡報）。  
- 藥廠需要**更快、更準**的臨床前功能驗證，並因應 **FDA 非動物替代**趨勢。

---

## 評審提問與回答

### Q1. 準確度 85% 到 92% 是根據多少次測試、多久得到的數據？

我們跟新竹、台中、廣州、台大平鎮等地醫院合作：收集患者手術檢體，在醫師治療**之前**對晶片上的腫瘤模型做測試，再對照病人實際治療後結果。晶片上有效、病人身上也有效，就定義為相關。分母約 **30–40 例**，分子落在 **85–92%**。

### Q2. 你們到底要賣晶片、帶檢驗，還是銷售檢驗器？也沒看到專利品狀。

我們定義自己是**設備開發商**，核心目標是銷售**設備 + 晶片**。先做 CRO／平行測試服務，是因為藥廠數十年慣用動物實驗，需用平行方式降低轉換阻力；服務是市場教育策略，長期仍以設備銷售為主。專利佈局在**晶片設計與架構**。

### Q3. 請再說明團隊組成。

對外兩位為主要經營角色（含現場秀俊，具 MBA／財務背景）；其餘多為科學家與早期行銷合作夥伴。此階段最重要的是說服藥廠採用平行、非動物替代方案。

### Q4. 前面案例選了親和性很低的兩個藥，親和性跟這個測試有什麼關係？

那是**胰臟癌**案例：要回答藥物在合理劑量下是否有效，以及是否產生**免疫介導的全身性肝毒性**。晶片可培養肝臟相關細胞，驗證肝毒性與合理劑量窗口。

## 我學到的 3 件事

1. **平台價值在「功能性」驗證**：第 17 組不是只做培養，而是要補足 ADC/雙抗在動物前的決策證據。  
2. **醫材生技需要分階段商模**：先以 CRO 服務教育市場，再推設備與耗材，最後才走資料訂閱，路徑更務實。  
3. **臨床轉譯是核心說服力**：案例縮時（6 個月到 2 週）與 85~92% 對照數據，是他們最有力的商業敘事。
