## Insight（課 1）

做局 ≠ 自己更努力，而是**設計一個不用你親自扛也能完成的系統**。核心轉折：當規模大到 279 間公司，你自然會放掉「只有我能做」的執念，改找總經理、建匯報機制——問題是**為什麼現在規模小時不這樣做**？

會議裡最關鍵的角色是**筆記者**（不是發言者）：局自己、記錄眾人想法、最後匯報時不會點自己。成事靠**共識**而非多數決（投票是看似公平的霸凌）。四步驟成事順環 + 咒語「**我能提供什麼，願意讓你做這件事**」可套用在談判、加薪、成交。布局四要素：**對的人、對的位置、對的目標、對的路**——錯一個後面不用看。成群（把人聚在一起）≠ 結隊（把事做完）；要用**眾人之力解決個人問題**，不是反過來。
